### <a name="compiler-support-for-type-forwarding-when-multi-targeting-mscorlib"></a>mscorlib の複数バージョン対応時の型転送のコンパイラ サポート

#### <a name="details"></a>説明

新しい CodeDOM の機能を使用すると、mscorlib.dll の .NET Framework 4.5 バージョンではなく、mscorlib.dll の対象バージョンに対してコンパイルできるようになります。

| 名前    | [値]       |
|:--------|:------------|
| スコープ   |エッジ|
|バージョン|4.5|
|種類|ランタイム|

#### <a name="affected-apis"></a>影響を受ける API

API 分析では検出できません。
