### <a name="install-the-global-tool"></a><span data-ttu-id="45653-101">グローバル ツールをインストールする</span><span class="sxs-lookup"><span data-stu-id="45653-101">Install the global tool</span></span>

<span data-ttu-id="45653-102">パッケージ アセットは `--tool-path` オプションを使用して保護された場所にインストールする必要があります。</span><span class="sxs-lookup"><span data-stu-id="45653-102">Package assets should be installed in a protected location using the `--tool-path` option.</span></span> <span data-ttu-id="45653-103">このように分けることにより、制限付きユーザー環境が特権環境と共有されることを回避できす。</span><span class="sxs-lookup"><span data-stu-id="45653-103">This separation avoids sharing a restricted user environment with an elevated environment.</span></span>

```bash
sudo dotnet tool install PACKAGEID --tool-path /usr/local/share/dotnet-tools
```

<span data-ttu-id="45653-104">`/usr/local/share/dotnet-tools` は `drwxr-xr-x` アクセス許可で作成されます。</span><span class="sxs-lookup"><span data-stu-id="45653-104">`/usr/local/share/dotnet-tools` will be created with permission `drwxr-xr-x`.</span></span> <span data-ttu-id="45653-105">ディレクトリが既に存在する場合は、`ls -l` コマンドを使用して、制限付きユーザーにディレクトリを編集するアクセス許可がないことを確認します。</span><span class="sxs-lookup"><span data-stu-id="45653-105">If the directory already exists, use the `ls -l` command to verify the restricted user doesn't have permission to edit the directory.</span></span> <span data-ttu-id="45653-106">ある場合は、`sudo chmod o-w -R /usr/share/dotnet-tools` コマンドを使用してそのアクセス許可を削除します。</span><span class="sxs-lookup"><span data-stu-id="45653-106">If so, use the `sudo chmod o-w -R /usr/share/dotnet-tools` command to remove the access.</span></span>

### <a name="run-the-global-tool"></a><span data-ttu-id="45653-107">グローバル ツールを実行する</span><span class="sxs-lookup"><span data-stu-id="45653-107">Run the global tool</span></span>

<span data-ttu-id="45653-108">**オプション 1** sudo で完全なパスを使用します。</span><span class="sxs-lookup"><span data-stu-id="45653-108">**Option 1** Use the full path with sudo:</span></span>

<span data-ttu-id="45653-109">**オプション 2** ツールごとに 1 回、ツールのシンボルのリンクを追加します。</span><span class="sxs-lookup"><span data-stu-id="45653-109">**Option 2** Add the symbol link of the tool, once per tool:</span></span>


<span data-ttu-id="45653-110">次で実行します。</span><span class="sxs-lookup"><span data-stu-id="45653-110">And run with:</span></span>
