> [!NOTE]
> .NET Framework 4.8 は .NET Framework の最終バージョンであり、これ以降のバージョン リリースはありません。 ただし、.NET Framework では、セキュリティと信頼性のバグ修正プログラムが[引き続き](https://devblogs.microsoft.com/dotnet/?s=%22.net+framework%22)月次で提供されます。 また、これは Windows に引き続き含まれ、削除される予定はありません。 お使いの .NET Framework アプリは移行する必要はありませんが、新規開発においては [.NET 5.0 以降](~/docs/core/introduction.md)をご利用ください。
