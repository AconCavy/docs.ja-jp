### <a name="shared-framework-assemblies-removed-from-microsoftaspnetcoreapp"></a><span data-ttu-id="c3f5a-101">共有フレームワーク: Microsoft.AspNetCore.App から削除されたアセンブリ</span><span class="sxs-lookup"><span data-stu-id="c3f5a-101">Shared framework: Assemblies removed from Microsoft.AspNetCore.App</span></span>

#### <a name="change-description"></a><span data-ttu-id="c3f5a-103">変更の説明</span><span class="sxs-lookup"><span data-stu-id="c3f5a-103">Change description</span></span>
