## <a name="prepare-the-raspberry-pi"></a>Raspberry Pi を準備する

コマンドを使用して `raspi-config` 、次の2つのサービスが有効になっていることを確認します。

- SSH
- SPI

の詳細については `raspi-config` 、 [Raspberry Pi のドキュメント](https://www.raspberrypi.org/documentation/configuration/raspi-config.md)を参照してください <span class="docon docon-navigate-external x-hidden-focus"></span> 。
