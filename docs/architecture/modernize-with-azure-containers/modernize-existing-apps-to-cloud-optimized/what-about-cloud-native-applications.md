## <a name="what-about-microservices"></a><span data-ttu-id="c7b91-136">マイクロサービスについて</span><span class="sxs-lookup"><span data-stu-id="c7b91-136">What about microservices?</span></span>

<span data-ttu-id="c7b91-137">組織のクラウドネイティブ アプリケーションを検討する際は、マイクロサービスとそのしくみについて理解していることが重要です。</span><span class="sxs-lookup"><span data-stu-id="c7b91-137">Understanding microservices and how they work is important when you are considering cloud-native applications for your organization.</span></span>

<span data-ttu-id="c7b91-138">マイクロサービス アーキテクチャは、ゼロから作成するアプリケーションや既存のアプリケーションをクラウドネイティブ アプリケーションに進化させるときに使用できる高度なアプローチです。</span><span class="sxs-lookup"><span data-stu-id="c7b91-138">The microservices architecture is an advanced approach that you can use for applications that are created from scratch or when you evolve existing applications toward cloud-native applications.</span></span> <span data-ttu-id="c7b91-139">新しいマイクロサービス パラダイムについて学習するには、まず、既存のアプリケーションにいくつかのマイクロサービスを追加します。</span><span class="sxs-lookup"><span data-stu-id="c7b91-139">You can start by adding a few microservices to existing applications to learn about the new microservices paradigms.</span></span> <span data-ttu-id="c7b91-140">ただし、この種類のアーキテクチャ アプローチには特に、設計とコーディングが必要なことは明らかです。</span><span class="sxs-lookup"><span data-stu-id="c7b91-140">But clearly, you need to architect and code, especially for this type of architectural approach.</span></span>
