## <a name="how-to-combine-serverless-and-kubernetes-with-keda"></a><span data-ttu-id="8903d-123">サーバーレスと Kubernetes を KEDA と組み合わせる方法</span><span class="sxs-lookup"><span data-stu-id="8903d-123">How to combine serverless and Kubernetes with KEDA</span></span>

<span data-ttu-id="8903d-124">この章では、Azure Functions のプラットフォームが需要に応じて自動的にスケールアウトされることを確認しました。</span><span class="sxs-lookup"><span data-stu-id="8903d-124">In this chapter, you've seen that the Azure Functions' platform automatically scales out to meet demand.</span></span> <span data-ttu-id="8903d-125">ただし、コンテナー化された関数を AKS に配置する場合、組み込みのスケーリング機能は失われます。</span><span class="sxs-lookup"><span data-stu-id="8903d-125">When deploying containerized functions to AKS, however, you lose the built-in scaling functionality.</span></span> <span data-ttu-id="8903d-126">これには、 [Kubernetes ベースのイベントドリブン (KEDA)](https://docs.microsoft.com/azure/azure-functions/functions-kubernetes-keda)が用意されています。</span><span class="sxs-lookup"><span data-stu-id="8903d-126">To the rescue comes [Kubernetes-based Event Driven (KEDA)](https://docs.microsoft.com/azure/azure-functions/functions-kubernetes-keda).</span></span> <span data-ttu-id="8903d-127">これにより、コンテナー化された関数を含めるための高度な自動スケールが可能になり `event-driven Kubernetes workloads,` ます。</span><span class="sxs-lookup"><span data-stu-id="8903d-127">It enables fine-grained autoscaling for `event-driven Kubernetes workloads,` including containerized functions.</span></span>
