## <a name="deploying-a-monolithic-application-as-a-container"></a><span data-ttu-id="4c4ba-145">モノリシック アプリケーションをコンテナーとして展開する</span><span class="sxs-lookup"><span data-stu-id="4c4ba-145">Deploying a monolithic application as a container</span></span>
