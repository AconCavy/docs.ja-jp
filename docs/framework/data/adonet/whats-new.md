## <a name="sqlclient-data-provider"></a><span data-ttu-id="ec61b-105">SqlClient Data Provider</span><span class="sxs-lookup"><span data-stu-id="ec61b-105">SqlClient Data Provider</span></span>

- <span data-ttu-id="ec61b-107">ConnectRetryCount と ConnectRetryInterval の接続文字列キーワード (<xref:System.Data.SqlClient.SqlConnection.ConnectionString%2A>) を使用すると、アイドル状態の接続の復元機能を制御できます。</span><span class="sxs-lookup"><span data-stu-id="ec61b-107">The ConnectRetryCount and ConnectRetryInterval connection string keywords (<xref:System.Data.SqlClient.SqlConnection.ConnectionString%2A>) let you control the idle connection resiliency feature.</span></span>

- <span data-ttu-id="ec61b-108">SQL Server からアプリケーションへのストリーミング サポートで、サーバー上のデータが構造化されていないシナリオがサポートされます。</span><span class="sxs-lookup"><span data-stu-id="ec61b-108">Streaming support from SQL Server to an application supports scenarios where data on the server is unstructured.</span></span>  <span data-ttu-id="ec61b-109">詳しくは、「[SqlClient ストリーミング サポート](sqlclient-streaming-support.md)」をご覧ください。</span><span class="sxs-lookup"><span data-stu-id="ec61b-109">See [SqlClient Streaming Support](sqlclient-streaming-support.md) for more information.</span></span>

- <span data-ttu-id="ec61b-118">`TrustServerCertificate` が false であり、`Encrypt` が true の場合は、SQL Server SSL 証明書のサーバー名 (または IP アドレス) が、接続文字列で指定されているサーバー名 (または IP アドレス) と正確に一致する必要があります。</span><span class="sxs-lookup"><span data-stu-id="ec61b-118">When `TrustServerCertificate` is false and `Encrypt` is true, the server name (or IP address) in a SQL Server SSL certificate must exactly match the server name (or IP address) specified in the connection string.</span></span> <span data-ttu-id="ec61b-119">それ以外の場合、接続試行は失敗します。</span><span class="sxs-lookup"><span data-stu-id="ec61b-119">Otherwise, the connection attempt will fail.</span></span> <span data-ttu-id="ec61b-120">詳細については、「`Encrypt`」の <xref:System.Data.SqlClient.SqlConnection.ConnectionString%2A> 接続オプションの説明を参照してください。</span><span class="sxs-lookup"><span data-stu-id="ec61b-120">For more information, see the description of the `Encrypt` connection option in <xref:System.Data.SqlClient.SqlConnection.ConnectionString%2A>.</span></span>

  <span data-ttu-id="ec61b-121">この変更によって既存のアプリケーションが接続しなくなる場合、次のいずれかを使用してアプリケーションを修正できます。</span><span class="sxs-lookup"><span data-stu-id="ec61b-121">If this change causes an existing application to no longer connect, you can fix the application using one of the following:</span></span>

  - <span data-ttu-id="ec61b-122">共通名 (CN) またはサブジェクト代替名 (SAN) フィールドに短い名前を指定する証明書を発行します。</span><span class="sxs-lookup"><span data-stu-id="ec61b-122">Issue a certificate that specifies the short name in the Common Name (CN) or Subject Alternative Name (SAN) field.</span></span> <span data-ttu-id="ec61b-123">このソリューションは、データベース ミラーリングをする場合に有効です。</span><span class="sxs-lookup"><span data-stu-id="ec61b-123">This solution will work for database mirroring.</span></span>

  - <span data-ttu-id="ec61b-124">短い名前を完全修飾ドメイン名にマップする別名を追加します。</span><span class="sxs-lookup"><span data-stu-id="ec61b-124">Add an alias that maps the short name to the fully-qualified domain name.</span></span>

  - <span data-ttu-id="ec61b-125">接続文字列では完全修飾ドメイン名を使用します。</span><span class="sxs-lookup"><span data-stu-id="ec61b-125">Use the fully-qualified domain name in the connection string.</span></span>

- <span data-ttu-id="ec61b-126">SqlClient は拡張保護をサポートしています。</span><span class="sxs-lookup"><span data-stu-id="ec61b-126">SqlClient supports Extended Protection.</span></span> <span data-ttu-id="ec61b-127">拡張保護に関する詳細については、「[拡張保護を使用したデータベース エンジンへの接続](/sql/database-engine/configure-windows/connect-to-the-database-engine-using-extended-protection)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="ec61b-127">For more information about Extended Protection, see [Connecting to the Database Engine Using Extended Protection](/sql/database-engine/configure-windows/connect-to-the-database-engine-using-extended-protection).</span></span>

- <span data-ttu-id="ec61b-128">SqlClient は LocalDB データベースへの接続をサポートします。</span><span class="sxs-lookup"><span data-stu-id="ec61b-128">SqlClient supports connections to LocalDB databases.</span></span> <span data-ttu-id="ec61b-129">詳しくは、「[SqlClient による LocalDB のサポート](./sql/sqlclient-support-for-localdb.md)」をご覧ください。</span><span class="sxs-lookup"><span data-stu-id="ec61b-129">For more information, see [SqlClient Support for LocalDB](./sql/sqlclient-support-for-localdb.md).</span></span>
