# <a name="how-to-customize-entity-classes-by-using-the-code-editor"></a>方法: コード エディターを使用してエンティティ クラスをカスタマイズする
Visual Studio を使用する開発者は、オブジェクトリレーショナルデザイナーを使用して、エンティティクラスを作成またはカスタマイズできます。  
  
 また、Visual Studio コードエディターを使用して、独自のマッピングコードを記述したり、既に生成されているコードをカスタマイズしたりすることもできます。 詳細については、「[属性ベースのマッピング](attribute-based-mapping.md)」を参照してください。
