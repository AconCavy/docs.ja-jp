 <span data-ttu-id="9a3b2-119">[!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] では、オブジェクト ID は <xref:System.Data.Linq.DataContext> によって管理されます。</span><span class="sxs-lookup"><span
