# <a name="querying-a-conceptual-model"></a>概念モデルに対するクエリ
ADO.NET Entity Framework を使用すると、概念モデルに対してクエリを実行できます。 最新バージョンの Entity Framework を使用して概念モデルに対してクエリを実行するには、「[データ](https://go.microsoft.com/fwlink/?LinkId=235282)のクエリ」を参照してください。
