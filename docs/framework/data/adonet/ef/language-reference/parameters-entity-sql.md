<span data-ttu-id="db0ac-103">パラメーターは、[!INCLUDE[esql](../../../../../../includes/esql-md.md)] の外部で定義される変数です。通常は、ホスト言語で使用されるバインド API を通じて定義されます。</span><span class="sxs-lookup"><span data-stu-id="db0ac-103">Parameters are variables that are defined outside [!INCLUDE[esql](../../../../../../includes/esql-md.md)], usually through a binding API that is used by a host language.</span></span> <span data-ttu-id="db0ac-104">それぞれのパラメーターには、名前と型があります。</span><span class="sxs-lookup"><span data-stu-id="db0ac-104">Each parameter has a name and a type.</span></span> <span data-ttu-id="db0ac-105">パラメーター名は、クエリ式の中で、先頭に @ 記号を付けることによって定義します。</span><span class="sxs-lookup"><span data-stu-id="db0ac-105">Parameter names are defined in query expressions with the at (@) symbol as a prefix.</span></span> <span data-ttu-id="db0ac-106">これにより、クエリ内で定義されている他の名前 (プロパティ名など) と明確に区別されます。</span><span class="sxs-lookup"><span data-stu-id="db0ac-106">This disambiguates them from the names of properties or other names that are defined in the query.</span></span>
