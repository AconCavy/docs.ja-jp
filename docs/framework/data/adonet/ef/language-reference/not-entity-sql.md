# <a name="-not-entity-sql"></a>! (NOT) (Entity SQL)
`Boolean` 型の式を否定します。  
  
## <a name="syntax"></a>構文  
  
```  
NOT boolean_expression  
or  
! boolean_expression  
```  
  
## <a name="arguments"></a>引数  
 `boolean_expression`  
 ブール値を返す任意の有効な式。  
  
## <a name="remarks"></a>Remarks  
 感嘆符 (!) には、NOT 演算子と同じ機能があります。
