# <a name="icorprofilerinfo10suspendruntime-method"></a>ICorProfilerInfo10:: SuspendRuntime メソッド

GC を実行せずにランタイムを中断します。

## <a name="syntax"></a>構文

```cpp
HRESULT SuspendRuntime();
```
