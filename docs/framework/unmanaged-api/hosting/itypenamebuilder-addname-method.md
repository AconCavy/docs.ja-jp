# <a name="itypenamebuilderaddname-method"></a>ITypeNameBuilder::AddName メソッド

このメソッドは、.NET Framework インフラストラクチャをサポートします。独自に作成したコードから直接使用するためのものではありません。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT AddName (  
    [in] LPCWSTR szName  
);  
```
