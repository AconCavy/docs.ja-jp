# <a name="itypenamebuilder-interface"></a>ITypeNameBuilder インターフェイス

型名を構築するためのメソッドを提供します。  
  
 このインターフェイスは、.NET Framework インフラストラクチャをサポートします。独自に作成したコードから直接使用するためのものではありません。

 **ライブラリ:** MSCorEE.dll にリソースとして含まれています
