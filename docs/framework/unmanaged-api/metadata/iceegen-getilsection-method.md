# <a name="iceegengetilsection-method"></a>ICeeGen::GetIlSection メソッド
指定したハンドルによって参照される基本の中間言語コードのセクションを取得します。  
  
 このメソッドは廃止され、使用する必要があります。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT GetIlSection (  
    [in] HCEESECTION  *section  
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `section`  
 [in]取得するセクションへのハンドル。
