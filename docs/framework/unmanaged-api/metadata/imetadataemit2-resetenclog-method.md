# <a name="imetadataemit2resetenclog-method"></a>IMetaDataEmit2::ResetENCLog メソッド

エディットコンティニュログをリセットし、新しいセッションを開始します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT ResetENCLog ();  
```  
  
## <a name="parameters"></a>パラメーター  

 `ResetENCLog` パラメーターを受け取りません。
