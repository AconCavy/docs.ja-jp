# <a name="iceegenallocatemethodbuffer-method"></a>ICeeGen::AllocateMethodBuffer メソッド
メソッドで指定されたサイズのバッファーを作成し、メソッドの相対仮想アドレスを取得します。  
  
 このメソッドは廃止され、使用する必要があります。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT AllocateMethodBuffer (   
    [in]  ULONG    cchBuffer,   
    [out] UCHAR    **lpBuffer,  
    [out] ULONG    *RVA  
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `cchBuffer`  
 [in]作成するバッファーの長さ。  
  
 `lpBuffer`  
 [out]返されたバッファー。  
  
 `RVA`  
 [out]メソッドの相対仮想アドレス。
