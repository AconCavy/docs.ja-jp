# <a name="imetadataemitdefineuserstring-method"></a>IMetaDataEmit::DefineUserString メソッド
指定したリテラル文字列のメタデータ トークンを取得します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT DefineUserString (
    [in]  LPCWSTR     szString,
    [in]  ULONG       cchString,
    [out] mdString    *pstk
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `szString`  
 [in]格納するユーザー文字列。  
  
 `cchString`  
 [in]のワイド文字の`szString`数。  
  
 `pstk`  
 [アウト]割り当てられた文字列トークン。
