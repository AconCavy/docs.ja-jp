# <a name="coiniticor-enumeration"></a>COINITICOR 列挙型

共通言語ランタイムを初期化するときに、 [Co初期化 Ecor](../hosting/coinitializecor-function.md) によって使用される定数を指定します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
typedef enum tagCOINITCOR  
{  
    COINITCOR = 0x0  
} COINITICOR;  
```  
  
## <a name="members"></a>メンバー  
  
|メンバー|説明|  
|------------|-----------------|  
|`COINITCOR`|既定の初期化モードを示します。|
