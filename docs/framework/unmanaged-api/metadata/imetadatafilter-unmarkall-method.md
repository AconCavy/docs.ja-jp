# <a name="imetadatafilterunmarkall-method"></a>IMetaDataFilter::UnmarkAll メソッド
現在のメタデータ スコープ内のすべてのトークンから処理のマークを削除します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT UnmarkAll ();  
```  
  
## <a name="parameters"></a>パラメーター  
 `UnmarkAll` パラメーターはありません。
