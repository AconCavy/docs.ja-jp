# <a name="imetadatavalidatevalidatemetadata-method"></a>IMetaDataValidate::ValidateMetaData メソッド

現在のメタデータ スコープ内にあるオブジェクトのメタデータ署名を検証します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT ValidateMetaData ();  
```
