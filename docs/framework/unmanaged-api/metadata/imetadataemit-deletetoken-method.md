# <a name="imetadataemitdeletetoken-method"></a>IMetaDataEmit::DeleteToken メソッド
現在のメタデータ スコープから指定したトークンを削除します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT DeleteToken (
    [in]  mdToken     tkObj
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `tkObj`  
 [in]削除するトークン。
