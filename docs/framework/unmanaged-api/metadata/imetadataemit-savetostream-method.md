# <a name="imetadataemitsavetostream-method"></a>IMetaDataEmit::SaveToStream メソッド
指定した現在のスコープ内のすべてのメタデータの保存`IStream`します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT SaveToStream (   
    [in]  IStream     *pIStream,  
    [in]  DWORD       dwSaveFlags  
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `pIStream`  
 [in]保存する書き込み可能なストリーム。  
  
 `dwSaveFlags`  
 [in] 予約されています。 ゼロを指定してください。
