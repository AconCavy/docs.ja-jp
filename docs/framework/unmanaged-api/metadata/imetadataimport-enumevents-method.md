# <a name="imetadataimportenumevents-method"></a>IMetaDataImport::EnumEvents メソッド
指定した TypeDef トークンのイベント定義トークンを列挙します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT EnumEvents (   
   [in, out] HCORENUM    *phEnum,   
   [in]      mdTypeDef   td,   
   [out]     mdEvent     rEvents[],   
   [in]      ULONG       cMax,  
   [out]    ULONG        *pcEvents  
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `phEnum`  
 [入力、出力]列挙子へのポインター。  
  
 `td`  
 [in]イベント定義を持つが列挙 TypeDef トークンです。  
  
 `rEvents`  
 [out]返されるイベントの配列。  
  
 `cMax`  
 [in] `rEvents` 配列の最大サイズ。  
  
 `pcEvents`  
 [out]実際に返されるイベントの数`rEvents`します。  
  
## <a name="return-value"></a>戻り値  
  
|HRESULT|説明|  
|-------------|-----------------|  
|`S_OK`|`EnumEvents` 正常に返されます。|  
|`S_FALSE`|列挙するためにイベントはありません。 その場合は、`pcEvents`は 0 です。|
