# <a name="iassemblyenumclone-method"></a>IAssemblyEnum::Clone メソッド
この[Iassemblyenum](iassemblyenum-interface.md)オブジェクトの簡易コピーを作成します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT Clone (  
    [out] IAssemblyEnum   **ppEnum  
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `ppEnum`  
 入出力コピーへのポインター。
