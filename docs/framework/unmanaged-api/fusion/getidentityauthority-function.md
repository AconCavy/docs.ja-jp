# <a name="getidentityauthority-function"></a>GetIdentityAuthority 関数

## <a name="syntax"></a>構文  
  
```cpp  
HRESULT GetIdentityAuthority (  
    [out] IIdentityAuthority   **ppIIdentityAuthority  
 );  
```  
  
## <a name="parameters"></a>パラメーター
