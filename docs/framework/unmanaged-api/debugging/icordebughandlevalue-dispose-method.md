# <a name="icordebughandlevaluedispose-method"></a>ICorDebugHandleValue::Dispose メソッド

インターフェイスポインターを明示的に解放せずに、このオブジェクトによって参照されるハンドルを解放します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT Dispose ();  
```
