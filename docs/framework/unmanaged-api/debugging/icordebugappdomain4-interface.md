# <a name="icordebugappdomain4-interface"></a>ICorDebugAppDomain4 インターフェイス
コードを論理的に拡張して、COM 呼び出し可能ラッパーからマネージオブジェクトを取得します。  
  
## <a name="method"></a>メソッド  
  
|メソッド|説明|  
|------------|-----------------|  
|[GetObjectForCCW メソッド](../../../../docs/framework/unmanaged-api/debugging/icordebugappdomain4-getobjectforccw-method.md)|COM 呼び出し可能ラッパー (CCW: COM Callable Wrapper) ポインターからマネージド オブジェクトを取得します。|
