# <a name="icordebugclassgettoken-method"></a>ICorDebugClass::GetToken メソッド

`TypeDef`このクラスの定義を参照するメタデータトークンを取得します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT GetToken (  
    [out] mdTypeDef          *pTypeDef  
);  
```  
  
## <a name="parameters"></a>パラメーター  

 `pTypeDef`  
 入出力 `mdTypeDef` このクラスの定義を参照するトークンへのポインター。
