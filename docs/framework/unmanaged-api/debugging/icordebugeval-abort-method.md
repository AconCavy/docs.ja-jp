# <a name="icordebugevalabort-method"></a>ICorDebugEval::Abort メソッド

こののオブジェクトが現在実行している計算を中止します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT Abort ();  
```  
  
## <a name="remarks"></a>解説  

 評価が入れ子になっていて、それが最新ではない場合、 `Abort` メソッドは失敗する可能性があります。
