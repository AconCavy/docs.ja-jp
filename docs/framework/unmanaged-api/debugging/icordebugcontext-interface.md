# <a name="icordebugcontext-interface"></a>ICorDebugContext インターフェイス

コンテキストのオブジェクトを表します。 このインターフェイスはまだ実装されていません。  
  
## <a name="remarks"></a>注釈  
  
> [!NOTE]
> このインターフェイスは、コンピューター間またはプロセス間でのリモート呼び出しをサポートしていません。
