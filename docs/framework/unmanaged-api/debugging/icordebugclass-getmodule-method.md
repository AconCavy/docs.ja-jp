# <a name="icordebugclassgetmodule-method"></a>ICorDebugClass::GetModule メソッド

このクラスを定義するモジュールを取得します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT GetModule (  
    [out] ICorDebugModule    **pModule  
);  
```  
  
## <a name="parameters"></a>パラメーター  

 `pModule`  
 入出力このクラスが定義されているモジュールを表す、モジュールオブジェクトのアドレスへのポインター。
