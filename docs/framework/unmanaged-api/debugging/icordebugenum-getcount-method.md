# <a name="icordebugenumgetcount-method"></a>ICorDebugEnum::GetCount メソッド
列挙に含まれる項目の数を取得します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT GetCount (  
    [out] ULONG *pcelt  
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `pcelt`  
 [out]列挙に含まれる項目数へのポインター。
