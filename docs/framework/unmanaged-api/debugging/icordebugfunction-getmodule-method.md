# <a name="icordebugfunctiongetmodule-method"></a>ICorDebugFunction::GetModule メソッド

この関数が定義されているモジュールを取得します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT GetModule (  
    [out] ICorDebugModule **ppModule  
);  
```  
  
## <a name="parameters"></a>パラメーター  

 `ppModule`  
 入出力この関数が定義されているモジュールを表す、モジュールオブジェクトのアドレスへのポインター。
