# <a name="policyimporters"></a>\<policyImporters>
この構成要素は、バインディングに関するカスタム ポリシー アサーションのインポートを制御するすべてのポリシー インポーターを指定します。 各子要素は、<`policyImporter`> アサーションが必要な機能を実装するカスタム バインディング要素の結びするほか、機能のバインドに関するカスタム ポリシー アサーションを検索に使用されます。
