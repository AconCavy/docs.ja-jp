## <a name="examples"></a><span data-ttu-id="9bf4a-208">使用例</span><span class="sxs-lookup"><span data-stu-id="9bf4a-208">Examples</span></span>  
 <span data-ttu-id="9bf4a-209">**Root\cimv2** 名前空間内の **Win32_LogicalDisk** WMI クラスに対してマネージド クラスを C# コードで生成するコマンドを次に示します。</span><span class="sxs-lookup"><span data-stu-id="9bf4a-209">The following command generates a managed class in C# code for the **Win32_LogicalDisk** WMI class in the **Root\cimv2** namespace.</span></span> <span data-ttu-id="9bf4a-210">このツールは、**ROOT.CIMV2.Win32** 名前空間内の c:\disk.cs にあるソース ファイルにマネージド クラスを書き込みます。</span><span class="sxs-lookup"><span data-stu-id="9bf4a-210">The tool writes the managed class to the source file at c:\disk.cs in the **ROOT.CIMV2.Win32** namespace.</span></span>  
  
```console  
mgmtclassgen Win32_LogicalDisk /n root\cimv2 /l CS /p c:\disk.cs  
```  
  
 <span data-ttu-id="9bf4a-211">生成したクラスをプログラムによって使用する方法を次のコード例に示します。</span><span class="sxs-lookup"><span data-stu-id="9bf4a-211">The following code example shows how to use a generated class programmatically.</span></span> <span data-ttu-id="9bf4a-212">まず、クラスのインスタンスが列挙され、パスが出力されます。</span><span class="sxs-lookup"><span data-stu-id="9bf4a-212">First, an instance of the class is enumerated and the path is printed.</span></span> <span data-ttu-id="9bf4a-213">次に、初期化の対象となる生成したクラスのインスタンスが、WMI のインスタンスを使用して作成されます。</span><span class="sxs-lookup"><span data-stu-id="9bf4a-213">Next, an instance of the generated class to be initialized is created with an instance of WMI.</span></span> <span data-ttu-id="9bf4a-214">**Root\cimv2** 名前空間において `Process` は **Win32_Process** に対して生成されたクラスであり、`LogicalDisk` は **Win32_LogicalDisk** に対して生成されたクラスです。</span><span class="sxs-lookup"><span data-stu-id="9bf4a-214">`Process` is the class generated for **Win32_Process** and `LogicalDisk` is the class generated for **Win32_LogicalDisk** in the **Root\cimv2** namespace.</span></span>  
  
```vb  
Imports System  
Imports System.Management  
Imports ROOT.CIMV2.Win32  
  
Public Class App     
   Public Shared Sub Main()        
      ' Enumerate instances of the Win32_process.  
      ' Print the Name property of the instance.  
      Dim ps As Process     
      For Each ps In  Process.GetInstances()  
         Console.WriteLine(ps.Name)  
      Next ps  
  
      ' Initialize the instance of LogicalDisk with  
      ' the WMI instance pointing to logical drive d:.  
      Dim dskD As New LogicalDisk(New _  
         ManagementPath("win32_LogicalDisk.DeviceId=""d:"""))  
      Console.WriteLine(dskD.Caption)  
   End Sub  
End Class  
```  
  
```csharp  
using System;  
using System.Management;  
using ROOT.CIMV2.Win32;  
  
public class App  
{  
   public static void Main()  
   {  
      // Enumerate instances of the Win32_process.  
      // Print the Name property of the instance.  
      foreach(Process ps in Process.GetInstances())  
      {  
         Console.WriteLine(ps.Name);  
      }  
  
      // Initialize the instance of LogicalDisk with  
      // the WMI instance pointing to logical drive d:.  
      LogicalDisk dskD = new LogicalDisk(new ManagementPath(  
        "win32_LogicalDisk.DeviceId=\"d:\""));  
      Console.WriteLine(dskD.Caption);  
   }  
}  
```
