 <span data-ttu-id="d95b5-301">Resgen.exe は厳密に型指定されたリソースをサポートします。このサポートでは、静的な読み取り専用プロパティを持つクラスを作成して、リソースへのアクセスをカプセル化します。</span><span class="sxs-lookup"><span data-stu-id="d95b5-301">Resgen.exe supports strongly typed resources, which encapsulates access to resources by creating classes that contain a set of static read-only properties.</span></span> <span data-ttu-id="d95b5-302">これによって、リソースを取得するために <xref:System.Resources.ResourceManager> クラスのメソッドを直接呼び出す方法の代替手段が可能になります。</span><span class="sxs-lookup"><span data-stu-id="d95b5-302">This provides an alternative to calling the methods of the <xref:System.Resources.ResourceManager> class directly to retrieve resources.</span></span> <span data-ttu-id="d95b5-303">`/str` クラスの機能をラップする、Resgen.exe の <xref:System.Resources.Tools.StronglyTypedResourceBuilder> オプションを使用して、厳密に型指定されたリソース サポートを有効にすることができます。</span><span class="sxs-lookup"><span data-stu-id="d95b5-303">You can enable strongly typed resource support by using the `/str` option in Resgen.exe, which wraps the functionality of the <xref:System.Resources.Tools.StronglyTypedResourceBuilder> class.</span></span> <span data-ttu-id="d95b5-304">`/str` オプションを指定した場合の Resgen.exe の出力は、入力パラメーターで参照されるリソースと一致する厳密に型指定されたプロパティを含むクラスです。</span><span class="sxs-lookup"><span data-stu-id="d95b5-304">When you specify the `/str` option, the output of Resgen.exe is a class that contains strongly typed properties that match the resources that are referenced in the input parameter.</span></span> <span data-ttu-id="d95b5-305">このクラスは、処理されたファイルで使用できるリソースに対する、厳密に型指定された読み取り専用アクセスを提供します。</span><span class="sxs-lookup"><span data-stu-id="d95b5-305">This class provides strongly typed read-only access to the resources that are available in the file processed.</span></span>

 <span data-ttu-id="d95b5-325">*classname*</span><span class="sxs-lookup"><span data-stu-id="d95b5-325">*classname*</span></span>

- <span data-ttu-id="d95b5-345">1 つの `static` (C#) または `Shared` (Visual Basic)、および .resources ファイルのリソースごとの読み取り専用プロパティ。</span><span class="sxs-lookup"><span data-stu-id="d95b5-345">One `static` (C#) or `Shared` (Visual Basic) and read-only property for each resource in the .resources file.</span></span> <span data-ttu-id="d95b5-346">プロパティの名前はリソースの名前です。</span><span class="sxs-lookup"><span data-stu-id="d95b5-346">The name of the property is the name of the resource.-</span></span>
