|||  
|-|-|  
|ID|1036|  
|キーワード|WFRuntime|  
|レベル|詳細|  
|チャネル|Microsoft-Windows-Application Server-Applications/Debug|

 アクティビティがランタイムのトランザクションの完了をスケジュールしたことを示します。

|DisplayName|xs:string|アクティビティの表示名。|  
|InstanceId|xs:string|アクティビティのインスタンス ID。|
