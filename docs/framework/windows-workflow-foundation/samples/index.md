このセクションの記事では、ダウンロードパッケージのいくつかのサンプルについて説明します。 すべてのサンプルを網羅した完全なドキュメントセットについては、 [.NET Framework 4 WF サンプル](/previous-versions/dotnet/netframework-4.0/dd483375(v=vs.100)) を参照してください。

> [!NOTE]
