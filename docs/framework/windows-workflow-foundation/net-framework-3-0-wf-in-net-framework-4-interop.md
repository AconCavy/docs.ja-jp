 <span data-ttu-id="a1b6a-108"><xref:System.Activities.Statements.Interop> アクティビティ内で WF 3 アクティビティを正常に実行するには、次の基準を満たす必要があります。</span><span class="sxs-lookup"><span data-stu-id="a1b6a-108">For a WF 3 activity to successfully execute within an <xref:System.Activities.Statements.Interop> activity, the following criteria must be met:</span></span>
