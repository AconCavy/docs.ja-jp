# <a name="wcf-and-net-framework-client-profile"></a>WCF および .NET Framework のクライアント プロファイル
.NET framework Client Profile は、全体のフレームワークを必要としないクライアント用に設計された完全な .NET Framework の軽量バージョンです。 クライアント フレームワークでは、Windows Communication Foundation のすべてがサポートされるわけではありません。  
  
## <a name="wcf-features-supported-by-the-net-framework-client-profile"></a>.NET Framework Client Profile でサポートされている WCF の機能  
 .NET Framework Client Profile では、次の Windows Communication Foundation の機能がサポートされています。  
  
- Cardspace と Web ホスティングを除くすべての WCF がサポートされます。  
  
- TCP/IP チャネルのリモート処理がサポートされます。  
  
- Asmx (Web サービス) はサポートされません。
