# <a name="wcf-system-requirements"></a>WCF システム要件
Windows Communication Foundation (WCF) は、分散アプリケーションを作成するために使用される通信インフラストラクチャです。 WCF 実行を有効にする要件を次に示します。  
  
## <a name="system-requirements"></a>システム要件  
 WCF は既定でインストール[!INCLUDE[wv](../../../includes/wv-md.md)]します。  
  
 WCF をインストールすることもできます。 [!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)]、 [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)]、または[!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)]します。  
  
 **注**WCF の機能をメッセージ キュー (MSMQ) でのみサポートされます[!INCLUDE[wv](../../../includes/wv-md.md)]、 [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)]、 [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)]、および[!INCLUDE[wxppro](../../../includes/wxppro-md.md)]します。
