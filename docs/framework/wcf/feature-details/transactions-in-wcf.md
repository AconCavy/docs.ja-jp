# <a name="transactions-in-wcf"></a>WCF でのトランザクション

Windows Communication Foundation (WCF) トランザクション機能を使用すると、複数のコンポーネントを同時にデータ操作に参加させることができます。 このセクションのトピックでは、この機能について詳しく説明します。
