## <a name="exception-list"></a>例外の一覧  
  
|リソース コード|リソースの文字列|  
|-------------------|---------------------|  
|Hosting_TransportBindingNotFound|指定したアドレスに一致するプロトコル バインディングがありません。 プロトコル バインディングは、インターネット インフォメーション サービスまたは Windows プロセス アクティブ化サービスの構成でサイト レベルで構成されます。|
