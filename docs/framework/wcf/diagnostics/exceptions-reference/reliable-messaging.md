## <a name="exception-list"></a>例外の一覧  
  
|リソース コード|リソースの文字列|  
|-------------------|---------------------|  
|InvalidBufferRemaining|バッファーの残りサイズが指定値に達したというメッセージが受信されました。 残りのバッファーをゼロ未満にすることはできません。|
