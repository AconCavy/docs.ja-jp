# <a name="systemservicemodeltxcompletionstatuscompletedforautocomplete"></a>System.ServiceModel.TxCompletionStatusCompletedForAutocomplete

`OperationBehaviorAttribute.TransactionAutoComplete` メンバーが `true` に設定されたため、指定された操作の指定されたトランザクションが完了しました。  
  
## <a name="description"></a>Description  

 ユーザーに代わって、インフラストラクチャによってトランザクションが自動的に完了したときにトレースされます。
