# <a name="endpoint-calls-failed"></a>エンドポイント : 失敗した呼び出し

カウンター名 : 失敗した呼び出し。  
  
## <a name="description"></a>Description  

 このエンドポイントで処理できない例外が発生した呼び出しの数です。 このカウンターは、このエンドポイントで未処理の例外が発生すると常にインクリメントされます。
