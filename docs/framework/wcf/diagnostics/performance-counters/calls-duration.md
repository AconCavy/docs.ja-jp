# <a name="calls-duration"></a>呼び出す期間
カウンター名:呼び出す期間  
  
## <a name="description"></a>説明  
 この操作の呼び出しの平均期間です。 平均時間は、次の式に基づいて計算されます。(N1-N0)/(D1-D0)。  
  
> [!WARNING]
> 非同期 WCF サービスで使用する場合、呼び出し期間カウンターは常に-1 を返します。
