# <a name="wmiunregistrationfailed"></a>WmiUnregistrationFailed

Id: 127  
  
 重大度 : エラー  
  
 カテゴリ : ServiceModel  
  
## <a name="description"></a>Description  

 このイベントは、WMI プロバイダーが登録解除されなかったことを示します。 イベントには、WMI オブジェクト、エラー、プロセス名、およびプロセス ID が表示されます。
