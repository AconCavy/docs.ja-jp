# <a name="bindingerror"></a>BindingError
Id:159  
  
 順Error  
  
 [カテゴリ]:Smsvchost.exe  
  
## <a name="description"></a>説明  
 このイベントは、特定のサイトの特定のバインディングの解析中にエラーが発生したため、このサイトのプロトコルが一時的に無効になっていることを示します。 イベントには、バインディング、ソース、例外、プロセス名、およびプロセス ID が表示されます。
