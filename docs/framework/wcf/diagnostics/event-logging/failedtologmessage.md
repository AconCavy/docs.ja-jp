# <a name="failedtologmessage"></a>FailedToLogMessage

Id: 113  
  
 重大度 : エラー  
  
 カテゴリ : ServiceModel  
  
## <a name="description"></a>Description  

 このイベントは、メッセージがログに記録されなかったことを示します。 イベントには、例外、プロセス名、およびプロセス ID が表示されます。
