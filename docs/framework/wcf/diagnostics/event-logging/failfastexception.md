# <a name="failfastexception"></a>FailFastException

ID : 103  
  
 重大度 : エラー  
  
 カテゴリ : トレース  
  
## <a name="description"></a>Description  

 FailFast 中に例外がスローされました。 イベントには、例外、プロセス名、およびプロセス ID が表示されます。
