# <a name="piiloggingon"></a>PiiLoggingOn
Id:107  
  
 順Error  
  
 [カテゴリ]:トレース  
  
## <a name="description"></a>説明  
 このイベントは、PII ログ記録が有効になっていることを示します。 重要な情報がログに記録されます。 イベントには、プロセス名とプロセス ID が表示されます。
