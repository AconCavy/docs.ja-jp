# <a name="wasconnectiontimedout"></a>WasConnectionTimedout
Id:163  
  
 重大度:Error  
  
 カテゴリ:SMSvcHost  
  
## <a name="description"></a>説明  
 このイベントは、Windows プロセス アクティブ化サービス (WAS) で必要な接続がタイムアウトになったことを示し、プロセス名とプロセス ID を表示します。
