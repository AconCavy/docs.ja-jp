|-|-|

|キーワード|RoutingServices|  
|レベル|情報|

## <a name="description"></a>説明  
 このイベントは、ルーティング サービスがメッセージを処理しているときに生成されます。  
  
## <a name="message"></a>メッセージ  
 ルーティング サービスが、トランザクション: %4 で受信された ID: '%1'、アクション: '%2'、着信 URL: '%3' のメッセージを処理しています。
