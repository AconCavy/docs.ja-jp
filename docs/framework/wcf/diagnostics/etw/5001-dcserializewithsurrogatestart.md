|-|-|  
|ID|5001|  
|Keywords|シリアル化|

## <a name="description"></a>Description  

 このイベントは、DataContract のサロゲートによるシリアル化を開始すると生成されます。  
  
## <a name="message"></a>Message  

 DataContract のサロゲートによる %1 のシリアル化を開始します。
