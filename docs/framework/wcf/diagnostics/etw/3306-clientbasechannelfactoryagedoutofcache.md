## <a name="description"></a>Description  

 このイベントは、キャッシュがその上限に達したため、チャネル ファクトリがキャッシュから削除されたときに生成されます。  
  
## <a name="message"></a>Message  

 キャッシュが上限の '%1' に達したため、チャネル ファクトリがキャッシュから削除されました。
