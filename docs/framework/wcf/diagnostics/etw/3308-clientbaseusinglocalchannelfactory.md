|キーワード|ServiceModel|

## <a name="description"></a>説明  
 このイベントは、キャッシュのチャネル ファクトリを使用しないときに生成されます。  
  
## <a name="message"></a>[メッセージ]  
 キャッシュのチャネル ファクトリは使用されません (つまり、インスタンスのキャッシュは無効になっています)。
