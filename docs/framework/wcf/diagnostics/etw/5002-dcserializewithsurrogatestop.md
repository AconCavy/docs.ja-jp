|||  
|-|-|  
|ID|5002|  
|キーワード|シリアル化|  
|レベル|詳細|

## <a name="description"></a>説明  
 このイベントは、DataContract のサロゲートによるシリアル化を停止すると生成されます。  
  
## <a name="message"></a>メッセージ
