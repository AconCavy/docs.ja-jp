|-|-|

|Keywords|ActivationServices|  
|Level|エラー|

## <a name="description"></a>Description  

 このイベントは、SharedListenerProxy の登録が失敗すると生成されます。  
  
## <a name="message"></a>Message  

 SharedListenerProxy の登録は状態 '%1' で失敗しました。
