# <a name="bitmap-effects-overview"></a><span data-ttu-id="24099-102">ビットマップ効果の概要</span><span class="sxs-lookup"><span data-stu-id="24099-102">Bitmap Effects Overview</span></span>
<span data-ttu-id="24099-103">ビットマップ効果を使用すると、設計者と開発者は、レンダリングされる Windows Presentation Foundation (WPF) コンテンツに視覚効果を適用できます。</span><span class="sxs-lookup"><span data-stu-id="24099-103">Bitmap effects enable designers and developers to apply visual effects to rendered Windows Presentation Foundation (WPF) content.</span></span> <span data-ttu-id="24099-104">たとえば、ビットマップ効果では、<xref:System.Windows.Media.Effects.DropShadowBitmapEffect> 効果またはぼかし効果をイメージやボタンに簡単に適用できます。</span><span class="sxs-lookup"><span data-stu-id="24099-104">For example, bitmap effects allow you to easily apply a <xref:System.Windows.Media.Effects.DropShadowBitmapEffect> effect or a blur effect to an image or a button.</span></span>  
  
> [!IMPORTANT]
> <span data-ttu-id="24099-105">.NET Framework 4 以降では、<xref:System.Windows.Media.Effects.BitmapEffect> クラスが廃止されています。</span><span class="sxs-lookup"><span data-stu-id="24099-105">In the .NET Framework 4 or later, the <xref:System.Windows.Media.Effects.BitmapEffect> class is obsolete.</span></span> <span data-ttu-id="24099-106"><xref:System.Windows.Media.Effects.BitmapEffect> クラスを使用しようとすると、廃止に関する例外が発生します。</span><span class="sxs-lookup"><span data-stu-id="24099-106">If you try to use the <xref:System.Windows.Media.Effects.BitmapEffect> class, you will get an obsolete exception.</span></span> <span data-ttu-id="24099-107"><xref:System.Windows.Media.Effects.BitmapEffect> クラスの廃止されていない代替クラスは、<xref:System.Windows.Media.Effects.Effect> クラスです。</span><span class="sxs-lookup"><span data-stu-id="24099-107">The non-obsolete alternative to the <xref:System.Windows.Media.Effects.BitmapEffect> class is the <xref:System.Windows.Media.Effects.Effect> class.</span></span> <span data-ttu-id="24099-108">ほとんどの場合、<xref:System.Windows.Media.Effects.Effect> クラスの方がはるかに高速です。</span><span class="sxs-lookup"><span data-stu-id="24099-108">In most situations, the <xref:System.Windows.Media.Effects.Effect> class is significantly faster.</span></span>

## <a name="wpf-bitmap-effects"></a><span data-ttu-id="24099-109">WPF のビットマップ効果</span><span class="sxs-lookup"><span data-stu-id="24099-109">WPF Bitmap Effects</span></span>  
 <span data-ttu-id="24099-110">ビットマップ効果 (<xref:System.Windows.Media.Effects.BitmapEffect> オブジェクト) は、簡単なピクセル処理操作です。</span><span class="sxs-lookup"><span data-stu-id="24099-110">Bitmap effects (<xref:System.Windows.Media.Effects.BitmapEffect> object) are simple pixel processing operations.</span></span> <span data-ttu-id="24099-111">ビットマップ効果は、入力として <xref:System.Windows.Media.Imaging.BitmapSource> を受け取り、効果 (ぼかしやドロップ シャドウなど) を適用した後で、新しい <xref:System.Windows.Media.Imaging.BitmapSource> を生成します。</span><span class="sxs-lookup"><span data-stu-id="24099-111">A bitmap effect takes a <xref:System.Windows.Media.Imaging.BitmapSource> as an input and produces a new <xref:System.Windows.Media.Imaging.BitmapSource> after applying the effect, such as a blur or drop shadow.</span></span> <span data-ttu-id="24099-112">各ビットマップ効果では、フィルター処理プロパティを制御できるプロパティが公開されています (<xref:System.Windows.Media.Effects.BlurBitmapEffect> の <xref:System.Windows.Media.Effects.BlurBitmapEffect.Radius%2A> など)。</span><span class="sxs-lookup"><span data-stu-id="24099-112">Each bitmap effect exposes properties that can control the filtering properties, such as <xref:System.Windows.Media.Effects.BlurBitmapEffect.Radius%2A> of <xref:System.Windows.Media.Effects.BlurBitmapEffect>.</span></span>  
  
 <span data-ttu-id="24099-113">特殊なケースとして、[!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] では、効果をライブ <xref:System.Windows.Media.Visual> オブジェクトのプロパティとして設定できます (<xref:System.Windows.Controls.Button> や <xref:System.Windows.Controls.TextBox> など)。</span><span class="sxs-lookup"><span data-stu-id="24099-113">As a special case, in [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)], effects can be set as properties on live <xref:System.Windows.Media.Visual> objects, such as a <xref:System.Windows.Controls.Button> or <xref:System.Windows.Controls.TextBox>.</span></span> <span data-ttu-id="24099-114">ピクセル処理は、実行時に適用されてレンダリングされます。</span><span class="sxs-lookup"><span data-stu-id="24099-114">The pixel processing is applied and rendered at run-time.</span></span> <span data-ttu-id="24099-115">この場合、レンダリング時に、<xref:System.Windows.Media.Visual> が同等の <xref:System.Windows.Media.Imaging.BitmapSource> に自動的に変換され、<xref:System.Windows.Media.Effects.BitmapEffect> への入力としてフィードされます。</span><span class="sxs-lookup"><span data-stu-id="24099-115">In this case, at the time of rendering, a <xref:System.Windows.Media.Visual> is automatically converted to its <xref:System.Windows.Media.Imaging.BitmapSource> equivalent and is fed as input to the <xref:System.Windows.Media.Effects.BitmapEffect>.</span></span> <span data-ttu-id="24099-116">出力は、<xref:System.Windows.Media.Visual> オブジェクトの既定のレンダリング動作を置き換えます。</span><span class="sxs-lookup"><span data-stu-id="24099-116">The output replaces the <xref:System.Windows.Media.Visual> object's default rendering behavior.</span></span> <span data-ttu-id="24099-117">このため、<xref:System.Windows.Media.Effects.BitmapEffect> オブジェクトでは、ビジュアル オブジェクトがソフトウェアのみでレンダリングされるよう強制されます。つまり、効果の適用時にハードウェア アクセラレーションは利用されません。</span><span class="sxs-lookup"><span data-stu-id="24099-117">This is why <xref:System.Windows.Media.Effects.BitmapEffect> objects force visuals to render in software only i.e. no hardware acceleration on visuals when effects are applied.</span></span>  
  
- <span data-ttu-id="24099-118"><xref:System.Windows.Media.Effects.BlurBitmapEffect> は、オブジェクトのピントが外れた表示をシミュレートします。</span><span class="sxs-lookup"><span data-stu-id="24099-118"><xref:System.Windows.Media.Effects.BlurBitmapEffect> simulates an object that appears out-of-focus.</span></span>  
  
- <span data-ttu-id="24099-119"><xref:System.Windows.Media.Effects.OuterGlowBitmapEffect> は、オブジェクトの周囲に色の付いた光輪を作成します。</span><span class="sxs-lookup"><span data-stu-id="24099-119"><xref:System.Windows.Media.Effects.OuterGlowBitmapEffect> creates a halo of color around the perimeter of an object.</span></span>  
  
- <span data-ttu-id="24099-120"><xref:System.Windows.Media.Effects.DropShadowBitmapEffect> は、オブジェクトの後ろに影を作成します。</span><span class="sxs-lookup"><span data-stu-id="24099-120"><xref:System.Windows.Media.Effects.DropShadowBitmapEffect> creates a shadow behind an object.</span></span>  
  
- <span data-ttu-id="24099-121"><xref:System.Windows.Media.Effects.BevelBitmapEffect> は、指定された曲線に従ってイメージの表面を浮き上がらせるベベルを作成します。</span><span class="sxs-lookup"><span data-stu-id="24099-121"><xref:System.Windows.Media.Effects.BevelBitmapEffect> creates a bevel which raises the surface of an image according to a specified curve.</span></span>  
  
- <span data-ttu-id="24099-122"><xref:System.Windows.Media.Effects.EmbossBitmapEffect> は、人工光源からの奥行きと質感の印象を与えるために、<xref:System.Windows.Media.Visual> のバンプ マッピングを作成します。</span><span class="sxs-lookup"><span data-stu-id="24099-122"><xref:System.Windows.Media.Effects.EmbossBitmapEffect> creates a bump mapping of a <xref:System.Windows.Media.Visual> to give the impression of depth and texture from an artificial light source.</span></span>  
  
> [!NOTE]
> <span data-ttu-id="24099-123">WPF のビットマップ効果は、ソフトウェア モードでレンダリングされます。</span><span class="sxs-lookup"><span data-stu-id="24099-123">WPF bitmap effects are rendered in software mode.</span></span> <span data-ttu-id="24099-124">効果を適用するオブジェクトも、ソフトウェアでレンダリングされます。</span><span class="sxs-lookup"><span data-stu-id="24099-124">Any object that applies an effect will also be rendered in software.</span></span> <span data-ttu-id="24099-125">大きなビジュアルにビットマップ効果を使うとき、またはビットマップ効果のプロパティをアニメーション化するときに、パフォーマンスが最も低下します。</span><span class="sxs-lookup"><span data-stu-id="24099-125">Performance is degraded the most when using Bitmap effects on large visuals or animating properties of a Bitmap effect.</span></span> <span data-ttu-id="24099-126">このような方法ではビットマップ効果をまったく使ってはならないということではありませんが、注意を払い、十分にテストを行って、ユーザー エクスペリエンスが期待どおりになることを確認する必要があります。</span><span class="sxs-lookup"><span data-stu-id="24099-126">This is not to say that you should not use Bitmap effects in this way at all, but you should use caution and test thoroughly to ensure that your users are getting the experience you expect.</span></span>  
  
> [!NOTE]
> <span data-ttu-id="24099-127">WPF のビットマップ効果は、部分信頼の実行をサポートしていません。</span><span class="sxs-lookup"><span data-stu-id="24099-127">WPF bitmap effects do not support partial trust execution.</span></span> <span data-ttu-id="24099-128">ビットマップ効果を使うには、アプリケーションが完全な信頼のアクセス許可を持つ必要があります。</span><span class="sxs-lookup"><span data-stu-id="24099-128">An application must have full trust permissions to use bitmap effects.</span></span>
