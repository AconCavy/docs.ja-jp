# <a name="bitmap-effects-overview"></a><span data-ttu-id="df522-102">ビットマップ効果の概要</span><span class="sxs-lookup"><span data-stu-id="df522-102">Bitmap Effects Overview</span></span>
<span data-ttu-id="df522-103">ビットマップ効果を使用すると、デザイナーと開発者は、レンダリングされた Windows Presentation Foundation (WPF) コンテンツに視覚効果を適用できます。</span><span class="sxs-lookup"><span data-stu-id="df522-103">Bitmap effects enable designers and developers to apply visual effects to rendered Windows Presentation Foundation (WPF) content.</span></span> <span data-ttu-id="df522-104">たとえば、ビットマップ効果を使用すると、イメージまたはボタンに <xref:System.Windows.Media.Effects.DropShadowBitmapEffect> 効果またはぼかし効果を簡単に適用できます。</span><span class="sxs-lookup"><span data-stu-id="df522-104">For example, bitmap effects allow you to easily apply a <xref:System.Windows.Media.Effects.DropShadowBitmapEffect> effect or a blur effect to an image or a button.</span></span>

## <a name="wpf-bitmap-effects"></a><span data-ttu-id="df522-109">WPF のビットマップ効果</span><span class="sxs-lookup"><span data-stu-id="df522-109">WPF Bitmap Effects</span></span>  
 <span data-ttu-id="df522-110">ビットマップ効果 (<xref:System.Windows.Media.Effects.BitmapEffect> オブジェクト) は、単純なピクセル処理操作です。</span><span class="sxs-lookup"><span data-stu-id="df522-110">Bitmap effects (<xref:System.Windows.Media.Effects.BitmapEffect> object) are simple pixel processing operations.</span></span> <span data-ttu-id="df522-111">ビットマップ効果は、入力として <xref:System.Windows.Media.Imaging.BitmapSource> を受け取り、ぼかしやドロップシャドウなどの効果を適用した後に新しい <xref:System.Windows.Media.Imaging.BitmapSource> を生成します。</span><span class="sxs-lookup"><span data-stu-id="df522-111">A bitmap effect takes a <xref:System.Windows.Media.Imaging.BitmapSource> as an input and produces a new <xref:System.Windows.Media.Imaging.BitmapSource> after applying the effect, such as a blur or drop shadow.</span></span> <span data-ttu-id="df522-112">各ビットマップ効果は、<xref:System.Windows.Media.Effects.BlurBitmapEffect>の <xref:System.Windows.Media.Effects.BlurBitmapEffect.Radius%2A> など、フィルターのプロパティを制御できるプロパティを公開します。</span><span class="sxs-lookup"><span data-stu-id="df522-112">Each bitmap effect exposes properties that can control the filtering properties, such as <xref:System.Windows.Media.Effects.BlurBitmapEffect.Radius%2A> of <xref:System.Windows.Media.Effects.BlurBitmapEffect>.</span></span>  
  
 <span data-ttu-id="df522-113">特殊なケースとして、[!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)]では、<xref:System.Windows.Controls.Button> や <xref:System.Windows.Controls.TextBox>など、ライブ <xref:System.Windows.Media.Visual> オブジェクトのプロパティとして効果を設定できます。</span><span class="sxs-lookup"><span data-stu-id="df522-113">As a special case, in [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)], effects can be set as properties on live <xref:System.Windows.Media.Visual> objects, such as a <xref:System.Windows.Controls.Button> or <xref:System.Windows.Controls.TextBox>.</span></span> <span data-ttu-id="df522-114">ピクセル処理は、実行時に適用されてレンダリングされます。</span><span class="sxs-lookup"><span data-stu-id="df522-114">The pixel processing is applied and rendered at run-time.</span></span> <span data-ttu-id="df522-115">この場合、レンダリング時には、<xref:System.Windows.Media.Visual> が自動的に同等の <xref:System.Windows.Media.Imaging.BitmapSource> に変換され、<xref:System.Windows.Media.Effects.BitmapEffect>に入力として渡されます。</span><span class="sxs-lookup"><span data-stu-id="df522-115">In this case, at the time of rendering, a <xref:System.Windows.Media.Visual> is automatically converted to its <xref:System.Windows.Media.Imaging.BitmapSource> equivalent and is fed as input to the <xref:System.Windows.Media.Effects.BitmapEffect>.</span></span> <span data-ttu-id="df522-116">出力は、<xref:System.Windows.Media.Visual> オブジェクトの既定の表示動作に置き換わるものです。</span><span class="sxs-lookup"><span data-stu-id="df522-116">The output replaces the <xref:System.Windows.Media.Visual> object's default rendering behavior.</span></span> <span data-ttu-id="df522-117">このため、<xref:System.Windows.Media.Effects.BitmapEffect> オブジェクトは、視覚エフェクトが適用されている場合に、ビジュアルにハードウェアアクセラレータが適用されないようにするだけで、ビジュアルをソフトウェアでレンダリングすることになります。</span><span class="sxs-lookup"><span data-stu-id="df522-117">This is why <xref:System.Windows.Media.Effects.BitmapEffect> objects force visuals to render in software only i.e. no hardware acceleration on visuals when effects are applied.</span></span>  
  
- <span data-ttu-id="df522-118"><xref:System.Windows.Media.Effects.BlurBitmapEffect> は、フォーカスがないオブジェクトをシミュレートします。</span><span class="sxs-lookup"><span data-stu-id="df522-118"><xref:System.Windows.Media.Effects.BlurBitmapEffect> simulates an object that appears out-of-focus.</span></span>  
  
- <span data-ttu-id="df522-119"><xref:System.Windows.Media.Effects.OuterGlowBitmapEffect> は、オブジェクトの周囲に色のハローを作成します。</span><span class="sxs-lookup"><span data-stu-id="df522-119"><xref:System.Windows.Media.Effects.OuterGlowBitmapEffect> creates a halo of color around the perimeter of an object.</span></span>  
  
- <span data-ttu-id="df522-120"><xref:System.Windows.Media.Effects.DropShadowBitmapEffect> は、オブジェクトの背後に影を作成します。</span><span class="sxs-lookup"><span data-stu-id="df522-120"><xref:System.Windows.Media.Effects.DropShadowBitmapEffect> creates a shadow behind an object.</span></span>  
  
- <span data-ttu-id="df522-121"><xref:System.Windows.Media.Effects.BevelBitmapEffect> は、指定された曲線に従ってイメージの表面を生成するベベルを作成します。</span><span class="sxs-lookup"><span data-stu-id="df522-121"><xref:System.Windows.Media.Effects.BevelBitmapEffect> creates a bevel which raises the surface of an image according to a specified curve.</span></span>  
  
- <span data-ttu-id="df522-122"><xref:System.Windows.Media.Effects.EmbossBitmapEffect> は <xref:System.Windows.Media.Visual> のバンプマッピングを作成して、人工光源の奥行とテクスチャの印象を与えます。</span><span class="sxs-lookup"><span data-stu-id="df522-122"><xref:System.Windows.Media.Effects.EmbossBitmapEffect> creates a bump mapping of a <xref:System.Windows.Media.Visual> to give the impression of depth and texture from an artificial light source.</span></span>  
  
> [!NOTE]
> <span data-ttu-id="df522-123">WPF ビットマップ効果は、ソフトウェアモードで表示されます。</span><span class="sxs-lookup"><span data-stu-id="df522-123">WPF bitmap effects are rendered in software mode.</span></span> <span data-ttu-id="df522-124">効果を適用するオブジェクトも、ソフトウェアでレンダリングされます。</span><span class="sxs-lookup"><span data-stu-id="df522-124">Any object that applies an effect will also be rendered in software.</span></span> <span data-ttu-id="df522-125">大きなビジュアルにビットマップ効果を使うとき、またはビットマップ効果のプロパティをアニメーション化するときに、パフォーマンスが最も低下します。</span><span class="sxs-lookup"><span data-stu-id="df522-125">Performance is degraded the most when using Bitmap effects on large visuals or animating properties of a Bitmap effect.</span></span> <span data-ttu-id="df522-126">このような方法ではビットマップ効果をまったく使ってはならないということではありませんが、注意を払い、十分にテストを行って、ユーザー エクスペリエンスが期待どおりになることを確認する必要があります。</span><span class="sxs-lookup"><span data-stu-id="df522-126">This is not to say that you should not use Bitmap effects in this way at all, but you should use caution and test thoroughly to ensure that your users are getting the experience you expect.</span></span>  
  
> [!NOTE]
> <span data-ttu-id="df522-127">WPF ビットマップ効果では、部分信頼実行はサポートされていません。</span><span class="sxs-lookup"><span data-stu-id="df522-127">WPF bitmap effects do not support partial trust execution.</span></span> <span data-ttu-id="df522-128">ビットマップ効果を使うには、アプリケーションが完全な信頼のアクセス許可を持つ必要があります。</span><span class="sxs-lookup"><span data-stu-id="df522-128">An application must have full trust permissions to use bitmap effects.</span></span>
