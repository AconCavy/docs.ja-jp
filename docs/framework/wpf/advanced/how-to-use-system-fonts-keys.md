# <a name="how-to-use-system-fonts-keys"></a><span data-ttu-id="70a92-102">方法: システム フォント キーを使用する</span><span class="sxs-lookup"><span data-stu-id="70a92-102">How to: Use System Fonts Keys</span></span>
<span data-ttu-id="70a92-103">システム リソースは、開発者がシステム設定と一貫性のあるビジュアルを作成できるようにするために、多くのシステム メトリックをリソースとして公開します。</span><span class="sxs-lookup"><span data-stu-id="70a92-103">System resources expose a number of system metrics as resources to help developers create visuals that are consistent with system settings.</span></span> <span data-ttu-id="70a92-104"><xref:System.Windows.SystemFonts> は、システム フォント値および値にバインドされるシステム フォント リソースの両方を含むクラスです (<xref:System.Windows.SystemFonts.CaptionFontFamily%2A>、<xref:System.Windows.SystemFonts.CaptionFontFamilyKey%2A> など)。</span><span class="sxs-lookup"><span data-stu-id="70a92-104"><xref:System.Windows.SystemFonts> is a class that contains both system font values and system font resources that bind to the values—for example, <xref:System.Windows.SystemFonts.CaptionFontFamily%2A> and <xref:System.Windows.SystemFonts.CaptionFontFamilyKey%2A>.</span></span>  
  
 <span data-ttu-id="70a92-105">システム フォント メトリックは、静的なリソースとしても、動的なリソースとしても使用できます。</span><span class="sxs-lookup"><span data-stu-id="70a92-105">System font metrics can be used as either static or dynamic resources.</span></span> <span data-ttu-id="70a92-106">アプリケーションの実行時にフォント メトリックを自動的に更新する場合は、動的リソースを使用します。それ以外の場合は、静的リソースを使用します。</span><span class="sxs-lookup"><span data-stu-id="70a92-106">Use a dynamic resource if you want the font metric to update automatically while the application runs; otherwise use a static resource.</span></span>  
  
> [!NOTE]
> <span data-ttu-id="70a92-107">動的リソースのプロパティ名には、キーワード *Key* が追加されています。</span><span class="sxs-lookup"><span data-stu-id="70a92-107">Dynamic resources have the keyword *Key* appended to the property name.</span></span>  
  
 <span data-ttu-id="70a92-108">次の例では、システム フォント動的リソースにアクセスして使用し、ボタンのスタイル設定またはカスタマイズを行う方法を示します。</span><span class="sxs-lookup"><span data-stu-id="70a92-108">The following example shows how to access and use system font dynamic resources to style or customize a button.</span></span> <span data-ttu-id="70a92-109">この [!INCLUDE[TLA2#tla_xaml](../../../../includes/tla2sharptla-xaml-md.md)] の例では、ボタンに <xref:System.Windows.SystemFonts> の値を割り当てるボタン スタイルを作成します。</span><span class="sxs-lookup"><span data-stu-id="70a92-109">This [!INCLUDE[TLA2#tla_xaml](../../../../includes/tla2sharptla-xaml-md.md)] example creates a button style that assigns <xref:System.Windows.SystemFonts> values to a button.</span></span>
