## <a name="writing-message-loops"></a><span data-ttu-id="711c6-115">メッセージループの作成</span><span class="sxs-lookup"><span data-stu-id="711c6-115">Writing Message Loops</span></span>

## <a name="using-componentdispatcher-and-existing-message-handling"></a><span data-ttu-id="711c6-125">ComponentDispatcher と既存のメッセージ処理の使用</span><span class="sxs-lookup"><span data-stu-id="711c6-125">Using ComponentDispatcher and Existing Message Handling</span></span>
