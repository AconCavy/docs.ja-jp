<span data-ttu-id="23658-103">Windows フォームと [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] テクノロジには、似ていますが異なる 2 つのプロパティ モデルがあります。</span><span class="sxs-lookup"><span data-stu-id="23658-103">The Windows Forms and [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] technologies have two similar but different property models.</span></span> <span data-ttu-id="23658-104">"*プロパティ マッピング*" は、2 つのアーキテクチャ間の相互運用をサポートし、次の機能を提供しています。</span><span class="sxs-lookup"><span data-stu-id="23658-104">*Property mapping* supports interoperation between the two architectures and provides the following capabilities:</span></span>  
  
- <span data-ttu-id="23658-105">ホスト環境内の関連するプロパティの変更を、ホストされているコントロールまたは要素に簡単にマップすることができます。</span><span class="sxs-lookup"><span data-stu-id="23658-105">Makes it easy to map relevant property changes in the host environment to the hosted control or element.</span></span>  
  
- <span data-ttu-id="23658-106">よく使用されるプロパティ マッピングには既定の処理が用意されています。</span><span class="sxs-lookup"><span data-stu-id="23658-106">Provides default handling for mapping the most commonly used properties.</span></span>  
  
- <span data-ttu-id="23658-107">既定のプロパティを簡単に削除、上書き、または拡張できます。</span><span class="sxs-lookup"><span data-stu-id="23658-107">Allows easy removal, overriding, or extending of default properties.</span></span>  
  
- <span data-ttu-id="23658-108">ホスト上のプロパティ値の変更が自動的に確実に検出され、ホストされているコントロールまたは要素に変換されます。</span><span class="sxs-lookup"><span data-stu-id="23658-108">Ensures that property value changes on the host are automatically detected and translated to the hosted control or element.</span></span>  
  
> [!NOTE]
> <span data-ttu-id="23658-109">プロパティ変更イベントは、ホスティング コントロールまたは要素の階層に伝達されません。</span><span class="sxs-lookup"><span data-stu-id="23658-109">Property-change events are not propagated up the hosting control or element hierarchy.</span></span> <span data-ttu-id="23658-110">直接設定、スタイル、継承、データ バインディング、またはプロパティの値を変更するその他のメカニズムのために、プロパティのローカル値が変更されない場合、プロパティの変換は実行されません。</span><span class="sxs-lookup"><span data-stu-id="23658-110">Property translation is not performed if the local value of a property does not change because of direct setting, styles, inheritance, data binding, or other mechanisms that change the value of the property.</span></span>
