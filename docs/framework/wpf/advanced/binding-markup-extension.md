# <a name="binding-markup-extension"></a><span data-ttu-id="1898e-102">バインドのマークアップ拡張機能</span><span class="sxs-lookup"><span data-stu-id="1898e-102">Binding Markup Extension</span></span>
<span data-ttu-id="1898e-103">プロパティ値をデータ バインド値にするのを遅延し、中間式オブジェクトを作成し、実行時に要素とそのバインディングに適用されるデータ コンテキストを解釈します。</span><span class="sxs-lookup"><span data-stu-id="1898e-103">Defers a property value to be a data-bound value, creating an intermediate expression object and interpreting the data context that applies to the element and its binding at run time.</span></span>  
  
## <a name="binding-expression-usage"></a><span data-ttu-id="1898e-104">バインド式の使用</span><span class="sxs-lookup"><span data-stu-id="1898e-104">Binding Expression Usage</span></span>  
  
```xaml  
<object property="{Binding}" .../>  
-or-  
<object property="{Binding  bindProp1=value1[, bindPropN=valueN]*}" ...  
/>  
-or-  
<object property="{Binding path}" .../>  
-or  
<object property="{Binding path[, bindPropN=valueN]*}" .../>  
```
