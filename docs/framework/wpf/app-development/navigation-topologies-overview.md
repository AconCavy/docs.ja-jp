## <a name="structured-navigation-topologies"></a><span data-ttu-id="bdab6-122">構造化ナビゲーション トポロジ</span><span class="sxs-lookup"><span data-stu-id="bdab6-122">Structured Navigation Topologies</span></span>

- <span data-ttu-id="bdab6-126">**動的トポロジ**: ユーザー、アプリケーション、またはシステムから収集された入力に基づいて実行時に定義されます。</span><span class="sxs-lookup"><span data-stu-id="bdab6-126">**Dynamic Topology**: defined at run time based on input that is collected from the user, the application, or the system.</span></span> <span data-ttu-id="bdab6-127">動的トポロジは、ページがさまざまなシーケンスでナビゲートされる可能性があるときに便利です。</span><span class="sxs-lookup"><span data-stu-id="bdab6-127">Dynamic topologies are useful when pages can be navigated in different sequences.</span></span>
