<span data-ttu-id="91b7c-103">.NET Client Profile は、.NET Framework のサブセットで、.NET Framework 4 以前のバージョンに含まれ、クライアント アプリケーション用に最適化されていました。</span><span class="sxs-lookup"><span data-stu-id="91b7c-103">The .NET Client Profile is a subset of the .NET Framework, which was provided with .NET Framework 4 and earlier versions and was optimized for client applications.</span></span> <span data-ttu-id="91b7c-104">.NET Framework は、Windows、Windows Phone、および Microsoft Azure 用の開発プラットフォームであり、マネージド アプリの実行環境と .NET Framework クラス ライブラリを提供します。</span><span class="sxs-lookup"><span data-stu-id="91b7c-104">The .NET Framework is a development platform for Windows, Windows Phone and Microsoft Azure and provides a managed app execution environment and the .NET Framework class library.</span></span> <span data-ttu-id="91b7c-105">.NET Framework 4 以前のバージョンでは、2 種類の配置オプション (完全な .NET Framework と Client Profile) が用意されていました。</span><span class="sxs-lookup"><span data-stu-id="91b7c-105">The .NET Framework 4 and earlier versions provided two deployment options: the full .NET Framework and the Client Profile.</span></span> <span data-ttu-id="91b7c-106">完全な .NET Framework に比べて、Client Profile では、配置の迅速化とアプリ インストール パッケージの小型化が実現されました。</span><span class="sxs-lookup"><span data-stu-id="91b7c-106">The Client Profile enabled faster deployment and smaller app installation packages than the full .NET Framework.</span></span>
