> <span data-ttu-id="f75f8-121">デフォルトでは、DPI を認識しないアプリケーションのスケーリングは、ユーザーが dpi を 120 に設定した場合にはスケーリングを実行しませんが、dpi がカスタム値 144 以上に設定されている場合はスケーリングを実行します。</span><span class="sxs-lookup"><span data-stu-id="f75f8-121">By default, the DWM does not perform scaling for non-dpi-aware applications when the user sets the dpi to 120, but does perform it when the dpi is set to a custom value of 144 or higher.</span></span> <span data-ttu-id="f75f8-122">ただし、この既定の動作は、ユーザーがオーバーライドできます。</span><span class="sxs-lookup"><span data-stu-id="f75f8-122">However, the user can override the default behavior.</span></span>  
  
 <span data-ttu-id="f75f8-123">画面の拡大縮小は、画面座標に何らかの関わりを持つアプリケーションに新しい課題をもたらします。</span><span class="sxs-lookup"><span data-stu-id="f75f8-123">Screen scaling creates new challenges for applications that are concerned in any way with screen coordinates.</span></span> <span data-ttu-id="f75f8-124">現在、画面には物理座標と論理座標という 2 つの座標系が含まれています。</span><span class="sxs-lookup"><span data-stu-id="f75f8-124">The screen now contains two coordinate systems: physical and logical.</span></span> <span data-ttu-id="f75f8-125">あるポイントの物理座標は、原点の左上からの実際のオフセット (ピクセル数) です。</span><span class="sxs-lookup"><span data-stu-id="f75f8-125">The physical coordinates of a point are the actual offset in pixels from the top left of the origin.</span></span> <span data-ttu-id="f75f8-126">論理座標は、ピクセル自体が拡大縮小された場合のオフセットです。</span><span class="sxs-lookup"><span data-stu-id="f75f8-126">The logical coordinates are the offsets as they would be if the pixels themselves were scaled.</span></span>  
  
 <span data-ttu-id="f75f8-127">たとえば、ダイアログ ボックスを設計し、座標 (100, 48) にボタンを配置するとします。</span><span class="sxs-lookup"><span data-stu-id="f75f8-127">Suppose you design a dialog box with a button at coordinates (100, 48).</span></span> <span data-ttu-id="f75f8-128">このダイアログ ボックスが既定の 96 dpi に表示されている場合、ボタンは物理座標 (100, 48) にあります。</span><span class="sxs-lookup"><span data-stu-id="f75f8-128">When this dialog box is displayed at the default 96 dpi, the button is located at physical coordinates of (100, 48).</span></span> <span data-ttu-id="f75f8-129">120dpiでは、(125,60)の物理的座標に位置する。</span><span class="sxs-lookup"><span data-stu-id="f75f8-129">At 120 dpi, it is located at physical coordinates of (125, 60).</span></span> <span data-ttu-id="f75f8-130">しかし、論理座標は任意のdpi設定で同じです:(100、48)。</span><span class="sxs-lookup"><span data-stu-id="f75f8-130">But the logical coordinates are the same at any dpi setting: (100, 48).</span></span>
