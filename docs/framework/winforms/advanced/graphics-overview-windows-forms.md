# <a name="graphics-overview-windows-forms"></a>グラフィックスの概要 (Windows フォーム)
GDI + は、グラフィックス デバイス インターフェイスにより、デバイスに依存しないアプリケーションを記述するプログラマです。 GDI + のサービスは、一連のマネージ クラスを通じて公開されます。
