# <a name="using-a-brush-to-fill-shapes"></a>ブラシを使用した図形の塗りつぶし
GDI +<xref:System.Drawing.Brush>オブジェクトは、閉じた形状の内部を塗りつぶすために使用します。 GDI + いくつかの塗りつぶしのスタイルを定義します: 純色、ハッチ パターン、イメージのテクスチャ、および色のグラデーションします。
