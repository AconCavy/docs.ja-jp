# <a name="application-defined-or-object-defined-error"></a>アプリケーション定義またはオブジェクト定義のエラーです。

アプリケーション定義またはユーザー定義のエラーが発生しました。 エラーを確認します。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- エラーを参照して、エラーの定義場所および対処方法を確認します。
