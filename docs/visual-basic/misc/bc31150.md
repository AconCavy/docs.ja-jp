# <a name="end-tag-tagname-expected"></a>終了タグが \</tagName> 必要です

XML 要素リテラルの終了タグが、対応する開始タグに一致しません。  
  
 **エラー ID:** BC31150  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 終了 XML タグを修正して、XML 要素リテラルの開始 XML タグに一致させます。
