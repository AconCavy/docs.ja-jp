# <a name="membername-is-already-declared-by-procedurename-which-was-generated-for-this-containername"></a>'\<membername >' で既に宣言されて'\<procedurename >'、これに対して生成された\<containername >

共通言語ランタイムによって宣言されているメンバーと同じ名前のメンバーが宣言されました。

**エラー ID:** BC30733

## <a name="to-correct-this-error"></a>このエラーを解決するには

- メンバーの名前を変更して競合を解決します。
