# <a name="resource-name-resourcename-cannot-be-used-more-than-once"></a>リソース名 '\<resourcename>' を 2 回以上使用することはできません。

ダイナミックリンク ライブラリまたは実行可能ファイルで指定されたリソースの名前が複数回使用されました。  
  
 **エラー ID:** BC31502  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- リソースの名前を一意に付けます。
