# <a name="membername1-cannot-override-membername2-because-they-differ-by-the-types-of-optional-parameters"></a>'\<membername1 >' をオーバーライドできません'\<membername2 >' の省略可能なパラメーターの型が異なるため、
オーバーロードされたメンバーのデータ型は、1 つまたは複数の必須パラメーターについて、異なっている必要があります。  
  
 **エラー ID:** BC30697  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 別のデータ型として、少なくとも 1 つの必須パラメーターを定義します。
