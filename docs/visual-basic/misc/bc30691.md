# <a name="typename-is-a-type-in-containername-and-cannot-be-used-as-an-expression"></a>'\<typename >' 内の型は、'\<containername >' と、式として使用することはできません
使用中の名前が型の名前として既に宣言されているため、式では使用できません。  
  
 **エラー ID:** BC30691  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- メンバーに一意の名前を付けます。  
  
- メンバーの名前を角かっこで囲みます。
