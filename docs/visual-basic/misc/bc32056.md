# <a name="type-parameter-not-allowed-in-implements-clause"></a><span data-ttu-id="cfb75-102">型パラメーターは 'Implements' 句では許可されていません</span><span class="sxs-lookup"><span data-stu-id="cfb75-102">Type parameter not allowed in 'Implements' clause</span></span>
<span data-ttu-id="cfb75-103">ジェネリック型の `Implements` 句は、実装するメンバーとして型パラメーターを指定しています。</span><span class="sxs-lookup"><span data-stu-id="cfb75-103">An `Implements` clause in a generic type specifies a type parameter as the member to be implemented.</span></span>  
  
 <span data-ttu-id="cfb75-104">`Implements` 句は、インターフェイスとメンバーを指定する必要があります。</span><span class="sxs-lookup"><span data-stu-id="cfb75-104">An `Implements` clause must specify an interface and a member.</span></span> <span data-ttu-id="cfb75-105">これは、インターフェイスに型パラメーターを渡すことはできますが、メンバーに渡すことやメンバーの名前として使用することはできません。</span><span class="sxs-lookup"><span data-stu-id="cfb75-105">It can pass a type parameter to the interface, but it cannot pass it to the member, nor use it as the name of the member.</span></span>  
  
 <span data-ttu-id="cfb75-106">次のステートメントでは、このエラーが生成される可能性があります。</span><span class="sxs-lookup"><span data-stu-id="cfb75-106">The following statements can generate this error.</span></span>

```vb  
    Public Sub doSomething() Implements t  
End Class  
```
