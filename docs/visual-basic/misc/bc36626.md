# <a name="lambda-expression-cannot-be-converted-to-typename-because-type-typename-is-declared-mustinherit-and-cannot-be-created"></a>型 '\<\<typename > ' は ' MustInherit ' として宣言されていて作成できないため、ラムダ式を ' typename > ' に変換できません
ステートメントがラムダ式を型に変換しようとしていますが、その型は基底クラスにすることのみができ、インスタンスの作成には使用できません。  
  
 **エラー ID:** BC36626
