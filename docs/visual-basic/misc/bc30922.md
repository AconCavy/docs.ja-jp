# <a name="derivedtypename-cannot-inherit-from-type-constructedbasetypename-because-it-expands-the-access-of-type-internaltypename-outside-the-assembly"></a><span data-ttu-id="6d18b-102">型 ' \<internaltypename > ' のアクセスをアセンブリの外側に展開しているため、' \<derivedtypename > ' は \< 型 > ' \<constructedbasetypename > ' から継承することはできません</span><span class="sxs-lookup"><span data-stu-id="6d18b-102">'\<derivedtypename>' cannot inherit from \<type> '\<constructedbasetypename>' because it expands the access of type '\<internaltypename>' outside the assembly</span></span>
<span data-ttu-id="6d18b-103">派生クラスまたはインターフェイスが、基底クラスまたはインターフェイスの型引数として使用することで、制限がある型のアクセス レベルを拡張しようとしています。</span><span class="sxs-lookup"><span data-stu-id="6d18b-103">A derived class or interface attempts to expand the access level of a restricted type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="6d18b-104">次のコードでは、このエラーが生成される可能性があります。</span><span class="sxs-lookup"><span data-stu-id="6d18b-104">The following code can generate this error.</span></span>  
  
```vb  
Public Class baseClass(Of t)  
End Class  
Public Class derivedClass  
    Inherits baseClass(Of restrictedStructure)  
End Class  
Friend Structure restrictedStructure  
    Dim firstMember As Integer  
End Structure  
```  
  
 <span data-ttu-id="6d18b-105">アセンブリの外部のコードが `restrictedStructure`にアクセスすることはできません。</span><span class="sxs-lookup"><span data-stu-id="6d18b-105">Code outside the assembly is not allowed to access `restrictedStructure`.</span></span> <span data-ttu-id="6d18b-106">ただし、 `derivedClass` には、それを参照できるすべてのコードからアクセスできます。</span><span class="sxs-lookup"><span data-stu-id="6d18b-106">However, `derivedClass` can be accessed from any code that can reference it.</span></span> <span data-ttu-id="6d18b-107">そのため、任意のアセンブリのコードに `derivedClass` が公開される可能性があるので、 `baseClass` を型引数として使用する場合、 `restrictedStructure` は `restrictedStructure` を継承できません。</span><span class="sxs-lookup"><span data-stu-id="6d18b-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `restrictedStructure` as a type argument, because that could expose `restrictedStructure` to code in any assembly.</span></span>  
  
 <span data-ttu-id="6d18b-108">**エラー ID:** BC30922</span><span class="sxs-lookup"><span data-stu-id="6d18b-108">**Error ID:** BC30922</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="6d18b-109">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="6d18b-109">To correct this error</span></span>  
  
- <span data-ttu-id="6d18b-110">制限がある型のアクセス レベルを派生型が拡張しないように、クラスまたはインターフェイスのアクセス レベルを調整します。</span><span class="sxs-lookup"><span data-stu-id="6d18b-110">Adjust the access levels of the classes or interfaces so the derived type does not expand the access level of the restricted type.</span></span>  
  
     <span data-ttu-id="6d18b-111">\- または -</span><span class="sxs-lookup"><span data-stu-id="6d18b-111">-or-</span></span>  
  
- <span data-ttu-id="6d18b-112">アクセス レベルを調整できない場合、基底クラスまたはインターフェイスを構築するときに、制限がある型を型引数として使用しないでください。</span><span class="sxs-lookup"><span data-stu-id="6d18b-112">If you cannot adjust the access levels, do not use the restricted type as a type argument when constructing the base class or interface.</span></span>
