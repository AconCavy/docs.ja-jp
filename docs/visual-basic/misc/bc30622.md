# <a name="end-module-must-be-preceded-by-a-matching-module"></a>'End Module' の前には、対応する 'Module' を指定しなければなりません

モジュール ブロックは `Module` ステートメントで始まり、 `End Module` コンストラクトで終わります。  
  
 **エラー ID:** BC30622  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- モジュール ブロックが `Module` ステートメントで始まることを確認してください。
