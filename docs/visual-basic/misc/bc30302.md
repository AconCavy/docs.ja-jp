# <a name="type-character-character-cannot-be-used-in-a-declaration-with-an-explicit-type"></a>文字入力 '\<文字 >' では、明示的な型の宣言では使用できません
データ型を明示的に指定する際に、 `$`などの型文字が使用されています。  
  
 **エラー ID:** BC30302  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 宣言から型文字を削除するか、 `As <Type>` 句を削除します。
