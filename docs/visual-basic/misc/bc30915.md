このコードは、パブリック クラスの外部でプライベート型を公開しています。  
  
 **エラー ID:** BC30915  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 型 `Public`を宣言します。  
  
     - または -  
  
- 別の型を使用します。
