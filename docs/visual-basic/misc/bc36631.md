# <a name="join-expected"></a><span data-ttu-id="e1824-102">'Join' が必要です</span><span class="sxs-lookup"><span data-stu-id="e1824-102">'Join' expected</span></span>

<span data-ttu-id="e1824-103">`Group Join` 句が `Join` キーワードを使用せずに指定されています。</span><span class="sxs-lookup"><span data-stu-id="e1824-103">A `Group Join` clause has been specified without a `Join` keyword.</span></span>  
  
 <span data-ttu-id="e1824-104">**エラー ID:** BC36631</span><span class="sxs-lookup"><span data-stu-id="e1824-104">**Error ID:** BC36631</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="e1824-105">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="e1824-105">To correct this error</span></span>  
  
1. <span data-ttu-id="e1824-106">次の例で示すように、 `Join` キーワードを `Group Join` 句に追加します。</span><span class="sxs-lookup"><span data-stu-id="e1824-106">Add the `Join` keyword to the `Group Join` clause, as shown in the following example:</span></span>  
  
```vb  
Dim query = From var1 In collection1 _  
            Join var2 In collection2 _  
            Group Join var3 In collection3 _  
            On var2.ID Equals var3 Into Matches = Group _  
            On var1 Equals var2.ID _  
            Select JoinID = var1, var2.Name, Matches  
```
