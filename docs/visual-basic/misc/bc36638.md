# <a name="instance-members-and-me-cannot-be-used-within-a-lambda-expression-in-structures"></a><span data-ttu-id="3803c-102">インスタンス メンバーおよび 'Me' を構造体のラムダ式内で使用することはできません</span><span class="sxs-lookup"><span data-stu-id="3803c-102">Instance members and 'Me' cannot be used within a lambda expression in structures</span></span>
<span data-ttu-id="3803c-103">構造体の内部から、この構造体のインスタンス メンバーを参照するか `Me`を使用するラムダ式を定義しました。</span><span class="sxs-lookup"><span data-stu-id="3803c-103">From within a structure, you have defined a lambda expression that refers to an instance member of the structure or uses `Me`.</span></span> <span data-ttu-id="3803c-104">次のコードで、これらの 2 つの正しくない参照の例を示します。</span><span class="sxs-lookup"><span data-stu-id="3803c-104">The following code illustrates these two invalid references.</span></span>  
  
```vb  
Structure Structure1  
  
    Public InstanceMember As Integer  
  
    Public Function ExampleFun() As Integer  
        '' The error is caused by use of InstanceMember.  
        'Dim fun1 = Function() InstanceMember  
        '' The error is caused by use of Me.  
        'Dim fun2 = Function() Me.InstanceMember  
        'Return fun1()  
    End Function  
  
End Structure  
```  
  
 <span data-ttu-id="3803c-105">**エラー ID:** BC36638</span><span class="sxs-lookup"><span data-stu-id="3803c-105">**Error ID:** BC36638</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="3803c-106">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="3803c-106">To correct this error</span></span>

```vb  
    Dim fun1 = Function() temp  
    Return fun1()  
End Function  
```
