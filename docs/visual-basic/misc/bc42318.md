# <a name="xml-comment-type-parameter-must-have-a-name-attribute"></a>XML コメントの型パラメーターには 'name' 属性がなければなりません
XML コメントの型パラメーターには 'name' 属性を指定しなければなりません。 XML コメントは無視されます。  
  
 XML コメント タグに必要な `name` 属性がないか、不完全です。  
  
 **エラー ID:** BC42318  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- タグに必要な `name` 属性を追加するか、完成させます。
