# <a name="unable-to-open-key-file-filename-error"></a>キーファイル ' ' を開くことができません \<filename> : \<error>

このプロジェクトに指定したキー ファイルを開けませんでした。  
  
 **エラー ID:** BC31025  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- キー ファイルが存在し、別のアプリケーションによってロックされていないことを確認します。
