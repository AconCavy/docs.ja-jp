<span data-ttu-id="6f0a4-103">クラスが複数の部分宣言で定義されており、そのうちの 1 つは `MustInherit` を指定し、別の 1 つは `NotInheritable`を指定しています。</span><span class="sxs-lookup"><span data-stu-id="6f0a4-103">A class is defined in multiple partial declarations, one of which specifies `MustInherit` and another of which specifies `NotInheritable`.</span></span>  
  
 <span data-ttu-id="6f0a4-104">1 つのクラスの定義を複数の部分宣言間で分割すると、コンパイラはすべての部分宣言の和集合としてこのクラスを処理します。</span><span class="sxs-lookup"><span data-stu-id="6f0a4-104">When you divide the definition of a class among several partial declarations, the compiler treats the class as the union of all its partial declarations.</span></span> <span data-ttu-id="6f0a4-105">これは、メンバーだけでなく、実装、継承、およびアクセス レベルにも適用されます。</span><span class="sxs-lookup"><span data-stu-id="6f0a4-105">This applies not only to the members but also to the implementation, inheritance, and access level.</span></span>  
  
 <span data-ttu-id="6f0a4-106">クラスを *abstract* と *sealed*の両方に設定することはできません。つまり、継承を必要とし、同時に禁止することはできません。</span><span class="sxs-lookup"><span data-stu-id="6f0a4-106">A class cannot be both *abstract* and *sealed*, meaning it cannot both require and forbid inheritance.</span></span> <span data-ttu-id="6f0a4-107">このため、同じクラスに対して `MustInherit` と `NotInheritable` を両方指定することはできません。</span><span class="sxs-lookup"><span data-stu-id="6f0a4-107">Therefore you cannot specify both `MustInherit` and `NotInheritable` for the same class.</span></span>
