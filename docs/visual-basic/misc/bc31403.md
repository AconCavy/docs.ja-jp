# <a name="imports-alias-name-conflicts-with-name2-declared-in-the-root-namespace"></a>Imports エイリアス\<名 >' と競合 '\<name2 >' のルート名前空間で宣言されています。
`Imports` ステートメントで使用されるエイリアスは、ルート名前空間で宣言されている別の名前と競合しています。  
  
 **エラー ID:** BC31403  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- `Imports` ステートメントで使用されるエイリアスに一意の名前を選択します。
