# <a name="end-raiseevent-must-be-preceded-by-a-matching-raiseevent-declaration"></a>'End RaiseEvent' の前に、対応する 'RaiseEvent' 宣言を指定する必要があります

`End RaiseEvent` ステートメントが出現しましたが、対応する `RaiseEvent` ステートメントがありません。 `End RaiseEvent` の前に、対応する `RaiseEvent` ステートメントが必要です。  
  
 **エラー ID:** BC31126  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 先行する `RaiseEvent` ステートメントが有効であり、スペルが正しいことを確認します。
