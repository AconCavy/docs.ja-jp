# <a name="property-propertyname-is-readonly"></a>プロパティ '\<propertyname >' は 'ReadOnly'
`ReadOnly`で宣言されているプロパティに値を代入しようとしました。  
  
 **エラー ID:** BC30526  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- プロパティ宣言から `ReadOnly` 指定子を削除します。
