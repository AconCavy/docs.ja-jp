# <a name="name-is-ambiguous-in-the-application-objects-list"></a>'\<名 >' で、アプリケーション オブジェクトのあいまいな\<リスト >'
あいまいな名前を指定したため、別の名前と競合しています。 Visual Basic コンパイラには、競合の解決ルールはありません。名前は、自分で明確する必要があります。 エラー メッセージに示されるリストには、名前があいまいなアプリケーション オブジェクトに関する詳細が記載されています。  
  
 **エラー ID:** BC30563  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 名前を完全修飾します。
