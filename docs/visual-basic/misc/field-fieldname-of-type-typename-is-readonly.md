# <a name="field-fieldname-of-type-typename-is-readonly"></a>フィールド '\<fieldname >' 型の'\<typename >' は 'ReadOnly'
`ReadOnly`に宣言されているフィールドに書き込もうとしました。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- `ReadOnly`に宣言されないようにフィールドを変更します。  
  
- または  
  
- このフィールドに書き込まないでください。
