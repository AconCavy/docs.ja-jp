# <a name="lib-expected"></a>'Lib' が必要です。
`Declare` ステートメントは、 `Lib` キーワードを使用せずにライブラリ名を指定します。  
  
 **エラー ID:** BC30218  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- ライブラリ名の前に `Lib` キーワードを追加します。
