# <a name="value-of-type-type1-cannot-be-converted-to-type2-because-the-array-types-have-different-numbers-of-dimensions"></a>型の値 '\<type1 >' に変換できません'\<type2 >' 配列型がディメンションの数が異なるため、。
1 つの配列をディメンションの数が異なる別の配列に変換しようとしました。  
  
 **エラー ID:** BC30414  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- ディメンションの数が一致するように、片方または両方の配列を変更します。
