# <a name="cannot-infer-a-common-type-for-the-first-and-second-operands-of-the-binary-if-operator"></a><span data-ttu-id="ce8cc-102">バイナリ 'If' 演算子の 1 番目と 2 番目のオペランドの共通型を推論できません</span><span class="sxs-lookup"><span data-stu-id="ce8cc-102">Cannot infer a common type for the first and second operands of the binary 'If' operator</span></span>

<span data-ttu-id="ce8cc-103">バイナリ 'If' 演算子の 1 番目と 2 番目のオペランドの共通型を推論できません。</span><span class="sxs-lookup"><span data-stu-id="ce8cc-103">Cannot infer a common type for the first and second operands of the binary 'If' operator.</span></span> <span data-ttu-id="ce8cc-104">一方が他方に対する拡大変換を持つ必要があります。</span><span class="sxs-lookup"><span data-stu-id="ce8cc-104">One must have a widening conversion to the other.</span></span>

<span data-ttu-id="ce8cc-105">バイナリ `If` 演算子では、引数の 1 つともう 1 つの引数の間に拡大変換が必要です。</span><span class="sxs-lookup"><span data-stu-id="ce8cc-105">The binary `If` operator requires that there be a widening conversion between one of the arguments and the other argument.</span></span> <span data-ttu-id="ce8cc-106">たとえば、 `Integer` と `String`の間にはどちらの方向にも拡大変換がないため、次のコードでこのエラーが発生します。</span><span class="sxs-lookup"><span data-stu-id="ce8cc-106">For example, because there is not a widening conversion in either direction between `Integer` and `String`, the following code causes this error.</span></span>

```vb
Dim first? As Integer
Dim second As String = "First is Nothing"
'' Not valid.
' Console.WriteLine(If(first, second))
```

<span data-ttu-id="ce8cc-107">**エラー ID:** BC33110</span><span class="sxs-lookup"><span data-stu-id="ce8cc-107">**Error ID:** BC33110</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="ce8cc-108">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="ce8cc-108">To correct this error</span></span>

- <span data-ttu-id="ce8cc-109">コードで可能な場合は、オペランドの 1 つに明示的な変換を提供します。</span><span class="sxs-lookup"><span data-stu-id="ce8cc-109">Provide an explicit conversion for one of the operands, if that is possible in your code:</span></span>

  ```vb
  Console.WriteLine(If(first, CInt(second)))
  ```

- <span data-ttu-id="ce8cc-110">別の条件構造を使用して、コードを書き直します。</span><span class="sxs-lookup"><span data-stu-id="ce8cc-110">Rewrite the code by using a different conditional construction.</span></span>

  ```vb
  If first IsNot Nothing Then
      Console.WriteLine(first)
  Else
      Console.WriteLine(second)
  End If
  ```
