# <a name="propertyname1-cannot-override-propertyname2-because-they-differ-by-readonly-or-writeonly"></a>' ReadOnly ' または ' WriteOnly\<' が異なるため、'propertyname1>'は'propertyname2>'をオーバーライドできません\<
1 番目のプロパティを、 `ReadOnly` か `WriteOnly`かの違いしかない 2 番目のプロパティでオーバーライドしようとしました。  
  
 **エラー ID:** BC30362  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- プロパティどうしの間に `ReadOnly` と `WriteOnly`以外の違いがあるようにします。
