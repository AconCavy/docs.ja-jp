# <a name="loop-control-variable-cannot-be-a-property-or-a-late-bound-indexed-array"></a>Loop コントロール変数を、プロパティまたは遅延バインディングされたインデックス付き配列に指定することはできません。
`For` ループによる反復処理に使用する変数は、数値データ型にする必要があります。  
  
 **エラー ID:** BC30039  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- Loop コントロール変数の宣言を変更して、 `Integer`、 `Short`、 `Long`、 `Byte`、 `Single`、 `Double`、または `Decimal`を指定します。
