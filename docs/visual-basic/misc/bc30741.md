# <a name="typename1-cannot-be-converted-to-typename2"></a>'\<typename1>' を '\<typename2>'に変換できません。
2 つの互換性のない型どうしで型変換を行おうとしました。  
  
 **エラー ID:** BC30741  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- キャストされている型を確認します。
