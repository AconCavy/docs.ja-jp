# <a name="modifier-is-not-valid-on-an-event-declaration"></a>' \<modifier> ' は、イベント宣言では有効ではありません。

`Event` ステートメントに `ReadOnly`などの無効なキーワードが含まれています。  
  
 **エラー ID:** BC30243  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1. `Event` ステートメントから使用できないキーワードを削除します。
