# <a name="unable-to-open-response-file-filename"></a>応答ファイルを開けません '\<filename >'。
アクセスしようとしているファイルは使用不可です。  
  
 **エラー ID:** BC2011  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- ネットワーク接続がアクティブであることを確認してください。  
  
- ファイルのアクセス許可を確認します。
