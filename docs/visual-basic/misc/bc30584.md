# <a name="type-cannot-be-inherited-more-than-once"></a>'\<型 >' は複数回継承することはできません
`Inherits` ステートメントが重複しています。  
  
 **エラー ID:** BC30584  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 不要な `Inherits` ステートメントを削除します。
