# <a name="specified-event-log-does-not-exist-on-this-machine"></a>指定したイベント ログは、このコンピューター上に存在しません

このコンピューター上に存在しないイベント ログにアクセスしようとしました。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 正しいファイル名とパスを指定していることを確認します。
