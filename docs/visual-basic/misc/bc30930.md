# <a name="name-necessary-for-compiling-this-construct-cannot-be-found"></a>\<name>このコンストラクトをコンパイルするために必要な ' ' が見つかりません。

コンパイルに必要な要素が見つかりません。  
  
 **エラー ID:** BC30930  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1. 必要なヘルパー要素を指定します。  
  
2. コードが有効なマネージド コードであることを確認します。
