# <a name="unable-to-sign-assembly-error-message"></a>アセンブリに署名できません:\<エラー メッセージ >
Visual Basic コンパイラでは、マニフェストを持つアセンブリを生成するには、アセンブリ リンカー (Al.exe、Alink とも呼ばれます) を呼び出します。 リンカーは、作成された後のアセンブリに署名する際にエラーを報告しました。 これは、無効なキーが原因である可能性があります。  
  
 **エラー ID:** BC30146  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1. 引用符で囲まれたエラー メッセージを確認し、トピックを参照してください。 [Al.exe (アセンブリ リンカー)](../../framework/tools/al-exe-assembly-linker.md)のより詳細な説明とアドバイスを参照します。  
  
2. エラーが続く場合は、状況に関する情報を収集し、マイクロソフト プロダクト サポート サービスに通知してください。
