# <a name="shared-cannot-be-combined-with-specifier-on-a-method-declaration"></a>メソッド宣言で、'Shared' を '\<specifier>' と組み合わせて使用することはできません。

メソッド宣言で有効ではない組み合わせで、 `Shared` 修飾子と指定子 ( `Overridable`、 `NotOverridable`、 `MustOverride`など) を組み合わせようとしました。  
  
 **エラー ID:** BC30501  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 指定子を削除します。
