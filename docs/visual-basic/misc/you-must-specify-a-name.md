# <a name="you-must-specify-a-name"></a>名前を指定する必要があります

ファイル名の変更などのファイル操作を実行するときは、名前を指定する必要があります。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 名前を指定します。
