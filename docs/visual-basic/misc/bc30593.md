# <a name="variables-in-modules-cannot-be-declared-specifier"></a>モジュールの変数を宣言することはできません '\<指定子 >'
`MustInherit`ステートメント内の変数に `Module` などの指定子を使用しました。 モジュールは、インスタンス化することができず、継承をサポートしません。また、インターフェイスを実装することもできません。  
  
 **エラー ID:** BC30593  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 指定子を削除します。
