# <a name="character-character-0xhexadecimal-is-not-allowed-at-the-beginning-of-an-xml-name"></a>文字 'character' (0 x\<16 進数 >) は、XML 名の先頭には許可されていません
サポートされていない文字が、XML リテラルまたは XML 軸プロパティの名前の先頭に含まれています。  
  
 **エラー ID:** BC31169  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- XML リテラルまたは XML 軸プロパティの名前の先頭から、サポートされていない文字を削除します。
