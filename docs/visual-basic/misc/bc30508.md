# <a name="name-cannot-expose-type-type1-in-specifier-type2-typename-through-name"></a>'\<名 >' 型を公開できません\<type1 > で\<指定子 > \<type2 >'\<typename >' から\<名 >。
パブリック クラス `MyClass`の外部でプライベート型を公開しようとしました。  
  
 **エラー ID:** BC30508  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 型 `Public`を宣言します。
