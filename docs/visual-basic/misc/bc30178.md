# <a name="specifier-is-duplicated"></a>指定子が重複しています。
同じ宣言内で複数回出現するキーワードがあります。  
  
 **エラー ID:** BC30178  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 重複したキーワードの出現箇所を宣言から削除します。
