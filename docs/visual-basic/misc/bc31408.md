# <a name="specifier1-and-specifier2-cannot-be-combined"></a>' \<specifier1> ' と ' \<specifier2> ' を組み合わせて使用することはできません。

`Shadows` と `Overrides` など、2 つの互換性のない指定子が一緒に使用されています。  
  
 **エラー ID:** BC31408  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- いずれかの指定子を削除します。
