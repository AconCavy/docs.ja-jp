# <a name="xml-name-expected"></a>XML 名が必要です

XML リテラルまたは XML 軸プロパティを名前なしで指定しました。  
  
 **エラー ID:** BC31146  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- XML リテラルまたは XML 軸プロパティの名前を追加します。
