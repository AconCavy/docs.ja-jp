<span data-ttu-id="b991e-103">ジェネリック インターフェイスが 2 つ以上のジェネリック インターフェイスから継承されており、その継承のうちの 2 つが、型引数の特定の値について競合している可能性があります。</span><span class="sxs-lookup"><span data-stu-id="b991e-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>

```vb

    Inherits interfaceA(Of v)

Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="b991e-105">`derivedInterface` が `t1` と `t2`の両方に同じ型を指定して構築または実装されている場合、同一の型引数を含む 2 つのバージョンの `interfaceA` を継承する必要があります。</span><span class="sxs-lookup"><span data-stu-id="b991e-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="b991e-106">この場合、どのバージョンにアクセスするかがあいまいになります。</span><span class="sxs-lookup"><span data-stu-id="b991e-106">Doing so would produce an ambiguity about which version to access.</span></span>
