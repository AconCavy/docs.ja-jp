# <a name="unable-to-ping-because-a-network-connection-is-not-available"></a>ネットワーク接続が利用できないため、ping を実行できません

ネットワーク接続が利用できないため、操作を実行できません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- ネットワークに接続し、操作をやり直します。
