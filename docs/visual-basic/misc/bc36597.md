# <a name="goto-to-label-labelname-cannot-be-used-within-lambda-expressions"></a>GoTo ラベルに '\<labelname >' ラムダ式内で使用することはできません
ラムダ式の本体は、計算を実行して単一の値を返す式を指定する必要があります。 `GoTo` ステートメントを含む、すべてのステートメントが許可されません。  
  
 **エラー ID:** BC36597
