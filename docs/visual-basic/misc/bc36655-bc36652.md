<span data-ttu-id="e89e6-103">' ' で定義されている拡張メソッド ' ' 内の型パラメーターのデータ型 \<methodname> \<typename> をこれらの引数から推論することはできません。複数の型が考えられるためです。</span><span class="sxs-lookup"><span data-stu-id="e89e6-103">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in '\<typename>' cannot be inferred from these arguments because more than one type is possible.</span></span> <span data-ttu-id="e89e6-104">データ型を明示的に指定すると、このエラーが修正される可能性があります。</span><span class="sxs-lookup"><span data-stu-id="e89e6-104">Specifying the data type(s) explicitly might correct this error.</span></span>

<span data-ttu-id="e89e6-105">ジェネリック拡張メソッドを呼び出すときに、型の推定を使用して型パラメーターの型を決定しようとしました。</span><span class="sxs-lookup"><span data-stu-id="e89e6-105">An attempt has been made to use type inference to determine the type (or types) of the type parameter (or parameters) in a call to a generic extension method.</span></span> <span data-ttu-id="e89e6-106">コンパイラによって 1 つ以上の型パラメーターについて複数のデータ型の候補が検出されると、このエラーが報告されます。</span><span class="sxs-lookup"><span data-stu-id="e89e6-106">The compiler finds more than one possible data type for one or more of the type parameters, and it reports this error.</span></span>

> [!NOTE]
> <span data-ttu-id="e89e6-107">引数の指定がオプションではない場合 (たとえば、クエリ式内のクエリ演算子など)、エラー メッセージの 2 つ目の文は表示されません。</span><span class="sxs-lookup"><span data-stu-id="e89e6-107">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>

<span data-ttu-id="e89e6-108">次のコードはエラーを示しています。</span><span class="sxs-lookup"><span data-stu-id="e89e6-108">The following code demonstrates the error.</span></span>

```vb
Option Strict Off
Imports System.Runtime.CompilerServices
Module Module1
    Sub Main()

        Dim caller As New Class1
        '' Not valid.
        'caller.targetExtension(1, "2")

    End Sub

    <Extension()> _
    Sub targetExtension(Of T)(ByVal p0 As Class1, ByVal p1 As T, ByVal p2 As T)
    End Sub

    Class Class1
    End Class

End Module
```
