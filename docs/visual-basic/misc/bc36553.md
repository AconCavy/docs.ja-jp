## <a name="example"></a><span data-ttu-id="fae68-115">例</span><span class="sxs-lookup"><span data-stu-id="fae68-115">Example</span></span>  

 <span data-ttu-id="fae68-116">次の例の最初のパラメーターは、 `Print` メソッドが `String` データ型を拡張することを示すに過ぎません。</span><span class="sxs-lookup"><span data-stu-id="fae68-116">The first parameter in the following example is the only indication that the `Print` method extends the `String` data type.</span></span> <span data-ttu-id="fae68-117">したがって、省略可能にはできません。</span><span class="sxs-lookup"><span data-stu-id="fae68-117">Therefore, it cannot be optional.</span></span>

End Sub  
```  
  
 <span data-ttu-id="fae68-118">拡張メソッドが次のように呼び出される場合、そのメソッドのパラメーター `str` は `greeting`( `String` を呼び出す `Print`のインスタンス) にバインドされます。</span><span class="sxs-lookup"><span data-stu-id="fae68-118">When the extension method is called as follows, parameter `str` in the method is bound to `greeting`, the instance of `String` that calls `Print`.</span></span> <span data-ttu-id="fae68-119">コンパイラは、拡張メソッド `greeting` の引数として `Print`を使用します。</span><span class="sxs-lookup"><span data-stu-id="fae68-119">The compiler uses `greeting` as the argument to extension method `Print`.</span></span>

```
