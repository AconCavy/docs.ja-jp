# <a name="me-cannot-be-the-target-of-an-assignment"></a>'Me' を代入式のターゲットにすることはできません

通常、 `Me` キーワードは、コードが現在実行されているクラスの特定のインスタンスを参照します。 この値は変更できません。  
  
 **エラー ID:** BC30062  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 値を `Me`に割り当てるステートメントを削除します。
