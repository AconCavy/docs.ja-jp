# <a name="expected-closing--for-xml-entity"></a>XML エンティティの終わりに ';' が必要です
XML エンティティ参照 (例: `>`など) で最後のセミコロン (;) が見つかりません。  
  
 **エラー ID:** BC31178  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- XML エンティティ参照の末尾にセミコロンを含めます。
