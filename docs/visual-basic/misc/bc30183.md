# <a name="keyword-is-not-valid-as-an-identifier"></a>キーワードは、識別子として有効ではありません

要素名を指定する必要がある場所にキーワードが指定されています。 このコンテキストでは、変数やプロシージャなどの要素が必要です。  
  
 **エラー ID:** BC30183  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- キーワードを有効な要素名に置き換えます。
