# <a name="collection-index-must-be-in-the-range-1-to-the-size-of-the-collection"></a>コレクション インデックスのサイズは、1 以上かつコレクションのサイズ以下の範囲でなければなりません
指定されたインデックス値は有効な範囲、つまり 1 以上コレクションのサイズ以下に含まれません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1. 指定された値を有効なものに変更します。
