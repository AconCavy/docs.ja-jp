# <a name="name-is-declared-in-project-projectname1-which-is-not-referenced-by-project-projectname2"></a>' \<name> ' はプロジェクト ' ' で宣言されていますが、プロジェクト ' ' で参照され \<projectname1> ていません。 \<projectname2>

プログラミング要素が、現在のプロジェクトに参照が含まれていないプロジェクトで定義されています。  
  
 **エラー ID:** BC32004  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 現在のプロジェクトで、目的の要素が入っているプロジェクトへの参照を追加します。
