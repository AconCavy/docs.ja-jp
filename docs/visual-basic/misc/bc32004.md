# <a name="name-is-declared-in-project-projectname1-which-is-not-referenced-by-project-projectname2"></a>' \<name > ' は、プロジェクト ' \<projectname2 > ' によって参照されていないプロジェクト ' \<projectname1 > ' で宣言されています
プログラミング要素が、現在のプロジェクトに参照が含まれていないプロジェクトで定義されています。  
  
 **エラー ID:** BC32004  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 現在のプロジェクトで、目的の要素が入っているプロジェクトへの参照を追加します。
