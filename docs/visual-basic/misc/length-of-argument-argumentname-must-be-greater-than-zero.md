# <a name="length-of-argument-argumentname-must-be-greater-than-zero"></a>引数長 '\<argumentname>' は 0 より大きくなければなりません。

引数に、0 以下の値が含まれています。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 0 より大きい引数を指定します。
