# <a name="expected-closing----for-xml-comment"></a>XML コメントの終わりに '--> ' が必要です

XML コメント リテラルの終わりに必要な '-->' 文字がありません。  
  
 **エラー ID:** BC31161  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- XML コメント リテラルの終わりに必要な '-->' 文字を追加します。
