# <a name="set-method-cannot-have-more-than-one-parameter"></a>'Set' メソッドに 2 つ以上のパラメーターを指定することはできません

`Set` プロパティ プロシージャについて複数のパラメーターが定義されています。  
  
 **エラー ID:** BC31063  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- `Set` プロパティ プロシージャから余分なパラメータを削除します。
