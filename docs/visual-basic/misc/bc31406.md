# <a name="methodname-cannot-override-a-method-that-has-been-shadowed"></a>'\<methodname >' シャドウされたメソッドをオーバーライドすることはできません
メソッドは、別のメンバーをシャドウするメソッドをオーバーライドしようとしました。  
  
 **エラー ID:** BC31406  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- `Shadows` 修飾子を使用して、シャドウされたメンバーをオーバーライドします。
