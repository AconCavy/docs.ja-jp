# <a name="the-file-filename-is-not-a-text-file"></a>ファイル '\<filename >' はテキスト ファイルではありません
指定したファイルはバイナリ ファイルのため、書き込みができません。  
  
 **エラー ID:** BC2015  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 正しいファイル名とパスを指定していることを確認します。
