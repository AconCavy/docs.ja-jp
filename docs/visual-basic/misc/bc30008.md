# <a name="reference-required-to-module-modulename-containing-the-base-class-classname"></a>モジュールに必要な参照 '\<modulename >' 基底クラスを含む\<classname >'
モジュールに必要な参照 '\<modulename >' 基底クラスを含む\<classname >'。 プロジェクトに参照を追加してください。  
  
 プロジェクト内で直接参照されないモジュールでクラスが定義されています。 Visual Basic コンパイラでは、クラスが 1 つ以上のモジュールで定義されている場合に備えて、あいまいさを避けるためへの参照が必要です。  
  
 **エラー ID:** BC30008  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 参照されていないモジュールの名前をプロジェクト参照に含めます。
