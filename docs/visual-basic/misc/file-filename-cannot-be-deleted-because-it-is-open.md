# <a name="file-filename-cannot-be-deleted-because-it-is-open"></a>ファイル '\<filename>' が開いているため、削除できません。

開いているファイルを削除しようとしました。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- ファイルを閉じてから削除します。
