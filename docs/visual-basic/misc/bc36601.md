# <a name="type-characters-cannot-be-used-in-range-variable-declarations"></a>型文字を範囲変数の宣言で使用することはできません
型文字が、LINQ クエリの範囲変数宣言の名前の末尾に含まれています。  
  
 **エラー ID:** BC36601  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1. 範囲変数の宣言から型文字を削除します。
