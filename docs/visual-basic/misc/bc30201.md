# <a name="expression-expected"></a>式が必要です
`+` または `=` などの演算子の右オペランドには式が必要ですが、認識できる式がありません。  
  
 **エラー ID:** BC30201  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 演算子の右オペランドに正しい式を指定します。
