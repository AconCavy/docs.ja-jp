# <a name="elementname-is-not-a-method-parameter"></a>'\<elementname >' メソッド パラメーターではありません
プロシージャ呼び出しで、メソッドによって定義されていない追加の引数が指定されています。  
  
 **エラー ID:** BC36568  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 引数リストから余分な引数を削除します。
