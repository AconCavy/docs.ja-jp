# <a name="structures-visual-basic"></a>構造体 (Visual Basic)
A*構造*汎化、ユーザー定義型 (UDT) の以前のバージョンの Visual Basic でサポートされているです。 フィールドに加えて、構造体は、プロパティ、メソッド、およびイベントを公開できます。 構造体が 1 つまたは複数のインターフェイスを実装して、フィールドごとに個別のアクセス レベルを宣言することができます。  
  
 構造を作成するさまざまな種類のデータ項目を組み合わせることができます。 構造体は、1 つまたは複数を関連付けます*要素*と構造体そのものです。 構造体を宣言するときに、*複合データ型*、し、その型の変数を宣言することができます。  
  
 構造体は、いくつかの関連情報を保持するために 1 つの変数が必要な場合に便利です。 たとえば、従業員の名前、電話の拡張機能、および給与をまとめておくためにする場合があります。 詳細については、いくつかの変数を使用する可能性があります。 または構造体を定義し、1 人の従業員の変数を使用する可能性があります。 多くの従業員とそのため、変数の多数のインスタンスがある場合、構造体の利点は明らかになります。
