## <a name="lambda-expression-syntax"></a><span data-ttu-id="5935a-118">ラムダ式の構文</span><span class="sxs-lookup"><span data-stu-id="5935a-118">Lambda Expression Syntax</span></span>

     <span data-ttu-id="5935a-132">たとえば、`Return` ステートメントに指定された式に `Integer`、`Long`、および `Double`型の値が含まれている場合、結果の配列の型は `Double`になります。</span><span class="sxs-lookup"><span data-stu-id="5935a-132">For example, if the expressions supplied to the `Return` statement contain values of type `Integer`, `Long`, and `Double`, the resulting array is of type `Double`.</span></span> <span data-ttu-id="5935a-133">`Integer` と `Long` はどちらも `Double` に広げ、`Double`のみです。</span><span class="sxs-lookup"><span data-stu-id="5935a-133">Both `Integer` and `Long` widen to `Double` and only `Double`.</span></span> <span data-ttu-id="5935a-134">そのため、 `Double` が最も優先度の高い型になります。</span><span class="sxs-lookup"><span data-stu-id="5935a-134">Therefore, `Double` is the dominant type.</span></span> <span data-ttu-id="5935a-135">詳細については、「 [Widening and Narrowing Conversions](../../../../visual-basic/programming-guide/language-features/data-types/widening-and-narrowing-conversions.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="5935a-135">For more information, see [Widening and Narrowing Conversions](../../../../visual-basic/programming-guide/language-features/data-types/widening-and-narrowing-conversions.md).</span></span>
