# <a name="trycatchfinally-statement-visual-basic"></a><span data-ttu-id="cd693-103">Try...Catch...Finally ステートメント (Visual Basic)</span><span class="sxs-lookup"><span data-stu-id="cd693-103">Try...Catch...Finally Statement (Visual Basic)</span></span>

<span data-ttu-id="cd693-104">コード内の所定のブロックで発生する可能性のあるエラーの一部またはすべてに対して、コードを実行しながらエラー処理を実行できます。</span><span class="sxs-lookup"><span data-stu-id="cd693-104">Provides a way to handle some or all possible errors that may occur in a given block of code, while still running code.</span></span>

## <a name="syntax"></a><span data-ttu-id="cd693-105">構文</span><span class="sxs-lookup"><span data-stu-id="cd693-105">Syntax</span></span>

```vb
Try
    [ tryStatements ]
    [ Exit Try ]
[ Catch [ exception [ As type ] ] [ When expression ]
    [ catchStatements ]
    [ Exit Try ] ]
[ Catch ... ]
[ Finally
    [ finallyStatements ] ]
End Try
```
