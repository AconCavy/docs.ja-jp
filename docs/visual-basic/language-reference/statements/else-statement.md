# <a name="else-statement-visual-basic"></a>Else ステートメント (Visual Basic)

他に条件付きグループのステートメントが実行またはコンパイルされていない場合に、実行またはコンパイルされるステートメントのグループについて説明します。

 キーワード `Else` は次のコンテキストで使用できます。
