## <a name="memory-consumption"></a><span data-ttu-id="136c9-171">メモリの使用量</span><span class="sxs-lookup"><span data-stu-id="136c9-171">Memory Consumption</span></span>

### <a name="composite-data-types"></a><span data-ttu-id="136c9-180">複合データ型</span><span class="sxs-lookup"><span data-stu-id="136c9-180">Composite Data Types</span></span>  

 <span data-ttu-id="136c9-181">構造体や配列など、複合データ型の各メンバーにも同じ考慮事項が適用されます。</span><span class="sxs-lookup"><span data-stu-id="136c9-181">The same considerations apply to each member of a composite data type, such as a structure or an array.</span></span> <span data-ttu-id="136c9-182">型のメンバーの公称ストレージ割り当てを簡単に追加することはできません。</span><span class="sxs-lookup"><span data-stu-id="136c9-182">You cannot rely on simply adding together the nominal storage allocations of the type's members.</span></span> <span data-ttu-id="136c9-183">さらに、次のような考慮事項があります。</span><span class="sxs-lookup"><span data-stu-id="136c9-183">Furthermore, there are other considerations, such as the following:</span></span>
