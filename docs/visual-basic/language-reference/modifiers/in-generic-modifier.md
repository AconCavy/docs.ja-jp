<span data-ttu-id="8b4f9-118">反変のデリゲートには、型は同じでありながらより弱い派生ジェネリック型パラメーターを持つ別のデリゲートを割り当てることができます。</span><span class="sxs-lookup"><span data-stu-id="8b4f9-118">A contravariant delegate can be assigned another delegate of the same type, but with a less derived generic type parameter.</span></span>
