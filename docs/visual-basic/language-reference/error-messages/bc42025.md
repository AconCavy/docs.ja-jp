## <a name="example"></a><span data-ttu-id="973af-113">例</span><span class="sxs-lookup"><span data-stu-id="973af-113">Example</span></span>

<span data-ttu-id="973af-114">次の例では、bc42025 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="973af-114">The following example generates bc42025:</span></span>

```vb

    Public Shared Sub SayHello()

    End Sub
End Class

    Public Sub Main()
        Dim tc As New TestClass()
        tc.SayHello() ' BC42025.
    End Sub
End Module
```

## <a name="to-address-this-warning"></a><span data-ttu-id="973af-115">この警告に対処するには</span><span class="sxs-lookup"><span data-stu-id="973af-115">To address this warning</span></span>
