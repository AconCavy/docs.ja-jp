## <a name="example"></a><span data-ttu-id="6ac89-113">例</span><span class="sxs-lookup"><span data-stu-id="6ac89-113">Example</span></span>

<span data-ttu-id="6ac89-114">次の例では bc42025 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="6ac89-114">The following example generates bc42025:</span></span>

```vb
Public Class TestClass
    Public Shared Sub SayHello()
        Console.WriteLine("Hello")
    End Sub
End Class

Module Program
    Public Sub Main()
        Dim tc As New TestClass()
        tc.SayHello() ' BC42025.
    End Sub
End Module
```

## <a name="to-address-this-warning"></a><span data-ttu-id="6ac89-115">この警告に対処するには</span><span class="sxs-lookup"><span data-stu-id="6ac89-115">To address this warning</span></span>

<span data-ttu-id="6ac89-116">次の例に示すように、`Shared` メンバーを定義するクラスまたは構造体の名前を使用して、それにアクセスします。</span><span class="sxs-lookup"><span data-stu-id="6ac89-116">Use the name of the class or structure that defines the `Shared` member to access it, as shown in the following example:</span></span>

```vb

    Public Shared Sub SayHello()
        Console.WriteLine("Hello")
    End Sub
End Class

Module Program
    Public Sub Main()
        TestClass.SayHello()
    End Sub
End Module
```

> [!NOTE]
