# <a name="statement-is-not-valid-inside-a-methodmultiline-lambda"></a>メソッドや複数行のラムダの内部では有効でないステートメントです。
ステートメントが内で無効、 `Sub`、 `Function`、プロパティ`Get`、またはプロパティ`Set`プロシージャ。 モジュールまたはクラス レベルでは、いくつかのステートメントを配置できます。 他のユーザーなど`Option Strict`、する必要があります名前空間レベルであるし、その他のすべての宣言の前にします。  
  
 **エラー ID:** BC30024  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- ステートメントをプロシージャから削除します。
