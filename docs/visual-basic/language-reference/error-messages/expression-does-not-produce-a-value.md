# <a name="expression-does-not-produce-a-value"></a>式は値を生成しません。
呼び出しなどの価値を生み出すのコンテキスト内の値を生成しない式を使用しようとしている、`Sub`コンテキストで場所、`Function`が必要です。  
  
 **エラー ID:** BC30491  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 値を生成するには、式を変更します。
