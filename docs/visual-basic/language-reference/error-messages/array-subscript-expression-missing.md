# <a name="array-subscript-expression-missing"></a>配列インデックス式が見つかりません。
配列の初期化は、1 つ以上の配列の境界を定義するための添字を残します。 たとえば、ステートメントは、式を含めることが`myArray (5,5,,10)`、3 番目の添字をままになります。  
  
 **エラー ID:** BC30306  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 不足しているインデックスを指定します。
