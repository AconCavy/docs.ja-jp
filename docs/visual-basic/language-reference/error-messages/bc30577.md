# <a name="bc30577-addressof-operand-must-be-the-name-of-a-method-without-parentheses"></a><span data-ttu-id="2e03f-102">BC30577:'AddressOf' オペランドはメソッドの名前でなければなりません。かっこは不要です。</span><span class="sxs-lookup"><span data-stu-id="2e03f-102">BC30577: 'AddressOf' operand must be the name of a method (without parentheses)</span></span>

<span data-ttu-id="2e03f-103">`AddressOf` 演算子は、特定のプロシージャを参照するプロシージャ デリゲート インスタンスを作成します。</span><span class="sxs-lookup"><span data-stu-id="2e03f-103">The `AddressOf` operator creates a procedure delegate instance that references a specific procedure.</span></span> <span data-ttu-id="2e03f-104">構文は次のとおりです。</span><span class="sxs-lookup"><span data-stu-id="2e03f-104">The syntax is as follows:</span></span>

```vb
AddressOf procedurename
```

<span data-ttu-id="2e03f-105">`AddressOf` の後に引数を囲むかっこを挿入していますが、ここでは何も必要ありません。</span><span class="sxs-lookup"><span data-stu-id="2e03f-105">You inserted parentheses around the argument following `AddressOf`, where none are needed.</span></span>

<span data-ttu-id="2e03f-106">**エラー ID:** BC30577</span><span class="sxs-lookup"><span data-stu-id="2e03f-106">**Error ID:** BC30577</span></span>

## <a name="example"></a><span data-ttu-id="2e03f-107">例</span><span class="sxs-lookup"><span data-stu-id="2e03f-107">Example</span></span>

<span data-ttu-id="2e03f-108">次の例では bc30577 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="2e03f-108">The following example generates bc30577:</span></span>

```vb
Public Sub CountZeroToTen()
    For i = 0 To 10
        Console.WriteLine($"Counted: {i}")
        Threading.Thread.Sleep(500)
    Next
End Sub

Sub Main()
    ' Any of the following two lines generates bc30577.
    'Dim t As New Threading.Thread(AddressOf(CountZeroToTen))
    'Dim t As New Threading.Thread(AddressOf CountZeroToTen())
    t.Start()
End Sub
```

## <a name="to-correct-this-error"></a><span data-ttu-id="2e03f-109">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="2e03f-109">To correct this error</span></span>

1. <span data-ttu-id="2e03f-110">次の例に示すように、`AddressOf` の後の引数を囲むかっこを削除します。</span><span class="sxs-lookup"><span data-stu-id="2e03f-110">Remove the parentheses around the argument following `AddressOf` as shown in the following example:</span></span>

    ```vb
    Public Sub CountZeroToTen()
        For i = 0 To 10
            Console.WriteLine($"Counted: {i}")
            Threading.Thread.Sleep(500)
        Next
    End Sub

    Sub Main()
        Dim t As New Threading.Thread(AddressOf CountZeroToTen)
        t.Start()
    End Sub
    ```

2. <span data-ttu-id="2e03f-111">引数がメソッド名であることを確認してください。</span><span class="sxs-lookup"><span data-stu-id="2e03f-111">Make sure the argument is a method name.</span></span>
