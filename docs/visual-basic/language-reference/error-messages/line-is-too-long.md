# <a name="line-is-too-long"></a>行が長すぎます。

ソース テキスト行は、65535 文字以内にする必要があります。  
  
 **エラー ID:** BC30494  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 行の長さを 65535 文字以下まで短くします。
