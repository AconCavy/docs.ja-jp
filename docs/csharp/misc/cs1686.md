## <a name="example"></a><span data-ttu-id="0c9d4-105">例</span><span class="sxs-lookup"><span data-stu-id="0c9d4-105">Example</span></span>  

 <span data-ttu-id="0c9d4-106">次の例では CS1686 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="0c9d4-106">The following sample generates CS1686.</span></span>  
  
```csharp  
// CS1686.cs  
// compile with: /unsafe /target:library  
class MyClass  
{  
   public unsafe delegate int * MyDelegate();  
  
   public unsafe int * Test()  
   {  
      int j = 0;  
      MyDelegate d = delegate { return &j; };   // CS1686  
      return &j;   // OK  
   }  
}  
```
