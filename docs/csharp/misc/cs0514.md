## <a name="example"></a><span data-ttu-id="1feb0-108">例</span><span class="sxs-lookup"><span data-stu-id="1feb0-108">Example</span></span>  
 <span data-ttu-id="1feb0-109">次の例では CS0514 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="1feb0-109">The following example generates CS0514:</span></span>  
  
```csharp  
// CS0514.cs

    {  
    }  
  
    public A(object o)  
    {  
    }  
}

{  
    static B() : this(null) // CS0514  
    {  
    }  
  
    public B(object o)  
    {  
    }  
}  
```
