<span data-ttu-id="6ea7a-104">エイリアスが型を参照しているため、エイリアス '識別子' を '::' と使用できません。</span><span class="sxs-lookup"><span data-stu-id="6ea7a-104">Cannot use alias 'identifier' with '::' since the alias references a type.</span></span> <span data-ttu-id="6ea7a-105">代わりに、'.' を使用してください。</span><span class="sxs-lookup"><span data-stu-id="6ea7a-105">Use '.' instead.</span></span>

 <span data-ttu-id="6ea7a-108">次の例では CS0431 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="6ea7a-108">The following sample generates CS0431:</span></span>  
  
```csharp  
// CS0431.cs  
using A = Outer;  
  
public class Outer
{  
   public class Inner
   {  
      public static void Meth() {}  
   }  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      A::Inner.Meth();   // CS0431  
      A.Inner.Meth();   // OK  
   }  
}  
```
