## <a name="example"></a><span data-ttu-id="2f755-107">例</span><span class="sxs-lookup"><span data-stu-id="2f755-107">Example</span></span>  
 <span data-ttu-id="2f755-108">次の例では CS0136 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="2f755-108">The following sample generates CS0136:</span></span>  
  
```csharp
// CS0136.cs

{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         {  
            char i = 'a';   // CS0136, hides int i  
         }  
         i++;  
      }  
   }  
}  
```
