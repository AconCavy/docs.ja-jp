# <a name="compiler-warning-level-1-cs1592"></a>コンパイラの警告 (レベル 1) CS1592
組み込みコメント ファイルの中の XML 形式が正しくありません -- 'reason'  
  
 ***理由***として報告された問題が、 [ \<include >](../programming-guide/xmldoc/include.md)タグによって指定されたファイルに見つかりました。
