# <a name="compiler-warning-level-1-cs2023"></a>コンパイラの警告 (レベル 1) CS2023
応答ファイルで指定されているため、/noconfig オプションを無視します  
  
 [/noconfig](../../csharp/language-reference/compiler-options/noconfig-compiler-option.md) コンパイラ オプションが応答ファイルに指定されましたが、これは許可されていません。  
  
 この警告は [/nowarn](../../csharp/language-reference/compiler-options/nowarn-compiler-option.md) オプションで抑制できません。
