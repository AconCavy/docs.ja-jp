## <a name="example"></a><span data-ttu-id="44549-106">例</span><span class="sxs-lookup"><span data-stu-id="44549-106">Example</span></span>  
 <span data-ttu-id="44549-107">次の例では、CS0440 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="44549-107">The following example generates CS0440:</span></span>  
  
```csharp  
// CS0440.cs  
// Compile with: /W:1  
  
using global = MyClass;   // CS0440  
class MyClass  
{  
    static void Main()  
    {  
        // Note how global refers to the global namespace  
        // even though it is redefined above.  
        global::System.Console.WriteLine();  
    }  
}  
```
