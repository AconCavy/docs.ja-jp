# <a name="compiler-error-cs0170"></a><span data-ttu-id="11e52-103">コンパイラ エラー CS0170</span><span class="sxs-lookup"><span data-stu-id="11e52-103">Compiler Error CS0170</span></span>

<span data-ttu-id="11e52-104">フィールド 'field' は、割り当てられていない可能性があります</span><span class="sxs-lookup"><span data-stu-id="11e52-104">Use of possibly unassigned field 'field'</span></span>  
  
 <span data-ttu-id="11e52-105">構造内のフィールドが、最初に初期化しないで使用されています。</span><span class="sxs-lookup"><span data-stu-id="11e52-105">A field in a structure was used without first being initialized.</span></span> <span data-ttu-id="11e52-106">この問題を解決するには、初期化されていないフィールドをまず確認し、そのフィールドにアクセスする前に初期化します。</span><span class="sxs-lookup"><span data-stu-id="11e52-106">To solve this problem, first determine which field was uninitialized and then initialize it before you try to access it.</span></span> <span data-ttu-id="11e52-107">構造体の初期化の詳細については、「 [構造体型](../language-reference/builtin-types/struct.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="11e52-107">For more information about initializing structs, see [Structure types](../language-reference/builtin-types/struct.md).</span></span>
  
 <span data-ttu-id="11e52-108">次の例では CS0170 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="11e52-108">The following sample generates CS0170:</span></span>  
  
```csharp  
// CS0170.cs  
public struct error  
{  
   public int i;  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      error e;  
      // uncomment the next line to resolve this error  
      // e.i = 0;  
      System.Console.WriteLine( e.i );   // CS0170 because
                                         //e.i was never assigned  
   }  
}  
```
