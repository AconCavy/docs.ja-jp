# <a name="compiler-error-cs1673"></a><span data-ttu-id="52304-102">コンパイラ エラー CS1673</span><span class="sxs-lookup"><span data-stu-id="52304-102">Compiler Error CS1673</span></span>

<span data-ttu-id="52304-103">構造体内の匿名メソッド、ラムダ式、およびクエリ式は、'this' のインスタンス メンバーにアクセスできません。</span><span class="sxs-lookup"><span data-stu-id="52304-103">Anonymous methods, lambda expressions, and query expressions inside structs cannot access instance members of 'this'.</span></span> <span data-ttu-id="52304-104">'this' を匿名メソッド、ラムダ式、またはクエリ式の外部のローカル変数にコピーし、このローカルを代わりに使用します。</span><span class="sxs-lookup"><span data-stu-id="52304-104">Consider copying 'this' to a local variable outside the anonymous method, lambda expression or query expression and using the local instead.</span></span>  
  
 <span data-ttu-id="52304-105">次の例では CS1673 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="52304-105">The following sample generates CS1673:</span></span>  
  
```csharp  
// CS1673.cs  
delegate int MyDelegate();  
  
public struct S  
{  
   int member;  
  
   public int F(int i)  
   {  
       member = i;  
       // Try assigning to a local variable  
       // S s = this;  
       MyDelegate d = delegate()  
       {  
          i = this.member;  // CS1673  
          // And use the local variable instead of "this"  
          // i =  s.member;  
          return i;  
  
       };  
       return d();  
   }  
}  
  
class CMain  
{  
   public static void Main()  
   {  
   }  
}  
```
