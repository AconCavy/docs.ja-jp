# <a name="compiler-error-cs0013"></a>コンパイラ エラー CS0013
メタデータをファイル 'ModelStore.dll' に書き込み中に予期しないエラーが発生しました -- 'ユーザー文字列をこれ以上作成する論理空間が残っていません。'  
  
 .NET ランタイムはメタデータを出力しませんでした。 パスが正しく指定されていることと、ディスクの空き容量が十分に残っていることを確認します。 問題が引き続き発生する場合は、Visual Studio または .NET の修復または再インストールが必要になることがあります。
