# <a name="compiler-error-cs1013"></a>コンパイラ エラー CS1013
無効な数値です。  
  
 正しくない形式の数値が検出されました。 整数型の詳細については、[整数型テーブル](../language-reference/builtin-types/integral-numeric-types.md) に関するページを参照してください。  
  
## <a name="example"></a>例  
 次の例では CS1013 が生成されます。  
  
```csharp  
// CS1013.cs  
class Sample  
{  
    static void Main()  
    {  
        int i = 0x;   // CS1013  
    }  
}  
```
