# <a name="compiler-error-cs0451"></a><span data-ttu-id="2fde8-102">コンパイラ エラー CS0451</span><span class="sxs-lookup"><span data-stu-id="2fde8-102">Compiler Error CS0451</span></span>
<span data-ttu-id="2fde8-103">'new()' 制約は 'struct' 制約と一緒には使用できません。</span><span class="sxs-lookup"><span data-stu-id="2fde8-103">The 'new()' constraint cannot be used with the 'struct' constraint</span></span>  
  
 <span data-ttu-id="2fde8-104">ジェネリック型に対して制約を指定するとき、 `new()` 制約は、クラス型制約、インターフェイス型制約、参照型制約、型パラメーター制約と組み合わせた場合のみ使用できます。値型の制約と組み合わせて使用することはできません。</span><span class="sxs-lookup"><span data-stu-id="2fde8-104">When specifying constraints on the type of a generic, the `new()` constraint may only be used with class type constraints, interface type constraints, reference type constraints, and type parameter constraints, but not with value type constraints.</span></span>  
  
## <a name="example"></a><span data-ttu-id="2fde8-105">例</span><span class="sxs-lookup"><span data-stu-id="2fde8-105">Example</span></span>  
 <span data-ttu-id="2fde8-106">次の例では CS0451 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="2fde8-106">The following sample generates CS0451.</span></span>  
  
```csharp  
// CS0451.cs  
using System;  
public class C4
{  
   public void F4<T>() where T : struct, new() {}   // CS0451  
}  
  
// OK  
public class C5  
{  
   public void F5<T>() where T : struct {}  
}  
  
public class C6  
{  
   public void F6<T>() where T : new() {}  
}  
```
