# <a name="compiler-warning-level-3-cs0219"></a><span data-ttu-id="63f83-102">コンパイラの警告 (レベル 3) CS0219</span><span class="sxs-lookup"><span data-stu-id="63f83-102">Compiler Warning (level 3) CS0219</span></span>
<span data-ttu-id="63f83-103">変数 'variable' が割り当てられていますが、その値は使用されていません</span><span class="sxs-lookup"><span data-stu-id="63f83-103">The variable 'variable' is assigned but its value is never used</span></span>

 <span data-ttu-id="63f83-104">変数を宣言して代入するときにレベル 3 の警告が発行されますが、その変数は使用されません。</span><span class="sxs-lookup"><span data-stu-id="63f83-104">The compiler issues a level-three warning, when you declare and assign a variable, but do not use it.</span></span>

 > [!NOTE]
 > <span data-ttu-id="63f83-105">コンパイラは、変数値がコンパイル時定数である場合にのみ、この警告を生成します。</span><span class="sxs-lookup"><span data-stu-id="63f83-105">The compiler generates this warning only when the variable value is a compile-time constant.</span></span> <span data-ttu-id="63f83-106">非定数式またはメソッドの結果をローカル変数に割り当てると、デバッガーでこれらの式を簡単に確認できます。</span><span class="sxs-lookup"><span data-stu-id="63f83-106">Assigning a non-constant expression or method result to a local variable makes it easier to observe those expressions in the debugger.</span></span> <span data-ttu-id="63f83-107">また、結果に到達可能になり、変数に到達可能な間にガベージ コレクションを回避します。</span><span class="sxs-lookup"><span data-stu-id="63f83-107">It also makes the result reachable, preventing garbage collection while that variable is reachable.</span></span>

 <span data-ttu-id="63f83-108">次のサンプルは、警告が生成された場合と生成されない場合を示しています。</span><span class="sxs-lookup"><span data-stu-id="63f83-108">The following sample shows the cases when and when not the warning is generated:</span></span>

```csharp
// CS0219.cs
// compile with: /W:3
public class MyClass
{
   public static void Main()
   {
      var interpolated = "Interpolated";
      var a = 0; // CS0219
      int b = GetZero(); // Doesn't generate a warning.
      var c = "Regular string"; // CS0219
      var d = $"Constant interpolated string"; // Doesn't generate a warning.
      var e = $"{interpolated} string"; // Doesn't generate a warning.
   }

   private static int GetZero()
   {
       return 0;
   }
}  
```
