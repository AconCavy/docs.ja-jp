# <a name="compiler-warning-level-3-cs0219"></a><span data-ttu-id="4bef5-103">コンパイラの警告 (レベル 3) CS0219</span><span class="sxs-lookup"><span data-stu-id="4bef5-103">Compiler Warning (level 3) CS0219</span></span>
<span data-ttu-id="4bef5-104">変数 'variable' が割り当てられていますが、その値は使用されていません</span><span class="sxs-lookup"><span data-stu-id="4bef5-104">The variable 'variable' is assigned but its value is never used</span></span>

 <span data-ttu-id="4bef5-105">変数を宣言して代入するときにレベル 3 の警告が発行されますが、その変数は使用されません。</span><span class="sxs-lookup"><span data-stu-id="4bef5-105">The compiler issues a level-three warning, when you declare and assign a variable, but do not use it.</span></span>

 > [!NOTE]
 > <span data-ttu-id="4bef5-106">コンパイラは、変数の値がコンパイル時の定数である場合にのみ、この警告を生成します。</span><span class="sxs-lookup"><span data-stu-id="4bef5-106">The compiler generates this warning only when the variable value is a compile-time constant.</span></span> <span data-ttu-id="4bef5-107">定数以外の式またはメソッドの結果をローカル変数に割り当てると、デバッガーでこれらの式を見やすくなります。</span><span class="sxs-lookup"><span data-stu-id="4bef5-107">Assigning a non-constant expression or method result to a local variable makes it easier to observe those expressions in the debugger.</span></span> <span data-ttu-id="4bef5-108">また、結果が到達可能になり、その変数に到達可能な状態でガベージコレクションが実行されなくなります。</span><span class="sxs-lookup"><span data-stu-id="4bef5-108">It also makes the result reachable, preventing garbage collection while that variable is reachable.</span></span>

 <span data-ttu-id="4bef5-109">次の例では、警告が生成されなかった場合との場合を示します。</span><span class="sxs-lookup"><span data-stu-id="4bef5-109">The following sample shows the cases when and when not the warning is generated:</span></span>

```csharp
// CS0219.cs
// compile with: /W:3
public class MyClass
{
   public static void Main()
   {
      var interpolated = "Interpolated";
      var a = 0; // CS0219
      int b = GetZero(); // Doesn't generate a warning.
      var c = "Regular string"; // CS0219
      var d = $"Constant interpolated string"; // Doesn't generate a warning.
      var e = $"{interpolated} string"; // Doesn't generate a warning.
   }

   private static int GetZero()
   {
       return 0;
   }
}  
```
