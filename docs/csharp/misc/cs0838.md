## <a name="example"></a><span data-ttu-id="4a14a-108">例</span><span class="sxs-lookup"><span data-stu-id="4a14a-108">Example</span></span>  
 <span data-ttu-id="4a14a-109">次の例では CS0838 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="4a14a-109">The following example generates CS0838:</span></span>

using System.Linq;  
using System.Linq.Expressions;  
  
namespace TestNamespace  
{  
    class Test  
    {  
        static int Main()  
        {

                () => new int[2, 2] { { 1, 2 }, { 3, 4 } }; // CS0838

            return 1;  
        }  
    }  
}
