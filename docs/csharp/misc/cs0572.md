# <a name="compiler-error-cs0572"></a><span data-ttu-id="f289e-103">コンパイラ エラー CS0572</span><span class="sxs-lookup"><span data-stu-id="f289e-103">Compiler Error CS0572</span></span>
<span data-ttu-id="f289e-104">'type' : 式から型を参照することはできません。'path_to_type' を使用してください。</span><span class="sxs-lookup"><span data-stu-id="f289e-104">'type' : cannot reference a type through an expression; try 'path_to_type' instead</span></span>  
  
 <span data-ttu-id="f289e-105">識別子を通じてクラスのメンバーにアクセスしようとしましたが、この場合は許可されていません。</span><span class="sxs-lookup"><span data-stu-id="f289e-105">An attempt was made to access a member of a class through an identifier, which is not permitted in this situation.</span></span>  
  
 <span data-ttu-id="f289e-106">次の例では CS0572 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="f289e-106">The following sample generates CS0572:</span></span>  
  
```csharp  
// CS0572.cs  
using System;  
class C  
{  
   public class Inner  
   {  
      public static int v = 9;  
   }  
}  
  
class D : C  
{  
   public static void Main()  
   {  
      C cValue = new C();  
      Console.WriteLine(cValue.Inner.v);   // CS0572  
      // try the following line instead  
      // Console.WriteLine(C.Inner.v);  
   }  
}  
```
