# <a name="compiler-error-cs1630"></a>コンパイラ エラー CS1630
/errorreport に対する無効なオプション 'option' です。prompt、send、queue、または none のいずれかでなければなりません。  
  
 コマンド ライン オプション [/errorreport](../language-reference/compiler-options/errorreport-compiler-option.md) の後に、 **prompt**、 **send**、 **queue**、 **none**のいずれかを指定し、内部コンパイラ エラーが発生するときに行うアクションを示す必要があります。
