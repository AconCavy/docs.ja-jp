# <a name="compiler-error-cs1630"></a>コンパイラ エラー CS1630
/errorreport に対する無効なオプション 'option' です。prompt、send、queue、または none のいずれかでなければなりません。  
  
 コマンドラインオプション[errorreport](../language-reference/compiler-options/errorreport-compiler-option.md)の後に、**プロンプト**、**送信**、**キュー**、または**なし**を指定し、内部コンパイラエラーが発生したときに実行するアクションを指定する必要があります。
