# <a name="compiler-error-cs0243"></a><span data-ttu-id="8e4c9-102">コンパイラ エラー CS0243</span><span class="sxs-lookup"><span data-stu-id="8e4c9-102">Compiler error CS0243</span></span>

<span data-ttu-id="8e4c9-103">オーバーライド メソッドであるため、条件付き属性は 'method' では無効です</span><span class="sxs-lookup"><span data-stu-id="8e4c9-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
<span data-ttu-id="8e4c9-104">この<xref:System.Diagnostics.ConditionalAttribute>属性は[、override](../language-reference/keywords/override.md)キーワードでマークされているメソッドでは許可されません。</span><span class="sxs-lookup"><span data-stu-id="8e4c9-104">The <xref:System.Diagnostics.ConditionalAttribute> attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="8e4c9-105">詳細については、「[オーバーライドキーワードと新しいキーワードを使用するタイミングを知る](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="8e4c9-105">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
<span data-ttu-id="8e4c9-106">コンパイラは、オーバーライド メソッドにバインドすることはありません。</span><span class="sxs-lookup"><span data-stu-id="8e4c9-106">The compiler never binds to override methods.</span></span> <span data-ttu-id="8e4c9-107">基本メソッドにのみバインドされ、必要に応じて共通言語ランタイムがオーバーライドを呼び出します。</span><span class="sxs-lookup"><span data-stu-id="8e4c9-107">It only binds to the base method, and the common language runtime calls the override, as appropriate.</span></span>  
  
<span data-ttu-id="8e4c9-108">次のコードは CS0243 を生成します。</span><span class="sxs-lookup"><span data-stu-id="8e4c9-108">The following code generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
