# <a name="compiler-error-cs0243"></a><span data-ttu-id="0ae55-103">コンパイラエラー CS0243</span><span class="sxs-lookup"><span data-stu-id="0ae55-103">Compiler error CS0243</span></span>

<span data-ttu-id="0ae55-104">オーバーライド メソッドであるため、条件付き属性は 'method' では無効です</span><span class="sxs-lookup"><span data-stu-id="0ae55-104">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
<span data-ttu-id="0ae55-105"><xref:System.Diagnostics.ConditionalAttribute> [Override](../language-reference/keywords/override.md)キーワードでマークされているメソッドでは、属性は許可されません。</span><span class="sxs-lookup"><span data-stu-id="0ae55-105">The <xref:System.Diagnostics.ConditionalAttribute> attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="0ae55-106">詳細については、「 [Override キーワードと New キーワードを使用する場合](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md)について」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="0ae55-106">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
<span data-ttu-id="0ae55-107">コンパイラは、オーバーライドメソッドにバインドされません。</span><span class="sxs-lookup"><span data-stu-id="0ae55-107">The compiler never binds to override methods.</span></span> <span data-ttu-id="0ae55-108">基本メソッドのみにバインドされ、共通言語ランタイムは、必要に応じてオーバーライドを呼び出します。</span><span class="sxs-lookup"><span data-stu-id="0ae55-108">It only binds to the base method, and the common language runtime calls the override, as appropriate.</span></span>  
  
<span data-ttu-id="0ae55-109">次のコードでは、CS0243 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="0ae55-109">The following code generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
