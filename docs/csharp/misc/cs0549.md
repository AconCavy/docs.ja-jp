# <a name="compiler-error-cs0549"></a><span data-ttu-id="93bf1-103">コンパイラ エラー CS0549</span><span class="sxs-lookup"><span data-stu-id="93bf1-103">Compiler Error CS0549</span></span>

<span data-ttu-id="93bf1-104">'function' はシール クラス 'class' の新しい仮想メンバーです。</span><span class="sxs-lookup"><span data-stu-id="93bf1-104">'function' is a new virtual member in sealed class 'class'</span></span>  
  
 <span data-ttu-id="93bf1-105">[シール](../language-reference/keywords/sealed.md)[クラス](../language-reference/keywords/class.md)を基底クラスとして使用することはできません。</span><span class="sxs-lookup"><span data-stu-id="93bf1-105">A [sealed](../language-reference/keywords/sealed.md)[class](../language-reference/keywords/class.md) cannot be used as a base class.</span></span>  <span data-ttu-id="93bf1-106">したがって、シール クラスに仮想メソッドを使用しても意味がありません。</span><span class="sxs-lookup"><span data-stu-id="93bf1-106">Therefore, it is useless to have a virtual method in sealed class.</span></span>  
  
 <span data-ttu-id="93bf1-107">次の例では CS0549 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="93bf1-107">The following sample generates CS0549:</span></span>  
  
```csharp  
// CS0549.cs  
// compile with: /target:library  
sealed public class MyClass  
{  
   virtual public void TestMethod() {}   // CS0549  
   public void TestMethod2() {}   // OK  
}  
```
