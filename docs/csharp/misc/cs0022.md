# <a name="compiler-error-cs0022"></a>コンパイラ エラー CS0022
角かっこ [] 内のインデックス数が正しくありません。正しい数は 'number' です  
  
 配列アクセス操作で、角かっこ内に無効な次元数が指定されています。 詳細については、次を参照してください。[配列](../../csharp/programming-guide/arrays/index.md)します。  
  
## <a name="example"></a>例  
 次の例では CS0022 が生成されます。  
  
```csharp  
// CS0022.cs  
public class MyClass  
{  
    public static void Main()  
    {  
        int[] a = new int[10];  
        a[0] = 0;     // single-dimension array  
        a[0,1] = 9;   // CS0022, the array does not have two dimensions  
    }  
}  
```
