## <a name="example"></a><span data-ttu-id="247a5-109">例</span><span class="sxs-lookup"><span data-stu-id="247a5-109">Example</span></span>  
 <span data-ttu-id="247a5-110">次の例では CS0688 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="247a5-110">The following sample generates CS0688.</span></span> <span data-ttu-id="247a5-111">基底クラスを変更せずにこの警告を解決するには、オーバーライドするメソッドからセキュリティ属性を削除します。</span><span class="sxs-lookup"><span data-stu-id="247a5-111">To resolve the warning without modifying the base class, remove the security attribute from the overriding method.</span></span> <span data-ttu-id="247a5-112">この操作によって、セキュリティ上の問題は解決しません。</span><span class="sxs-lookup"><span data-stu-id="247a5-112">This will not solve the security problem.</span></span>  
  
```csharp  
// CS0688.cs  
// compile with: /W:1  
using System;  
using System.Security.Permissions;  
  
class Base
{  
    //Uncomment the following line to close the security hole  
    //[FileIOPermission(SecurityAction.LinkDemand, All=@"C:\\")]  
    public virtual void DoScaryFileStuff()  
    {  
    }  
}  
  
class Derived: Base  
{  
    [FileIOPermission(SecurityAction.LinkDemand, All=@"C:\\")] // CS0688  
    public override void DoScaryFileStuff()  
    {  
    }  
    static void Main()  
    {  
    }  
}  
```
