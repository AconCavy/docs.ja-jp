# <a name="compiler-error-cs1594"></a><span data-ttu-id="3c940-102">コンパイラ エラー CS1594</span><span class="sxs-lookup"><span data-stu-id="3c940-102">Compiler Error CS1594</span></span>
<span data-ttu-id="3c940-103">デリゲート 'delegate' に無効な引数があります</span><span class="sxs-lookup"><span data-stu-id="3c940-103">Delegate 'delegate' has some invalid arguments</span></span>  
  
 <span data-ttu-id="3c940-104">[デリゲート](../language-reference/builtin-types/reference-types.md) 呼び出しに渡される引数の型は、デリゲート宣言のパラメーターの型と一致しません。</span><span class="sxs-lookup"><span data-stu-id="3c940-104">The type of an argument passed to a [delegate](../language-reference/builtin-types/reference-types.md) invocation does not agree with the type of the parameter in the delegate declaration.</span></span>  
  
 <span data-ttu-id="3c940-105">次の例では CS1594 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="3c940-105">The following sample generates CS1594:</span></span>  
  
```csharp  
// CS1594.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello("8");   // CS1594  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
