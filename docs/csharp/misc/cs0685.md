# <a name="compiler-error-cs0685"></a><span data-ttu-id="359ad-103">コンパイラ エラー CS0685</span><span class="sxs-lookup"><span data-stu-id="359ad-103">Compiler Error CS0685</span></span>
<span data-ttu-id="359ad-104">条件付きメンバー 'member' には out パラメーターを指定できません</span><span class="sxs-lookup"><span data-stu-id="359ad-104">Conditional member 'member' cannot have an out parameter</span></span>  
  
 <span data-ttu-id="359ad-105">メソッドで <xref:System.Diagnostics.ConditionalAttribute> 属性を使用する場合、そのメソッドに out パラメーターがない可能性があります。</span><span class="sxs-lookup"><span data-stu-id="359ad-105">When using the <xref:System.Diagnostics.ConditionalAttribute> attribute on a method, that method may not have an out parameter.</span></span> <span data-ttu-id="359ad-106">これは、メソッド呼び出しが nothing にコンパイルされる場合、out パラメーターに使用される変数の値が定義されないためです。</span><span class="sxs-lookup"><span data-stu-id="359ad-106">This is because the value of the variable used for the out parameter would not be defined in the case that the method call is compiled to nothing.</span></span> <span data-ttu-id="359ad-107">このエラーを避けるためには、条件付きのメソッド宣言から out パラメーターを削除するか、条件付きの属性を使用しないでください。</span><span class="sxs-lookup"><span data-stu-id="359ad-107">To avoid this error, remove the out parameter from the conditional method declaration, or don't use the Conditional Attribute.</span></span>  
  
## <a name="example"></a><span data-ttu-id="359ad-108">例</span><span class="sxs-lookup"><span data-stu-id="359ad-108">Example</span></span>
