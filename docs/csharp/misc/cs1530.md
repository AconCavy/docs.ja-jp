# <a name="compiler-error-cs1530"></a>コンパイラ エラー CS1530
キーワード 'new' は、名前空間で定義された要素に対して使用できません  
  
 [名前空間](../../csharp/language-reference/keywords/new-modifier.md) の構成要素では、 [new](../../csharp/language-reference/keywords/namespace.md)キーワードを指定する必要はありません。  
  
 次の例では CS1530 が生成されます。  
  
```csharp  
// CS1530.cs  
namespace a  
{  
   new class i   // CS1530  
   {  
   }  
  
   // try the following instead  
   class ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
