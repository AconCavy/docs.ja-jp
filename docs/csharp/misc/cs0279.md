## <a name="example"></a><span data-ttu-id="468ed-110">例</span><span class="sxs-lookup"><span data-stu-id="468ed-110">Example</span></span>  
 <span data-ttu-id="468ed-111">次の例では、CS0279 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="468ed-111">The following example generates CS0279:</span></span>  
  
```csharp  
// CS0279.cs  
  
using System;  
using System.Collections;  
  
public class myTest : IEnumerable  
{  
    IEnumerator IEnumerable.GetEnumerator()  
    {  
        yield return 0;  
    }  
  
    internal IEnumerator GetEnumerator()  
    {  
        yield return 0;  
    }  
  
    public static void Main()  
    {  
        foreach (int i in new myTest()) {}  // CS0279  
    }  
}  
```
