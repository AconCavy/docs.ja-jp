# <a name="compiler-error-cs0081"></a><span data-ttu-id="e2a5e-103">コンパイラ エラー CS0081</span><span class="sxs-lookup"><span data-stu-id="e2a5e-103">Compiler Error CS0081</span></span>

<span data-ttu-id="e2a5e-104">型パラメーターの宣言は型ではなく識別子でなければなりません。</span><span class="sxs-lookup"><span data-stu-id="e2a5e-104">Type parameter declaration must be an identifier not a type</span></span>  
  
 <span data-ttu-id="e2a5e-105">ジェネリック メソッドまたは型を宣言する場合は、"T" や "inputType" など、識別子として型パラメーターを指定します。</span><span class="sxs-lookup"><span data-stu-id="e2a5e-105">When you declare a generic method or type, specify the type parameter as an identifier, for example "T" or "inputType".</span></span> <span data-ttu-id="e2a5e-106">クライアント コードでメソッドを呼び出すと型が指定されます。これにより、メソッドまたはクラス本体に出現する各識別子が置き換えられます。</span><span class="sxs-lookup"><span data-stu-id="e2a5e-106">When client code calls the method, it supplies the type, which replaces each occurrence of the identifier in the method or class body.</span></span> <span data-ttu-id="e2a5e-107">詳細については、「 [ジェネリック型パラメーター](../programming-guide/generics/generic-type-parameters.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="e2a5e-107">For more information, see [Generic Type Parameters](../programming-guide/generics/generic-type-parameters.md).</span></span>  
  
```csharp  
// CS0081.cs  
class MyClass  
{  
   public void F<int>() {}   // CS0081  
   public void F<T>(T input) {}   // OK  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
      a.F<int>(2);  
      a.F<double>(.05);  
   }  
}  
```
