# <a name="compiler-error-cs2011"></a>コンパイラ エラー CS2011

応答ファイル 'file' を開けません  
  
 コンパイルで応答ファイルが指定されましたが、コンパイラではそのファイルを見つけて開くことができませんでした。  
  
 応答ファイルについて詳しくは、「 [@ (応答ファイルの指定)](../language-reference/compiler-options/response-file-compiler-option.md)」をご覧ください。
