# <a name="compiler-error-cs0663"></a><span data-ttu-id="ef77c-103">コンパイラ エラー CS0663</span><span class="sxs-lookup"><span data-stu-id="ef77c-103">Compiler Error CS0663</span></span>
<span data-ttu-id="ef77c-104">ref と out でのみ異なるオーバーロードされたメソッドを定義することはできません。</span><span class="sxs-lookup"><span data-stu-id="ef77c-104">Cannot define overloaded methods that differ only on ref and out.</span></span>  
  
 <span data-ttu-id="ef77c-105">パラメーターの [in](../language-reference/keywords/in-parameter-modifier.md)、 [ref](../language-reference/keywords/ref.md) 、および [out](../language-reference/keywords/out-parameter-modifier.md) の使用のみが異なるメソッドは使用できません。</span><span class="sxs-lookup"><span data-stu-id="ef77c-105">Methods that differ only on their use of [in](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) and [out](../language-reference/keywords/out-parameter-modifier.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="ef77c-106">次の例では CS0663 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="ef77c-106">The following sample generates CS0663:</span></span>  
  
```csharp  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
