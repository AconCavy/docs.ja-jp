# <a name="compiler-warning-level-2-cs0253"></a><span data-ttu-id="f1509-103">コンパイラの警告 (レベル 2) CS0253</span><span class="sxs-lookup"><span data-stu-id="f1509-103">Compiler Warning (level 2) CS0253</span></span>
<span data-ttu-id="f1509-104">予期しない参照比較です。比較値を取得するには型 'type' に右辺をキャストしてください</span><span class="sxs-lookup"><span data-stu-id="f1509-104">Possible unintended reference comparison; to get a value comparison, cast the right hand side to type 'type'</span></span>  
  
 <span data-ttu-id="f1509-105">コンパイラは参照比較を行っています。</span><span class="sxs-lookup"><span data-stu-id="f1509-105">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="f1509-106">文字列の値を比較する場合は、式の右辺を `type`にキャストしてください。</span><span class="sxs-lookup"><span data-stu-id="f1509-106">If you want to compare the value of strings, cast the right side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="f1509-107">次の例では CS0253 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="f1509-107">The following sample generates CS0253:</span></span>  
  
```csharp
// CS0253.cs  
// compile with: /W:2  
using System;  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool c = s == o;   // CS0253  
      // try the following line instead  
      // bool c = s == (string)o;  
   }  
}  
```
