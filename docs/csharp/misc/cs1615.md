# <a name="compiler-error-cs1615"></a><span data-ttu-id="e2eed-103">コンパイラ エラー CS1615</span><span class="sxs-lookup"><span data-stu-id="e2eed-103">Compiler Error CS1615</span></span>
<span data-ttu-id="e2eed-104">引数 'number' を 'keyword' キーワードと共に渡すことはできません</span><span class="sxs-lookup"><span data-stu-id="e2eed-104">Argument 'number' should not be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="e2eed-105">引数に `ref` または **out** パラメーターを受け取らない関数で、キーワード `ref` または **out** のどちらかが使用されました。</span><span class="sxs-lookup"><span data-stu-id="e2eed-105">One of the keywords `ref` or **out** was used when the function did not take a `ref` or **out** parameter for that argument.</span></span> <span data-ttu-id="e2eed-106">このエラーを解決するには、正しくないキーワードを削除し、関数の宣言と一致する適切なキーワードを使用します (存在する場合)。</span><span class="sxs-lookup"><span data-stu-id="e2eed-106">To resolve this error, remove the incorrect keyword and use the appropriate keyword that matches the function declaration, if any.</span></span>  
  
 <span data-ttu-id="e2eed-107">次の例では CS1615 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="e2eed-107">The following sample generates CS1615:</span></span>  
  
```csharp  
// CS1615.cs  
class C  
{  
   public void f(int i) {}  
   public static void Main()  
   {  
      int i = 1;  
      f(ref i);  // CS1615  
   }  
}  
```
