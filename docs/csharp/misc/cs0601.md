# <a name="compiler-error-cs0601"></a><span data-ttu-id="a0447-103">コンパイラ エラー CS0601</span><span class="sxs-lookup"><span data-stu-id="a0447-103">Compiler Error CS0601</span></span>
<span data-ttu-id="a0447-104">static または extern に指定されているメソッドでは、DllImport 属性を指定する必要があります</span><span class="sxs-lookup"><span data-stu-id="a0447-104">The DllImport attribute must be specified on a method marked 'static' and 'extern'</span></span>  
  
 <span data-ttu-id="a0447-105">`DllImport` 属性は、適切なアクセス キーワードが含まれていないメソッドで使用されていました。</span><span class="sxs-lookup"><span data-stu-id="a0447-105">The `DllImport` attribute was used on a method that did not have the correct access keywords.</span></span>  
  
 <span data-ttu-id="a0447-106">次の例では CS0601 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="a0447-106">The following sample generates CS0601:</span></span>  
  
```csharp  
// CS0601.cs  
using System.Runtime.InteropServices;  
using System.Text;  
  
public class C  
{  
   [DllImport("KERNEL32.DLL")]  
   extern int GetCurDirectory(int bufSize, StringBuilder buf);   // CS0601  
   // Try the following line instead:  
   // static extern int GetCurDirectory(int bufSize, StringBuilder buf);  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
