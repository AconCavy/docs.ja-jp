# <a name="compiler-error-cs1583"></a>コンパイラ エラー CS1583
'file' は有効な Win32 のリソース ファイルではありません  
  
 このエラーは、 [-win32res](../language-reference/compiler-options/win32res-compiler-option.md)コンパイラオプションで、有効または正しく書式設定されていない Win32 リソースファイルのファイル名を指定した場合に発生します。 Visual Studio では、ファイル名はプロジェクト デザイナーのアプリケーション ウィンドウで指定されます。
