# <a name="compiler-error-cs0157"></a><span data-ttu-id="31270-103">コンパイラ エラー CS0157</span><span class="sxs-lookup"><span data-stu-id="31270-103">Compiler Error CS0157</span></span>

<span data-ttu-id="31270-104">コントロールが finally 句の本体から出られません。</span><span class="sxs-lookup"><span data-stu-id="31270-104">Control cannot leave the body of a finally clause</span></span>  
  
 <span data-ttu-id="31270-105">[finally](../language-reference/keywords/try-catch-finally.md) 句のすべてのステートメントを実行する必要があります。</span><span class="sxs-lookup"><span data-stu-id="31270-105">All of the statements in a [finally](../language-reference/keywords/try-catch-finally.md) clause must execute.</span></span> <span data-ttu-id="31270-106">詳細については、「 [例外と例外処理](../programming-guide/exceptions/index.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="31270-106">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="31270-107">次の例では CS0157 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="31270-107">The following sample generates CS0157:</span></span>  
  
```csharp  
// CS0157.cs  
using System;  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
         }  
  
         finally  
         {  
            return;   // CS0157, cannot leave finally clause  
         }  
      }  
   }  
}  
```
