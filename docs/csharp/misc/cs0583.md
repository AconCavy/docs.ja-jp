# <a name="compiler-error-cs0583"></a>コンパイラ エラー CS0583
内部コンパイラ エラー コンパイラで内部エラーが発生しました。 この問題を回避するには、以下にリストされている場所付近のプログラムを単純化するか変更してください。 一覧の上部の場所は、内部エラーが発生したポイントに近い場所です。 このようなエラーは、/errorreport オプションを使用して Microsoft に報告することができます。  
  
 '/bugreport' オプションを使用してバグ報告ファイルを作成し、割り当てられている問題報告連絡先にレポートを送信します。  
  
 [/bugreport](../language-reference/compiler-options/bugreport-compiler-option.md) コンパイラ オプションを使用して、問題をキャプチャします。
