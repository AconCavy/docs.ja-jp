# <a name="compiler-warning-level-1-cs3018"></a><span data-ttu-id="5bc79-102">コンパイラの警告 (レベル 1) CS3018</span><span class="sxs-lookup"><span data-stu-id="5bc79-102">Compiler Warning (level 1) CS3018</span></span>

<span data-ttu-id="5bc79-103">CLS に準拠していない型 'type' のメンバーであるため、'type' を CLS 準拠として設定できません</span><span class="sxs-lookup"><span data-stu-id="5bc79-103">'type' cannot be marked as CLS-Compliant because it is a member of non CLS-compliant type 'type'</span></span>  
  
 <span data-ttu-id="5bc79-104">この警告は、CLSCompliant 属性が `true` に設定された入れ子のクラスが、CLSCompliant 属性が `false`に設定されたクラスのメンバーとして宣言されている場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="5bc79-104">This warning occurs if a nested class with the CLSCompliant attribute set to `true` is declared as a member of a class declared with the CLSCompliant attribute set to `false`.</span></span> <span data-ttu-id="5bc79-105">CLS 準拠ではない外部クラスのメンバーである場合、入れ子になったクラスを CLS 準拠にすることはできないため、このような記述は使用できません。</span><span class="sxs-lookup"><span data-stu-id="5bc79-105">This is not allowed, since a nested class cannot be CLS-compliant if it is a member of an outer class that is not CLS-compliant.</span></span> <span data-ttu-id="5bc79-106">この警告を解決するには、入れ子になったクラスから CLSCompliant 属性を削除するか、属性の設定を `true` から `false`に変更します。</span><span class="sxs-lookup"><span data-stu-id="5bc79-106">To resolve this warning, remove the CLSCompliant attribute from the nested class, or change it from `true` to `false`.</span></span> <span data-ttu-id="5bc79-107">CLS 準拠の詳細については、「言語への [非依存性と言語に依存しないコンポーネント](../../standard/language-independence.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="5bc79-107">For more information on CLS compliance, see [Language independence and language-independent components](../../standard/language-independence.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="5bc79-108">例</span><span class="sxs-lookup"><span data-stu-id="5bc79-108">Example</span></span>  

 <span data-ttu-id="5bc79-109">次の例では CS3018 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="5bc79-109">The following sample generates CS3018.</span></span>  
  
```csharp  
// CS3018.cs  
// compile with: /target:library  
using System;  
  
[assembly: CLSCompliant(true)]  
[CLSCompliant(false)]  
public class Outer  
{  
   [CLSCompliant(true)]   // CS3018  
   public class Nested {}  
  
   // OK  
   public class Nested2 {}  
  
   [CLSCompliant(false)]  
   public class Nested3 {}  
}  
```
