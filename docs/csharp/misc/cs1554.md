# <a name="compiler-error-cs1554"></a><span data-ttu-id="58acd-103">コンパイラ エラー CS1554</span><span class="sxs-lookup"><span data-stu-id="58acd-103">Compiler Error CS1554</span></span>
<span data-ttu-id="58acd-104">宣言が有効ではありません。' \<type> operator op (... ' を使用します。その代わりに</span><span class="sxs-lookup"><span data-stu-id="58acd-104">Declaration is not valid; use '\<type> operator op (...' instead</span></span>  
  
<span data-ttu-id="58acd-105">オーバーロードされた [演算子](../language-reference/operators/operator-overloading.md) の戻り値の型は、キーワードの前に記述する必要があり `operator` ます。</span><span class="sxs-lookup"><span data-stu-id="58acd-105">The return type of an [overloaded operator](../language-reference/operators/operator-overloading.md) must appear before the `operator` keyword.</span></span>

```csharp  
// CS1554.cs  
class MyClass  
{  
   public static operator ++ MyClass (MyClass f)    // CS1554  
   // try the following line instead  
   // public static MyClass operator ++ (MyClass f)  
   {  
      return new MyClass ();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
