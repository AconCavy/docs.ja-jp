# <a name="compiler-error-cs2022"></a>コンパイラ エラー CS2022
ソース ファイル名の前にオプション /out および /target を指定する必要があります。  
  
 [/out (出力ファイル名の設定)](../../csharp/language-reference/compiler-options/out-compiler-option.md) および [/target (出力ファイル形式の指定)](../../csharp/language-reference/compiler-options/target-compiler-option.md) コンパイラ オプションをコマンド ラインで指定する場合は、ソース コード ファイルの前に指定する必要があります。
