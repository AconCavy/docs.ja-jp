# <a name="compiler-error-cs1606"></a>コンパイラ エラー CS1606
アセンブリ署名ができませんでした。出力は署名されていない可能性があります -- 理由  
  
 アセンブリが生成されましたが、コンパイラが署名を完了しようとしたときにエラーが発生しました。
