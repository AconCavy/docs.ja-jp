# <a name="compiler-error-cs1011"></a>コンパイラ エラー CS1011
空の文字リテラルです。  
  
 [char](../../csharp/language-reference/keywords/char.md) が宣言され、null に初期化されました。 `char` の初期化では、文字を指定する必要があります。  
  
 次の例では CS1011 が生成されます。  
  
```csharp  
// CS1011.cs  
class Sample  
{  
   public char CharField = '';   // CS1011  
}  
```
