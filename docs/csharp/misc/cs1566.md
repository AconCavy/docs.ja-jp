# <a name="compiler-error-cs1566"></a>コンパイラ エラー CS1566
リソース ファイル 'file' を読み込み中にエラーが発生しました -- 'reason'  
  
 コンパイラは、 [-resource](../language-reference/compiler-options/resource-compiler-option.md)コンパイラオプションに渡されたファイル名で問題が発生しました。
