# <a name="compiler-error-cs1647"></a>コンパイラ エラー CS1647

式が長すぎるか複雑すぎるため、'code' 付近ではコンパイルできません  
  
 コードを処理するコンパイラでスタック オーバーフローが発生しました。 このエラーを解決するには、コードを簡略化します。 コードが有効な場合は、 [開発者コミュニティ](https://developercommunity.visualstudio.com/spaces/61/index.html) サイトまたは [stackoverflow](https://stackoverflow.com/)のヘルプを参照してください。
