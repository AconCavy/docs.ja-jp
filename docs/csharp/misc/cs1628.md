 <span data-ttu-id="33b81-105">このエラーは `in` 、 `ref` 匿名メソッドブロック内で、、またはパラメーターを使用した場合に発生し `out` ます。</span><span class="sxs-lookup"><span data-stu-id="33b81-105">This error occurs if you use an `in`, `ref`, or `out` parameter inside an anonymous method block.</span></span> <span data-ttu-id="33b81-106">このエラーを回避するには、ローカル変数を使用するか、別のコンストラクトを使用します。</span><span class="sxs-lookup"><span data-stu-id="33b81-106">To avoid this error, use a local variable or some other construct.</span></span>
