# <a name="compiler-error-cs1621"></a><span data-ttu-id="ecf86-103">コンパイラ エラー CS1621</span><span class="sxs-lookup"><span data-stu-id="ecf86-103">Compiler Error CS1621</span></span>
<span data-ttu-id="ecf86-104">yield ステートメントは、匿名メソッドまたはラムダ式の内部では使用できません</span><span class="sxs-lookup"><span data-stu-id="ecf86-104">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
<span data-ttu-id="ecf86-105">[匿名メソッド](../language-reference/operators/delegate-operator.md)または[ラムダ式](../language-reference/operators/lambda-expressions.md)で[yield](../language-reference/keywords/yield.md)ステートメントを使用することはできません。</span><span class="sxs-lookup"><span data-stu-id="ecf86-105">You cannot use the [yield](../language-reference/keywords/yield.md) statement in an [anonymous method](../language-reference/operators/delegate-operator.md) or a [lambda expression](../language-reference/operators/lambda-expressions.md).</span></span>
  
<span data-ttu-id="ecf86-106">次の例では CS1621 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="ecf86-106">The following sample generates CS1621:</span></span>
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
