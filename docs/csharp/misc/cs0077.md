# <a name="compiler-error-cs0077"></a><span data-ttu-id="879fc-102">コンパイラ エラー CS0077</span><span class="sxs-lookup"><span data-stu-id="879fc-102">Compiler Error CS0077</span></span>
<span data-ttu-id="879fc-103">as 演算子は参照型または null 許容型で使用してください ('int' は null 非許容の値型です)。</span><span class="sxs-lookup"><span data-stu-id="879fc-103">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="879fc-104">[as](../language-reference/operators/type-testing-and-cast.md#as-operator) 演算子に [値型](../language-reference/builtin-types/value-types.md)が渡されました。</span><span class="sxs-lookup"><span data-stu-id="879fc-104">The [as](../language-reference/operators/type-testing-and-cast.md#as-operator) operator was passed a [value type](../language-reference/builtin-types/value-types.md).</span></span> <span data-ttu-id="879fc-105">`as` は[null](../language-reference/keywords/null.md)を返すことができるので、[参照型](../language-reference/keywords/reference-types.md)または[null 許容値型](../language-reference/builtin-types/nullable-value-types.md)のみを渡すことができます。</span><span class="sxs-lookup"><span data-stu-id="879fc-105">Because `as` can return [null](../language-reference/keywords/null.md), it can only be passed a [reference type](../language-reference/keywords/reference-types.md) or a [nullable value type](../language-reference/builtin-types/nullable-value-types.md).</span></span>
  
 <span data-ttu-id="879fc-106">次の例では CS0077 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="879fc-106">The following sample generates CS0077:</span></span>  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
