# <a name="compiler-error-cs1608"></a>コンパイラ エラー CS1608
Required 属性は C# 型で許可されていません。  
  
 C# コードで定義された型では、<xref:System.Runtime.CompilerServices.RequiredAttributeAttribute> を使用できません。
