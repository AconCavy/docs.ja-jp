# <a name="compiler-error-cs0555"></a><span data-ttu-id="576f3-103">コンパイラ エラー CS0555</span><span class="sxs-lookup"><span data-stu-id="576f3-103">Compiler Error CS0555</span></span>

<span data-ttu-id="576f3-104">ユーザー定義の演算子は、それを囲む型のオブジェクトの取得、およびそれを囲む型のオブジェクトへの変換を行えません。</span><span class="sxs-lookup"><span data-stu-id="576f3-104">User-defined operator cannot take an object of the enclosing type and convert to an object of the enclosing type</span></span>

<span data-ttu-id="576f3-105">外側のクラスの値へのユーザー定義の変換は許可されません。このような演算子は不要です。</span><span class="sxs-lookup"><span data-stu-id="576f3-105">User-defined conversions to values of the enclosing class are not allowed; you do not need such an operator.</span></span>

<span data-ttu-id="576f3-106">次の例では、CS0555 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="576f3-106">The following sample generates CS0555:</span></span>

```csharp
// CS0555.cs
public class MyClass
{
   // delete the following operator to resolve this CS0555
   public static implicit operator MyClass(MyClass aa)   // CS0555
   {
      return new MyClass();
   }

   public static void Main()
   {
   }
}
```
