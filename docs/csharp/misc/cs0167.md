# <a name="compiler-error-cs0167"></a>コンパイラ エラー CS0167
デリゲート 'delegate' に Invoke メソッドがありません。  
  
 別のコンパイラで作成したマネージド プログラム (.NET Framework 共通言語ランタイムを使用するプログラム) をインポートして使用しました。 そのコンパイラでは、正しくない形式の [デリゲート](../language-reference/keywords/delegate.md)が許可されています。 このため、 `Invoke` メソッドを使用できませんでした。 詳細については、「[デリゲート](../programming-guide/delegates/index.md)」を参照してください。
