# <a name="compiler-warning-level-2-cs0435"></a><span data-ttu-id="17ed8-103">コンパイラの警告 (レベル 2) CS0435</span><span class="sxs-lookup"><span data-stu-id="17ed8-103">Compiler Warning (level 2) CS0435</span></span>

 <span data-ttu-id="17ed8-106">この警告は、ソース ファイル (file_2) 内の名前空間が、file_1 のインポートされた型と競合している場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="17ed8-106">This warning is issued when a namespace in a source file (file_2) conflicts with an imported type in file_1.</span></span> <span data-ttu-id="17ed8-107">この場合、コンパイラは、ソース ファイル内のものを使用します。</span><span class="sxs-lookup"><span data-stu-id="17ed8-107">The compiler uses the one in the source file.</span></span>  
  
 <span data-ttu-id="17ed8-108">次の例では、CS0435 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="17ed8-108">The following example generates CS0435:</span></span>

```csharp

{  
   public class A { }  
}  
```  
  
 <span data-ttu-id="17ed8-110">続けて、次のファイルをコンパイルします。</span><span class="sxs-lookup"><span data-stu-id="17ed8-110">Then, compile this file:</span></span>  
  
```csharp  
// CS0435_2.cs  
// compile with: /r:CS0435_1.dll  
  
using System;  
  
namespace Util
{  
   public class A { }  
}  
  
public class Test
{  
   public static void Main()
   {  
      Console.WriteLine(typeof(Util.A)); // CS0435  
   }  
}  
```
