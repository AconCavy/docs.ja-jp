{  
   public class A { }  
}  
```  
  
 <span data-ttu-id="51cd8-109">続けて、次のファイルをコンパイルします。</span><span class="sxs-lookup"><span data-stu-id="51cd8-109">Then, compile this file:</span></span>

namespace Util
{  
   public class A { }  
}

{  
   public static void Main()
   {  
      Console.WriteLine(typeof(Util.A)); // CS0435  
   }  
}  
```
