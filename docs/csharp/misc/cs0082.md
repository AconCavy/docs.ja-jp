# <a name="compiler-error-cs0082"></a><span data-ttu-id="24d9e-103">コンパイラ エラー CS0082</span><span class="sxs-lookup"><span data-stu-id="24d9e-103">Compiler Error CS0082</span></span>

 <span data-ttu-id="24d9e-105">コンパイル時に、プロパティは、識別子の前の `get_` と `set_` 、またはそのいずれかでメソッドに変換されます。</span><span class="sxs-lookup"><span data-stu-id="24d9e-105">Properties at compile time are translated to methods with `get_` and/or `set_` in front of the identifier.</span></span> <span data-ttu-id="24d9e-106">メソッド名と競合する独自のメソッドを定義すると、エラーが生成されます。</span><span class="sxs-lookup"><span data-stu-id="24d9e-106">If you define your own method that conflicts with the method name, an error is generated.</span></span>  
  
## <a name="example"></a><span data-ttu-id="24d9e-107">例</span><span class="sxs-lookup"><span data-stu-id="24d9e-107">Example</span></span>  
 <span data-ttu-id="24d9e-108">次の例では CS0082 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="24d9e-108">The following example generates CS0082:</span></span>  
  
```csharp  
//cs0082.cs  
class MyClass  
{  
  
    //property  
    public int MyProp  
    {  
        get //CS0082  
        {  
            return 1;  
        }  
    }  
  
    //conflicting Get  
    public int get_MyProp()  
    {  
        return 2;  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
