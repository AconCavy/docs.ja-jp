# <a name="compiler-warning-level-3-cs0168"></a><span data-ttu-id="443d7-103">コンパイラの警告 (レベル 3) CS0168</span><span class="sxs-lookup"><span data-stu-id="443d7-103">Compiler Warning (level 3) CS0168</span></span>
<span data-ttu-id="443d7-104">変数 ' var ' は宣言されていますが使用されていません。</span><span class="sxs-lookup"><span data-stu-id="443d7-104">The variable 'var' is declared but never used</span></span>

<span data-ttu-id="443d7-105">変数を宣言するときに、コンパイラがレベル3の警告を発行しますが、変数は使用しません。</span><span class="sxs-lookup"><span data-stu-id="443d7-105">The compiler issues a level-three warning when you declare a variable, but do not use it.</span></span>
  
<span data-ttu-id="443d7-106">次の例では、CS0168 警告が生成されます。</span><span class="sxs-lookup"><span data-stu-id="443d7-106">The following sample generates a CS0168 warning:</span></span>  
  
```csharp
// CS0168.cs  
// compile with: /W:3  
public class clx
{
    public int i;
}

public class clz
{
    public static void Main() {
        clx a; // CS0168, the variable 'a' is declared but never used
        // try the following line instead
        // clx a = new clx();  // this does not generate a warning because the clx constructor must execute.
    }
}
```
