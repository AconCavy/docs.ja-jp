# <a name="compiler-error-cs0150"></a><span data-ttu-id="2ef6e-102">コンパイラ エラー CS0150</span><span class="sxs-lookup"><span data-stu-id="2ef6e-102">Compiler Error CS0150</span></span>
<span data-ttu-id="2ef6e-103">定数値が必要です。</span><span class="sxs-lookup"><span data-stu-id="2ef6e-103">A constant value is expected</span></span>  
  
 <span data-ttu-id="2ef6e-104">定数が必要な個所で、変数が見つかりました。</span><span class="sxs-lookup"><span data-stu-id="2ef6e-104">A variable was found where a constant was expected.</span></span> <span data-ttu-id="2ef6e-105">詳細については、「 [switch](../language-reference/keywords/switch.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="2ef6e-105">For more information, see [switch](../language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="2ef6e-106">次の例では CS0150 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="2ef6e-106">The following sample generates CS0150:</span></span>  
  
```csharp  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 <span data-ttu-id="2ef6e-107">このエラーは、配列のサイズが変数の値で指定され、配列初期化子で初期化されるときにも生成されます。</span><span class="sxs-lookup"><span data-stu-id="2ef6e-107">This error is also produced when an array size is specified with a variable value and initialized with an array initializer.</span></span> <span data-ttu-id="2ef6e-108">エラーを除去するには、別のステートメントで配列を初期化します。</span><span class="sxs-lookup"><span data-stu-id="2ef6e-108">To remove the error, initialize the array in a separate statement or statements.</span></span>  
  
```csharp  
// CS0150.cs  
    namespace MyNamespace  
    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```
