# <a name="compiler-warning-level-1-cs0183"></a><span data-ttu-id="7086a-102">コンパイラの警告 (レベル 1) CS0183</span><span class="sxs-lookup"><span data-stu-id="7086a-102">Compiler Warning (level 1) CS0183</span></span>
<span data-ttu-id="7086a-103">式は常に指定された型 ('type') です</span><span class="sxs-lookup"><span data-stu-id="7086a-103">The given expression is always of the provided ('type') type</span></span>  
  
 <span data-ttu-id="7086a-104">条件付きステートメントが常に **true**に評価される場合、条件付きステートメントは必要ありません。</span><span class="sxs-lookup"><span data-stu-id="7086a-104">If a conditional statement always evaluates to **true**, then you do not need a conditional statement.</span></span> <span data-ttu-id="7086a-105">この警告は、 **is** 演算子を使用して型を評価しようとしたときに発生します。</span><span class="sxs-lookup"><span data-stu-id="7086a-105">This warning occurs when you try to evaluate a type using the **is** operator.</span></span> <span data-ttu-id="7086a-106">評価が値型の場合、チェックは不要です。</span><span class="sxs-lookup"><span data-stu-id="7086a-106">If the evaluation is a value type, then the check is unnecessary.</span></span>  
  
 <span data-ttu-id="7086a-107">次の例では CS0183 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="7086a-107">The following sample generates CS0183:</span></span>  
  
```csharp  
// CS0183.cs  
// compile with: /W:1  
using System;  
public class Test  
{  
   public static void F(Int32 i32, String str)  
   {  
      if (str is Object)          // OK  
         Console.WriteLine( "str is an object" );  
      else  
         Console.WriteLine( "str is not an object" );  
  
      if (i32 is Object)   // CS0183  
         Console.WriteLine( "i32 is an object" );  
      else  
         Console.WriteLine( "i32 is not an object" ); // never reached  
   }  
  
   public static void Main()  
   {  
  
      F(0, "CS0183");  
      F(120, null);
   }  
}  
```
