# <a name="compiler-error-cs0743"></a><span data-ttu-id="a2c52-102">コンパイラ エラー CS0743</span><span class="sxs-lookup"><span data-stu-id="a2c52-102">Compiler Error CS0743</span></span>
<span data-ttu-id="a2c52-103">コンテキスト キーワード 'on' が必要です。</span><span class="sxs-lookup"><span data-stu-id="a2c52-103">Expected contextual keyword 'on'</span></span>  
  
 <span data-ttu-id="a2c52-104">`join` 句のパターンは、この例のように、 `join`...`in`...`on`...`equals`です。</span><span class="sxs-lookup"><span data-stu-id="a2c52-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="a2c52-105">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="a2c52-105">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="a2c52-107">例</span><span class="sxs-lookup"><span data-stu-id="a2c52-107">Example</span></span>  
 <span data-ttu-id="a2c52-108">次のコードでは CS0743 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="a2c52-108">The following code generates CS0743:</span></span>  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3, 4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
