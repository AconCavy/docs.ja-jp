# <a name="compiler-error-cs0522"></a>コンパイラ エラー CS0522
'constructor': 構造体は基底クラスのコンストラクターを呼び出すことができません  
  
 [構造体](../language-reference/keywords/struct.md) は基底クラスのコンストラクターを呼び出すことはできません。基底クラスのコンストラクターの呼び出しを削除します。  
  
 次の例では CS0522 が生成されます。  
  
```csharp  
// CS0522.cs  
public class clx  
{  
   public clx(int i)  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public struct cly  
{  
   public cly(int i):base(0)   // CS0522  
   // try the following line instead  
   // public cly(int i)  
   {  
   }  
}  
```
