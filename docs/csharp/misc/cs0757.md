## <a name="example"></a><span data-ttu-id="8b637-110">例</span><span class="sxs-lookup"><span data-stu-id="8b637-110">Example</span></span>  
 <span data-ttu-id="8b637-111">次の例では CS0757 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="8b637-111">The following example generates CS0757:</span></span>  
  
```csharp  
// cs0757.cs  
using System;  
  
    public partial class C  
    {  
        // Defining declaration.  
        partial void Part();  
  
        // Implementing declaration.  
        partial void Part()  
        {  
            //...Do something.  
        }  
  
        // Second implementing declaration.  
        partial void Part() // CS0757  
        {  
            //...Do something.  
        }  
  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
