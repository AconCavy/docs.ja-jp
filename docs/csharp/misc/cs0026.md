## <a name="example"></a><span data-ttu-id="fd9bd-107">例</span><span class="sxs-lookup"><span data-stu-id="fd9bd-107">Example</span></span>  
 <span data-ttu-id="fd9bd-108">次の例では、CS0026 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="fd9bd-108">The following example generates CS0026:</span></span>

```csharp  
// CS0026.cs

   public static void Main()  
   {

      // Try the following line instead:  
      // i = i + 1;  
   }  
}  
```
