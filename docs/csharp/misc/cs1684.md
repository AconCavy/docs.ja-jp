 <span data-ttu-id="69254-104">このエラーは、ある名前空間の内部の参照が別の名前空間の内部にあるはずの型を参照するとき、その型が存在しない場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="69254-104">This error can be caused by a reference inside one namespace referring to a type that it says exists inside a second namespace, but the type does not exist.</span></span> <span data-ttu-id="69254-105">たとえば、mydll.dll では型 `A` が yourdll.dll 内に存在することになっているのに、yourdll.dll 内にはそのような型が存在しない場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="69254-105">For example, mydll.dll says that type `A` exists inside yourdll.dll, but no such type exists inside yourdll.dll.</span></span> <span data-ttu-id="69254-106">このエラーの原因としては、使用している yourdll.dll のバージョンが古すぎて、 `A` がまだ定義されていないことが考えられます。</span><span class="sxs-lookup"><span data-stu-id="69254-106">One possible cause of this error is that the version of yourdll.dll you are using is too old and `A` has not yet been defined.</span></span>

## <a name="example"></a><span data-ttu-id="69254-108">例</span><span class="sxs-lookup"><span data-stu-id="69254-108">Example</span></span>  
  
```csharp

// compile with: /target:library /keyfile:CS1684.key

   public void Test() {}  
}  
  
public class C2 {}  
```  
  
## <a name="example"></a><span data-ttu-id="69254-109">例</span><span class="sxs-lookup"><span data-stu-id="69254-109">Example</span></span>

