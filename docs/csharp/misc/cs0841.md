# <a name="compiler-error-cs0841"></a><span data-ttu-id="1cc9e-102">コンパイラ エラー CS0841</span><span class="sxs-lookup"><span data-stu-id="1cc9e-102">Compiler Error CS0841</span></span>

## <a name="example"></a><span data-ttu-id="1cc9e-105">例</span><span class="sxs-lookup"><span data-stu-id="1cc9e-105">Example</span></span>

```csharp
// cs0841.cs

public class Program
{
    public static void Main()
    {
        j = 5; // CS0841
        int j;
    }
}
```

## <a name="to-correct-this-error"></a><span data-ttu-id="1cc9e-107">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="1cc9e-107">To correct this error</span></span>

```csharp

{
    public static void Main()
    {
        int j;
        j = 5;
    }
}
```
