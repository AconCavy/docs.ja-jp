 <span data-ttu-id="539a8-105">[Is](../language-reference/operators/type-testing-and-cast.md#is-operator)および[as](../language-reference/operators/type-testing-and-cast.md#as-operator)演算子は、ポインター型では使用できません。</span><span class="sxs-lookup"><span data-stu-id="539a8-105">The [is](../language-reference/operators/type-testing-and-cast.md#is-operator) and [as](../language-reference/operators/type-testing-and-cast.md#as-operator) operators are not valid for use on pointer types.</span></span> <span data-ttu-id="539a8-106">詳細については、「[アンセーフ コードとポインター](../programming-guide/unsafe-code-pointers/index.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="539a8-106">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>

```csharp  
// CS0244.cs  
// compile with: /unsafe  
  
class UnsafeTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      bool b = p is object;   // CS0244 p is pointer  
   }  
  
   unsafe public static void Main()  
   {  
      int i = 5;  
      SquarePtrParam (&i);  
   }  
}  
```
