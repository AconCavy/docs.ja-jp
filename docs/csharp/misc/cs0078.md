# <a name="compiler-warning-level-4-cs0078"></a><span data-ttu-id="0b22b-103">コンパイラの警告 (レベル 4) CS0078</span><span class="sxs-lookup"><span data-stu-id="0b22b-103">Compiler Warning (level 4) CS0078</span></span>
<span data-ttu-id="0b22b-104">'l' と 数字の '1' との混同を避けるため、'L' を使用してください。</span><span class="sxs-lookup"><span data-stu-id="0b22b-104">The 'l' suffix is easily confused with the digit '1' -- use 'L' for clarity</span></span>  
  
 <span data-ttu-id="0b22b-105">大文字の L ではなく小文字の l を使用する long 型へのキャストを検出すると、コンパイラによって警告が出されます。</span><span class="sxs-lookup"><span data-stu-id="0b22b-105">The compiler warns when it detects a cast to long using a lowercase l instead of an uppercase L.</span></span>  
  
 <span data-ttu-id="0b22b-106">次の例では CS0078 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="0b22b-106">The following sample generates CS0078:</span></span>  
  
```csharp  
// CS0078.cs  
// compile with: /W:4  
class test {  
   public static void TestL(long i)  
   {  
   }  
  
   public static void TestL(int i)  
   {  
   }  
  
   public static void Main()  
   {  
      TestL(25l);   // CS0078  
      // try the following line instead  
      // TestL(25L);  
   }  
}  
```
