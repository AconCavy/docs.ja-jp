# <a name="compiler-error-cs0030"></a><span data-ttu-id="94d46-103">コンパイラ エラー CS0030</span><span class="sxs-lookup"><span data-stu-id="94d46-103">Compiler Error CS0030</span></span>
<span data-ttu-id="94d46-104">型 'type' を 'type' に変換できません</span><span class="sxs-lookup"><span data-stu-id="94d46-104">Cannot convert type 'type' to 'type'</span></span>  
  
<span data-ttu-id="94d46-105">型の間に定義済みの変換はありません。</span><span class="sxs-lookup"><span data-stu-id="94d46-105">There is no predefined conversion between types.</span></span> <span data-ttu-id="94d46-106">これらの型の間のカスタム変換を定義できます。</span><span class="sxs-lookup"><span data-stu-id="94d46-106">You can define a custom conversion between those types.</span></span> <span data-ttu-id="94d46-107">詳細については、「[ユーザー定義の変換演算子](../language-reference/operators/user-defined-conversion-operators.md)」 に関するページを参照してください。</span><span class="sxs-lookup"><span data-stu-id="94d46-107">For more information, see [User-defined conversion operators](../language-reference/operators/user-defined-conversion-operators.md).</span></span>  
  
 <span data-ttu-id="94d46-108">次の例では CS0030 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="94d46-108">The following sample generates CS0030:</span></span>  
  
```csharp  
// CS0030.cs  
namespace x  
{  
   public class iii  
   {  
      /*  
      public static implicit operator iii(int aa)  
      {  
         return null;  
      }  
  
      public static implicit operator int(iii aa)  
      {  
         return 0;  
      }  
      */  
  
      public static iii operator ++(iii aa)  
      {  
         return (iii)0;   // CS0030  
         // uncomment the conversion routines to resolve CS0030  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
