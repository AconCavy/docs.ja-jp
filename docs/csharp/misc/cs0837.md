# <a name="compiler-error-cs0837"></a><span data-ttu-id="d315f-103">コンパイラ エラー CS0837</span><span class="sxs-lookup"><span data-stu-id="d315f-103">Compiler Error CS0837</span></span>

<span data-ttu-id="d315f-104">演算子 ' is ' または ' as ' の最初のオペランドを、ラムダ式、匿名メソッド、またはメソッドグループにすることはできません。</span><span class="sxs-lookup"><span data-stu-id="d315f-104">The first operand of an 'is' or 'as' operator may not be a lambda expression, anonymous method, or method group.</span></span>

 <span data-ttu-id="d315f-105">ラムダ式、匿名メソッド、およびメソッドグループは、 [is](../language-reference/operators/type-testing-and-cast.md#is-operator) または [as](../language-reference/operators/type-testing-and-cast.md#as-operator)の左側で使用することはできません。</span><span class="sxs-lookup"><span data-stu-id="d315f-105">Lambda expressions, anonymous methods and method groups may not be used on the left side of [is](../language-reference/operators/type-testing-and-cast.md#is-operator) or [as](../language-reference/operators/type-testing-and-cast.md#as-operator).</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="d315f-106">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="d315f-106">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="d315f-110">例</span><span class="sxs-lookup"><span data-stu-id="d315f-110">Example</span></span>

<span data-ttu-id="d315f-111">次の例では CS0837 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="d315f-111">The following example generates CS0837:</span></span>

```csharp
// cs0837.cs
namespace TestNamespace
{
    public delegate void Del();

    class Test
    {
        static int Main()
        {
            bool b1 = (() => { }) is Del;   // CS0837
            bool b2 = delegate() { } is Del;// CS0837
            Del d1 = () => { } as Del;      // CS0837  
            Del d2 = delegate() { } as Del; // CS0837
            return 1;
        }
    }
}
```
