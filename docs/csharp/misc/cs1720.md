 <span data-ttu-id="457e5-104">このエラーは、参照型 (クラスなど) であるジェネリック型変数の既定値を含む式を記述すると、発生します。</span><span class="sxs-lookup"><span data-stu-id="457e5-104">If you write an expression involving the default of a generic type variable that is a reference type (for example, a class), this error will occur.</span></span> <span data-ttu-id="457e5-105">次のような式があるとします。</span><span class="sxs-lookup"><span data-stu-id="457e5-105">Consider the following expression:</span></span>

```csharp

public class Tester
{  
    public static void GenericClass<T>(T t1) where T : class
    {  
        Console.WriteLine(default(T).ToString());  // CS1720  
    }  
    public static void Main() {}  
}  
```
