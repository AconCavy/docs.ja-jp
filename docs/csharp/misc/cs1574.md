 <span data-ttu-id="dc980-105">\<exception>現在のビルド環境内で使用できないメンバーを参照する、たとえば、タグ内で cref タグに渡される文字列。</span><span class="sxs-lookup"><span data-stu-id="dc980-105">A string passed to a cref tag, for example, within an \<exception> tag, referred to a member that is not available within the current build environment.</span></span> <span data-ttu-id="dc980-106">cref タグに渡す文字列は、メンバーまたはフィールドの構文的に正しい名前である必要があります。</span><span class="sxs-lookup"><span data-stu-id="dc980-106">The string that you pass to a cref tag must be the syntactically correct name of a member or field.</span></span>  
  
 <span data-ttu-id="dc980-107">詳細については、「 [ドキュメント コメント用の推奨タグ](../programming-guide/xmldoc/recommended-tags-for-documentation-comments.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="dc980-107">For more information, see [Recommended Tags for Documentation Comments](../programming-guide/xmldoc/recommended-tags-for-documentation-comments.md).</span></span>

```csharp  
using System;  
  
/// <exception cref="System.Console.WriteLin">An exception class.</exception>   // CS1574

// /// <exception cref="System.Console.WriteLine">An exception class.</exception>

{  
}  
  
class TestClass  
{  
   public static void Main()  
   {  
      try  
      {  
      }  
      catch(EClass)  
      {  
      }  
   }  
}  
```
