# <a name="compiler-error-cs0517"></a>コンパイラ エラー CS0517

'class' に基底クラスがないため、基底コンストラクターを呼び出せません。  
  
 CS0517 は、.NET ランタイムがオブジェクトクラスのソースコードをコンパイルする場合にのみ発生します。これは、基底クラスを持たない唯一のクラスです。
