# <a name="compiler-error-cs0076"></a>コンパイラ エラー CS0076
列挙子名 'value__' は予約されているため、使用できません  
  
 [列挙](../../csharp/language-reference/keywords/enum.md) 内の項目に、 **value__** という識別子を含めることはできません。
