## <a name="example"></a><span data-ttu-id="5807d-109">例</span><span class="sxs-lookup"><span data-stu-id="5807d-109">Example</span></span>  
 <span data-ttu-id="5807d-110">次の例では CS0403 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="5807d-110">The following sample generates CS0403.</span></span>  
  
```csharp  
// CS0403.cs  
// compile with: /target:library  
class C<T>  
{  
   public void f()  
   {  
      T t = null;  // CS0403  
      T t2 = default(T);   // OK  
    }  
}  
  
class D<T> where T : class
{  
   public void f()  
   {  
      T t = null;  // OK  
    }  
}  
```
