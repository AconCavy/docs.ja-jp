# <a name="compiler-error-cs0218"></a><span data-ttu-id="ab4dd-103">コンパイラ エラー CS0218</span><span class="sxs-lookup"><span data-stu-id="ab4dd-103">Compiler Error CS0218</span></span>

<span data-ttu-id="ab4dd-104">型 ('type') に演算子 true および演算子 false の宣言が含まれている必要があります</span><span class="sxs-lookup"><span data-stu-id="ab4dd-104">The type ('type') must contain declarations of operator true and operator false</span></span>  
  
<span data-ttu-id="ab4dd-105">ユーザー定義型で[& 演算子](../language-reference/operators/boolean-logical-operators.md#logical-and-operator-)または[&#124; 演算子](../language-reference/operators/boolean-logical-operators.md#logical-or-operator-)がオーバーロードされている場合、ショートサーキット[&& 演算子](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-)または[&#124;&#124; 演算子](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-)を定義するために、 [true および false](../language-reference/operators/true-false-operators.md)演算子も定義する必要があります。</span><span class="sxs-lookup"><span data-stu-id="ab4dd-105">If a user-defined type overloads the [& operator](../language-reference/operators/boolean-logical-operators.md#logical-and-operator-) or [&#124; operator](../language-reference/operators/boolean-logical-operators.md#logical-or-operator-), it must also define [true and false](../language-reference/operators/true-false-operators.md) operators, in order to make short-circuiting [&& operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) or [&#124;&#124; operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-) defined.</span></span>
  
 <span data-ttu-id="ab4dd-106">次の例では CS0218 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="ab4dd-106">The following sample generates CS0218:</span></span>  
  
```csharp  
// CS0218.cs  
using System;  
public class MyClass  
{  
   // uncomment these operator declarations to resolve this CS0218  
   /*  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
   */  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;   // CS0218, requires operators true and false  
   }  
}  
```
