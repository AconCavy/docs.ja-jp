# <a name="compiler-error-cs1022"></a>コンパイラ エラー CS1022
型、名前空間の定義、またはファイルの終わりが必要です。  
  
 ソース コード ファイルに、対応する中かっこのセットがありません。  
  
 次の例では CS1022 が生成されます。  
  
```csharp  
// CS1022.cs  
namespace x  
{  
}  
}   // CS1022  
```
