 <span data-ttu-id="d9e6a-105">メンバーは、 [シール](../language-reference/keywords/override.md) されている継承されたメンバーを [オーバーライド](../language-reference/keywords/sealed.md) することはできません。</span><span class="sxs-lookup"><span data-stu-id="d9e6a-105">A member cannot [override](../language-reference/keywords/override.md) a [sealed](../language-reference/keywords/sealed.md) inherited member.</span></span> <span data-ttu-id="d9e6a-106">詳細については、「 [Checked And Unchecked](../language-reference/keywords/checked-and-unchecked.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="d9e6a-106">For more information, see [Checked and Unchecked](../language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="d9e6a-107">次の例では CS0239 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="d9e6a-107">The following sample generates CS0239:</span></span>  
  
```csharp  
// CS0239.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public override sealed void f()  
   {  
   }  
}  
  
class MyClass3 : MyClass2  
{  
   public override void f()   // CS0239  
   // Try the following definition instead:  
   // public new void f()  
   {  
   }  
}  
```
