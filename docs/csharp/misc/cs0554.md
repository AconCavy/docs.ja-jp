# <a name="compiler-error-cs0554"></a><span data-ttu-id="9426d-103">コンパイラ エラー CS0554</span><span class="sxs-lookup"><span data-stu-id="9426d-103">Compiler Error CS0554</span></span>
<span data-ttu-id="9426d-104">'conversion routine': 派生クラスへ/からのユーザー定義変換です</span><span class="sxs-lookup"><span data-stu-id="9426d-104">'conversion routine' : user defined conversion to/from derived class</span></span>  
  
 <span data-ttu-id="9426d-105">派生クラスの値へのユーザー定義の変換は許可されません。このような演算子は不要です。</span><span class="sxs-lookup"><span data-stu-id="9426d-105">User-defined conversions to values of a derived class are not allowed; you do not need such an operator.</span></span>  
  
 <span data-ttu-id="9426d-106">ユーザー定義の変換について詳しくは、C# 言語仕様の第 6 章をご覧ください。</span><span class="sxs-lookup"><span data-stu-id="9426d-106">See chapter 6 in the C# language specification for more information on user-defined conversions.</span></span>  
  
 <span data-ttu-id="9426d-107">次の例では CS0554 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="9426d-107">The following sample generates CS0554:</span></span>  
  
```csharp  
// CS0554.cs  
namespace x  
{  
   public class ii  
   {  
      // delete the conversion routine to resolve CS0554  
      public static implicit operator ii(a aa) // CS0554  
      {  
         return new ii();  
      }  
   }  
  
   public class a : ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
