# <a name="compiler-error-cs0667"></a>コンパイラ エラー CS0667
機能 'invalid feature' は非推奨とされます。 代わりに 'valid feature' を使用してください。  
  
 使用しようとした機能は、現在非推奨とされました。 代わりに、正しい機能を使用するようにコードを更新します。
