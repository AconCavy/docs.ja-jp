# <a name="compiler-warning-level-1-cs3004"></a>コンパイラの警告 (レベル 1) CS3004
混合、分解された Unicode 文字は CLS に準拠していません。  
  
 共通言語仕様 (CLS) に準拠するために、 [public](../../csharp/language-reference/keywords/public.md)識別子、 [protected](../../csharp/language-reference/keywords/protected.md)識別子、または `protected internal` 識別子に使用できるのは、構成済みの UNICODE 文字だけです。 CLS 準拠の詳細については、次を参照してください。[言語の独立性および言語非依存コンポーネント](../../standard/language-independence.md)します。
