# <a name="compiler-error-cs0220"></a><span data-ttu-id="a19d1-103">コンパイラ エラー CS0220</span><span class="sxs-lookup"><span data-stu-id="a19d1-103">Compiler Error CS0220</span></span>
<span data-ttu-id="a19d1-104">この操作はチェック モードでコンパイルしたときにオーバーフローします。</span><span class="sxs-lookup"><span data-stu-id="a19d1-104">The operation overflows at compile time in checked mode</span></span>  
  
 <span data-ttu-id="a19d1-105">[checked](../language-reference/keywords/checked.md)で操作が検出されました。これは既定であり、データの損失が発生しました。</span><span class="sxs-lookup"><span data-stu-id="a19d1-105">An operation was detected by [checked](../language-reference/keywords/checked.md), which is the default, and a data loss resulted.</span></span> <span data-ttu-id="a19d1-106">このエラーを解決するには、代入を修正するか、または [unchecked](../language-reference/keywords/unchecked.md) を使用します。</span><span class="sxs-lookup"><span data-stu-id="a19d1-106">Either correct the inputs to the assignment or use [unchecked](../language-reference/keywords/unchecked.md) to resolve this error.</span></span> <span data-ttu-id="a19d1-107">詳細については、「 [Checked And Unchecked](../language-reference/keywords/checked-and-unchecked.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="a19d1-107">For more information, see [Checked and Unchecked](../language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="a19d1-108">次の例では CS0220 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="a19d1-108">The following sample generates CS0220:</span></span>  
  
```csharp  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```
