# <a name="compiler-error-cs1017"></a>コンパイラ エラー CS1017
catch 句を、try ステートメントの一般的な catch 句の後に置くことはできません。  
  
 パラメーターを一切受け取らない `catch` ブロックは、一連の `catch` ブロックの最後に配置する必要があります。 例外の詳細については、次を参照してください。[例外と例外処理](../../csharp/programming-guide/exceptions/index.md)します。  
  
## <a name="example"></a>例  
 次の例では CS1017 が生成されます。  
  
```csharp  
// CS1017.cs  
using System;  
  
namespace x  
{  
    public class b : Exception  
    {  
    }  
  
    public class a  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
  
            catch   // CS1017, must be last catch  
            {  
            }  
  
            catch(b)  
            {  
                throw;  
            }  
        }  
    }  
}  
```
