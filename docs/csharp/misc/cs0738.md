## <a name="example"></a><span data-ttu-id="1e277-109">例</span><span class="sxs-lookup"><span data-stu-id="1e277-109">Example</span></span>  
 <span data-ttu-id="1e277-110">クラス メソッドは `void` を返し、同じ名前のインターフェイス メンバーは `int`を返すため、次のコードでは、CS0738 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="1e277-110">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>  
  
```csharp

interface ITest  
{  
    int TestMethod();  
}  
public class Test: ITest  
{  
    public void TestMethod() { } // CS0738  
    // Try the following line instead.  
    // public int TestMethod();  
}  
```
