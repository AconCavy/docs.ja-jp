# <a name="compiler-error-cs0449"></a><span data-ttu-id="71a5d-102">コンパイラ エラー CS0449</span><span class="sxs-lookup"><span data-stu-id="71a5d-102">Compiler Error CS0449</span></span>
<span data-ttu-id="71a5d-103">'class' または 'struct' 制約は、他の制約の前に指定されなければなりません。</span><span class="sxs-lookup"><span data-stu-id="71a5d-103">The 'class' or 'struct' constraint must come before any other constraints</span></span>  
  
 <span data-ttu-id="71a5d-104">ジェネリック型またはメソッドの型パラメーターの制約は、特定の順序で行う必要があります。 `class` または `struct` を最初に指定し (存在する場合)、次にインターフェイスの制約、最後にコンストラクターの制約を指定する必要があります。</span><span class="sxs-lookup"><span data-stu-id="71a5d-104">The constraints on the type parameter of a generic type or method must occur in a specific order: `class` or `struct` must be first, if present, then any interface constraints, and finally any constructor constraints.</span></span> <span data-ttu-id="71a5d-105">このエラーは、 `class` または `struct` 制約が最初にないことが原因で発生します。</span><span class="sxs-lookup"><span data-stu-id="71a5d-105">This error is caused by the `class` or `struct` constraint not appearing first.</span></span> <span data-ttu-id="71a5d-106">このエラーを解決するには、制約句の順序を変更します。</span><span class="sxs-lookup"><span data-stu-id="71a5d-106">To resolve this error, reorder the constraint clauses.</span></span>  
  
## <a name="example"></a><span data-ttu-id="71a5d-107">例</span><span class="sxs-lookup"><span data-stu-id="71a5d-107">Example</span></span>  
 <span data-ttu-id="71a5d-108">次の例では CS0449 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="71a5d-108">The following sample generates CS0449.</span></span>  
  
```csharp  
// CS0449.cs  
// compile with: /target:library  
interface I {}  
public class C4
{  
   public void F1<T>() where T : class, struct, I {}   // CS0449  
   public void F2<T>() where T : I, struct {}   // CS0449  
   public void F3<T>() where T : I, class {}   // CS0449  
  
   // OK  
   public void F4<T>() where T : class {}  
   public void F5<T>() where T : struct {}  
   public void F6<T>() where T : I {}  
}  
```
