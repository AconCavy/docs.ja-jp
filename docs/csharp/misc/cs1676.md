# <a name="compiler-error-cs1676"></a><span data-ttu-id="61407-102">コンパイラ エラー CS1676</span><span class="sxs-lookup"><span data-stu-id="61407-102">Compiler Error CS1676</span></span>

<span data-ttu-id="61407-103">パラメーター 'number' は 'keyword' キーワードと共に宣言する必要があります</span><span class="sxs-lookup"><span data-stu-id="61407-103">Parameter 'number' must be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="61407-104">このエラーは、匿名メソッドのパラメーター型修飾子が、そのメソッドのキャスト先となるデリゲートの宣言に使用されるものと異なる場合に生じます。</span><span class="sxs-lookup"><span data-stu-id="61407-104">This error occurs when the parameter type modifier in an anonymous method is different from that used in the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="61407-105">次の例では CS1676 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="61407-105">The following sample generates CS1676:</span></span>  
  
```csharp  
// CS1676.cs  
delegate void E(ref int i);  
class Errors
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```
