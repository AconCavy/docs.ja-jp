# <a name="compiler-error-cs0668"></a><span data-ttu-id="b6f6d-103">コンパイラ エラー CS0668</span><span class="sxs-lookup"><span data-stu-id="b6f6d-103">Compiler Error CS0668</span></span>
<span data-ttu-id="b6f6d-104">2 つのインデクサーの名前が違います。1 つの型の中のそれぞれのインデクサーの IndexerName 属性は、同じでなければなりません。</span><span class="sxs-lookup"><span data-stu-id="b6f6d-104">Two indexers have different names; the IndexerName attribute must be used with the same name on every indexer within a type</span></span>  
  
 <span data-ttu-id="b6f6d-105">**IndexerName** 属性に渡される値は、型のすべてのインデクサーで同じでなければなりません。</span><span class="sxs-lookup"><span data-stu-id="b6f6d-105">The values passed to the **IndexerName** attribute must be the same for all indexers in a type.</span></span> <span data-ttu-id="b6f6d-106">**IndexerName** 属性の詳細については、「 [IndexerNameAttribute クラス](xref:System.Runtime.CompilerServices.IndexerNameAttribute)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="b6f6d-106">For more information on the **IndexerName** attribute, see [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span></span>  
  
 <span data-ttu-id="b6f6d-107">次の例では CS0668 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="b6f6d-107">The following sample generates CS0668:</span></span>  
  
```csharp
// CS0668.cs  
using System;  
using System.Runtime.CompilerServices;  
  
class IndexerClass  
{  
   [IndexerName("IName1")]  
   public int this [int index]   // indexer declaration  
   {  
      get  
      {  
         return index;  
      }  
      set  
      {  
      }  
   }  
  
   [IndexerName("IName2")]  
   public int this [string s]    // CS0668, change IName2 to IName1  
   {  
      get  
      {  
         return int.Parse(s);  
      }  
      set  
      {  
      }  
   }  
  
   void Main()  
   {  
   }  
}  
```
