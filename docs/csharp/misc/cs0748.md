<span data-ttu-id="fe511-104">ラムダパラメーターの使用方法に一貫性がありません。パラメーターの型はすべて明示的またはすべて暗黙的である必要があります。</span><span class="sxs-lookup"><span data-stu-id="fe511-104">Inconsistent lambda parameter usage; parameter types must be all explicit or all implicit.</span></span>
  
<span data-ttu-id="fe511-105">ラムダ式に複数の入力パラメーターがある場合、暗黙の型指定を使用できないパラメーターもあれば、明示的な型指定を使用するパラメーターもあります。</span><span class="sxs-lookup"><span data-stu-id="fe511-105">If a lambda expression has multiple input parameters, some parameters cannot use implicit typing while others use explicit typing.</span></span>
