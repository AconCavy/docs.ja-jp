 <span data-ttu-id="14843-105">[out](../language-reference/keywords/out-parameter-modifier.md) キーワードでマークされたパラメーターに、メソッド本体で値が代入されませんでした。</span><span class="sxs-lookup"><span data-stu-id="14843-105">A parameter marked with the [out](../language-reference/keywords/out-parameter-modifier.md) keyword was not assigned a value in the method body.</span></span> <span data-ttu-id="14843-106">詳細については、「[パラメーターの引き渡し](../programming-guide/classes-and-structs/passing-parameters.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="14843-106">For more information, see [Passing Parameters](../programming-guide/classes-and-structs/passing-parameters.md)</span></span>

```csharp  
// CS0177.cs

{  
   public static void Foo(out int i)   // CS0177  
   {  
   // uncomment the following line to resolve this error

   {  
       int x = -1;  
       Foo(out x);  
   }  
}  
```
