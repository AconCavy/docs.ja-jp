 <span data-ttu-id="4147c-105">**/reference**を使用してファイルからインポートされたクラスが、あるクラスの派生クラスであるか、または見つからないインターフェイスを実装しています。</span><span class="sxs-lookup"><span data-stu-id="4147c-105">A class that was imported from a file with **/reference**, is derived from a class or implements an interface that is not found.</span></span> <span data-ttu-id="4147c-106">このエラーは、必要な DLL が **/reference**を使用したコンパイル時にインクルードされていない場合に発生することがあります。</span><span class="sxs-lookup"><span data-stu-id="4147c-106">This can occur if a required DLL is not also included in the compilation with **/reference**.</span></span>

## <a name="example"></a><span data-ttu-id="4147c-108">例</span><span class="sxs-lookup"><span data-stu-id="4147c-108">Example</span></span>  
  
```csharp

// compile with: /target:library

{  
   public class B { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="4147c-109">例</span><span class="sxs-lookup"><span data-stu-id="4147c-109">Example</span></span>  
 <span data-ttu-id="4147c-110">2 つ目のファイルでは、前の例で作成した `C` クラスから派生した `B` クラスを定義する DLL を作成します。</span><span class="sxs-lookup"><span data-stu-id="4147c-110">The second file creates a DLL that defines a class `C` that is derived from the class `B` that was created in the previous example.</span></span>

```csharp  
// compile with: /target:library /reference:CS0011_1.dll  
// post-build command: del /f CS0011_1.dll

```  
  
## <a name="example"></a><span data-ttu-id="4147c-111">例</span><span class="sxs-lookup"><span data-stu-id="4147c-111">Example</span></span>
