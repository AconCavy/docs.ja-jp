# <a name="compiler-error-cs1555"></a>コンパイラ エラー CS1555
Main メソッドに指定された 'class' が見つかりませんでした  
  
 [-Main](../language-reference/compiler-options/main-compiler-option.md)コンパイラオプションにクラスが指定されましたが、クラス名がソースコードに見つかりませんでした。
