## <a name="example"></a><span data-ttu-id="7a88d-107">例</span><span class="sxs-lookup"><span data-stu-id="7a88d-107">Example</span></span>

 <span data-ttu-id="7a88d-108">次の例では CS0410 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="7a88d-108">The following example generates CS0410:</span></span>

```csharp
// CS0410.cs
// compile with: /langversion:ISO-1

class Test
{
    delegate void D(double d );
    static void F(int i) { }

    static void Main()
    {
        D d = new D(F);  // CS0410
    }
}
```
