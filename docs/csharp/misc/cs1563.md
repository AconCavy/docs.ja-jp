# <a name="compiler-error-cs1563"></a>コンパイラ エラー CS1563

出力 'output file' にソース ファイルがありません。  
  
 [-Out](../language-reference/compiler-options/out-compiler-option.md)コンパイラオプションが指定されましたが、ソースコードファイルがありません。 **/out**を使用しないか、または **/out**の後にソース コード ファイルを指定する必要があります。
