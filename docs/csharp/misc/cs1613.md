# <a name="compiler-error-cs1613"></a><span data-ttu-id="362e5-103">コンパイラ エラー CS1613</span><span class="sxs-lookup"><span data-stu-id="362e5-103">Compiler Error CS1613</span></span>

<span data-ttu-id="362e5-104">インターフェイス 'interface' のマネージド コクラス ラッパー クラス 'class' が見つかりません (アセンブリ参照が存在することを確認してください)</span><span class="sxs-lookup"><span data-stu-id="362e5-104">The managed coclass wrapper class 'class' for interface 'interface' cannot be found (are you missing an assembly reference?)</span></span>  
  
 <span data-ttu-id="362e5-105">インターフェイスから COM オブジェクトをインスタンス化しようとしました。</span><span class="sxs-lookup"><span data-stu-id="362e5-105">An attempt was made to instantiate a COM object from an interface.</span></span> <span data-ttu-id="362e5-106">インターフェイスに **ComImport** 属性と `CoClass` 属性がありますが、 `CoClass` 属性に指定された型をコンパイラが見つけられません。</span><span class="sxs-lookup"><span data-stu-id="362e5-106">The interface has the **ComImport** and `CoClass` attributes, but the compiler cannot find the type given for the `CoClass` attribute.</span></span>  
  
 <span data-ttu-id="362e5-107">このエラーを解決するには、次のいずれかを試すことができます。</span><span class="sxs-lookup"><span data-stu-id="362e5-107">To resolve this error, you can try one of the following:</span></span>  
  
- <span data-ttu-id="362e5-108">コクラスを含むアセンブリへの参照を追加します (ほとんどの場合、インターフェイスとコクラスは同じアセンブリにあります)。</span><span class="sxs-lookup"><span data-stu-id="362e5-108">Add a reference to the assembly that has the coclass (most of the time the interface and coclass should be in the same assembly).</span></span> <span data-ttu-id="362e5-109">詳細については [、](../language-reference/compiler-options/reference-compiler-option.md) 「参照」または「 [参照の追加」ダイアログボックス](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) を参照してください。</span><span class="sxs-lookup"><span data-stu-id="362e5-109">See [-reference](../language-reference/compiler-options/reference-compiler-option.md) or [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) for information.</span></span>  
  
- <span data-ttu-id="362e5-110">インターフェイスの `CoClass` 属性を修正します。</span><span class="sxs-lookup"><span data-stu-id="362e5-110">Fix the `CoClass` attribute on the interface.</span></span>  
  
 <span data-ttu-id="362e5-111">次の例では、 **CoClassAttribute**の正しい使用法を示しています。</span><span class="sxs-lookup"><span data-stu-id="362e5-111">The following sample demonstrates correct usage of **CoClassAttribute**:</span></span>  
  
```csharp  
// CS1613.cs  
using System;  
using System.Runtime.InteropServices;  
  
[Guid("1FFD7840-E82D-4268-875C-80A160C23296")]  
[ComImport()]  
[CoClass(typeof(A))]  
public interface IA{}  
public class A : IA {}  
  
public class AA  
{  
   public static void Main()  
   {  
      IA i;  
      i = new IA(); // This is equivalent to new A().  
                    // because of the CoClass attribute on IA  
   }  
}  
```
