# <a name="compiler-error-cs1527"></a><span data-ttu-id="0089b-103">コンパイラ エラー CS1527</span><span class="sxs-lookup"><span data-stu-id="0089b-103">Compiler Error CS1527</span></span>

<span data-ttu-id="0089b-104">名前空間で定義された要素は、明示的に private、protected、protected internal、または private protected として宣言することはできません。</span><span class="sxs-lookup"><span data-stu-id="0089b-104">Elements defined in a namespace cannot be explicitly declared as private, protected, protected internal or private protected.</span></span>
  
 <span data-ttu-id="0089b-105">名前空間での型宣言には、 [public](../language-reference/keywords/public.md) または [internal](../language-reference/keywords/internal.md) アクセスを指定できます。</span><span class="sxs-lookup"><span data-stu-id="0089b-105">Type declarations in a namespace can have either [public](../language-reference/keywords/public.md) or [internal](../language-reference/keywords/internal.md) access.</span></span> <span data-ttu-id="0089b-106">アクセシビリティが指定されていない場合は、 **internal** が既定値です。</span><span class="sxs-lookup"><span data-stu-id="0089b-106">If no accessibility is specified, **internal** is the default.</span></span>  
  
 <span data-ttu-id="0089b-107">次の例では CS1527 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="0089b-107">The following sample generates CS1527:</span></span>  
  
```csharp  
// CS1527.cs  
namespace Sample  
{  
   private class C1 {}             // CS1527  
   protected class C2 {}           // CS1527  
   protected internal class C3 {}  // CS1527  
   private protected class C4 {}   // CS1527
}  
```  
  
 <span data-ttu-id="0089b-108">プログラム コードで明示的に宣言されている名前空間がない場合は、すべての型宣言が暗黙的にグローバル名前空間内に配置されるため、次の例では CS1527 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="0089b-108">The following example generates CS1527 because when no namespace is explicitly declared in your program code, all type declarations are located implicitly within the global namespace.</span></span>  
  
```csharp  
//cs1527_2.cs  
using System;  
  
protected class C4 {}  
private struct S1 {}  
```
