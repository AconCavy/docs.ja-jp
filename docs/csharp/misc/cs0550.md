# <a name="compiler-error-cs0550"></a><span data-ttu-id="af640-103">コンパイラ エラー CS0550</span><span class="sxs-lookup"><span data-stu-id="af640-103">Compiler Error CS0550</span></span>
<span data-ttu-id="af640-104">'accessor' はインターフェイス メンバー 'property' にないアクセサーを追加します。</span><span class="sxs-lookup"><span data-stu-id="af640-104">'accessor' adds an accessor not found in interface member 'property'</span></span>  
  
 <span data-ttu-id="af640-105">派生クラスにおけるプロパティの実装に、基本インターフェイスで指定されていないアクセサーが含まれています。</span><span class="sxs-lookup"><span data-stu-id="af640-105">The implementation of a property in a derived class contains an accessor that was not specified in the base interface.</span></span>  
  
 <span data-ttu-id="af640-106">詳細については、「 [Using Properties](../programming-guide/classes-and-structs/using-properties.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="af640-106">For more information, see [Using Properties](../programming-guide/classes-and-structs/using-properties.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="af640-107">例</span><span class="sxs-lookup"><span data-stu-id="af640-107">Example</span></span>  
 <span data-ttu-id="af640-108">次の例では CS0550 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="af640-108">The following sample generates CS0550.</span></span>  
  
```csharp  
// CS0550.cs  
namespace x  
{  
   interface ii  
   {  
      int i  
      {  
         get;  
         // add the following accessor to resolve this CS0550  
         // set;  
      }  
   }  
  
   public class a : ii  
   {  
      int ii.i  
      {  
         get  
         {  
            return 0;  
         }  
         set {}   // CS0550  no set in interface  
      }  
  
      public static void Main() {}  
   }  
}  
```
