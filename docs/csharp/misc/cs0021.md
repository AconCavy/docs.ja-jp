## <a name="example"></a><span data-ttu-id="cef64-109">例</span><span class="sxs-lookup"><span data-stu-id="cef64-109">Example</span></span>

<span data-ttu-id="cef64-110">次の C++ の例は、.dll ファイルにコンパイルされます。</span><span class="sxs-lookup"><span data-stu-id="cef64-110">The following C++ example compiles to a .dll file.</span></span> <span data-ttu-id="cef64-111">エラーを生成するために、 [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) 属性がコメントアウトされていることに注意してください。</span><span class="sxs-lookup"><span data-stu-id="cef64-111">Note that the [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) attribute is commented out in order to generate the error.</span></span>

```cpp
// CPP0021.cpp
// compile with: /clr /LD
using namespace System::Reflection;
// Uncomment the following line to resolve
//[DefaultMember("myItem")]
public ref class MyClassMC
{
        public:
        property int myItem[int]
        {
            int get(int i){  return 5; }
            void set(int i, int value) {}
        }
};
```

<span data-ttu-id="cef64-112">次の C# の例では、.dll ファイルを呼び出します。</span><span class="sxs-lookup"><span data-stu-id="cef64-112">The following C# example calls the .dll file.</span></span> <span data-ttu-id="cef64-113">インデクサーを使用してクラスにアクセスしようとしますが、既定のインデクサーとして宣言されたメンバーがないため、エラーが生成されます。</span><span class="sxs-lookup"><span data-stu-id="cef64-113">It attempts to access the class via an indexer, but because no member was declared as the default indexer, the error is generated.</span></span> <span data-ttu-id="cef64-114">このエラーを解決するには、行をコメント解除し `[DefaultMember("myItem")]` ます。</span><span class="sxs-lookup"><span data-stu-id="cef64-114">You can address the error by uncommenting the line `[DefaultMember("myItem")]`.</span></span>

```csharp
// CS0021.cs
// compile with: /reference:CPP0021.dll
public class MyClass
{
    public static void Main()
    {
        MyClassMC myMC = new MyClassMC();
        int j = myMC[1]; // CS0021
    }
}
```
