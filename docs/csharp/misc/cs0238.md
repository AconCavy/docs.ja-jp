# <a name="compiler-error-cs0238"></a><span data-ttu-id="66b08-103">コンパイラ エラー CS0238</span><span class="sxs-lookup"><span data-stu-id="66b08-103">Compiler Error CS0238</span></span>
<span data-ttu-id="66b08-104">override ではないため、'member' をシールすることはできません</span><span class="sxs-lookup"><span data-stu-id="66b08-104">'member' cannot be sealed because it is not an override</span></span>  
  
 <span data-ttu-id="66b08-105">[override](../language-reference/keywords/sealed.md) でもマークされていないメンバーで [sealed](../language-reference/keywords/override.md)が使用されました。</span><span class="sxs-lookup"><span data-stu-id="66b08-105">[sealed](../language-reference/keywords/sealed.md) was used on a member that was not also marked with [override](../language-reference/keywords/override.md).</span></span> <span data-ttu-id="66b08-106">詳細については、「[継承](../programming-guide/classes-and-structs/inheritance.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="66b08-106">For more information, see [Inheritance](../programming-guide/classes-and-structs/inheritance.md).</span></span>  
  
 <span data-ttu-id="66b08-107">次の例では CS0238 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="66b08-107">The following sample generates CS0238:</span></span>  
  
```csharp  
// CS0238.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public sealed void f() // CS0238  
   // Try the following definition instead:  
   // public override sealed void f()  
   {  
   }  
}  
```
