<span data-ttu-id="2a389-103">'attribute' 属性に対する引数は、有効な識別子である必要があります</span><span class="sxs-lookup"><span data-stu-id="2a389-103">The argument to the 'attribute' attribute must be a valid identifier</span></span>  
  
 <span data-ttu-id="2a389-104"><xref:System.Diagnostics.ConditionalAttribute> または <xref:System.Runtime.CompilerServices.IndexerNameAttribute> 属性に渡す引数は、有効な識別子である必要があります。</span><span class="sxs-lookup"><span data-stu-id="2a389-104">Any argument that you pass to the <xref:System.Diagnostics.ConditionalAttribute> or <xref:System.Runtime.CompilerServices.IndexerNameAttribute> attributes must be a valid identifier.</span></span> <span data-ttu-id="2a389-105">これは、識別子で無効な "+" などの文字を引数に含めることができないことを意味します。</span><span class="sxs-lookup"><span data-stu-id="2a389-105">This means that it may not contain characters such as "+" that are illegal when they occur in identifiers.</span></span>  
  
## <a name="example"></a><span data-ttu-id="2a389-106">例</span><span class="sxs-lookup"><span data-stu-id="2a389-106">Example</span></span>  
 <span data-ttu-id="2a389-107"><xref:System.Diagnostics.ConditionalAttribute>を使用した場合の CS0633 の例を次に示します。</span><span class="sxs-lookup"><span data-stu-id="2a389-107">This example illustrates CS0633 using the <xref:System.Diagnostics.ConditionalAttribute>.</span></span> <span data-ttu-id="2a389-108">次の例では CS0633 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="2a389-108">The following sample generates CS0633.</span></span>  
  
```csharp  
// CS0633a.cs  
#define DEBUG  
using System.Diagnostics;  
public class Test  
{  
   [Conditional("DEB+UG")]   // CS0633  
   // try the following line instead  
   // [Conditional("DEBUG")]  
   public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="2a389-109">例</span><span class="sxs-lookup"><span data-stu-id="2a389-109">Example</span></span>  
 <span data-ttu-id="2a389-110"><xref:System.Runtime.CompilerServices.IndexerNameAttribute>を使用した場合の CS0633 の例を次に示します。</span><span class="sxs-lookup"><span data-stu-id="2a389-110">This example illustrates CS0633 using the <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span></span>  
  
```csharp  
// CS0633b.cs  
// compile with: /target:module  
#define DEBUG  
using System.Runtime.CompilerServices;  
public class Test  
{  
   [IndexerName("Invalid+Identifier")]   // CS0633  
   // try the following line instead  
   // [IndexerName("DEBUG")]  
   public int this[int i]
   {
      get { return i; }
   }  
}  
```
