# <a name="compiler-error-cs1626"></a><span data-ttu-id="cc78c-103">コンパイラ エラー CS1626</span><span class="sxs-lookup"><span data-stu-id="cc78c-103">Compiler Error CS1626</span></span>

<span data-ttu-id="cc78c-104">catch 句を含む try ブロックの本体で値を生成することはできません。</span><span class="sxs-lookup"><span data-stu-id="cc78c-104">Cannot yield a value in the body of a try block with a catch clause</span></span>  
  
 <span data-ttu-id="cc78c-105">try ブロックに関連付けられている catch 句がある場合、その try ブロックでは yield ステートメントを使用できません。</span><span class="sxs-lookup"><span data-stu-id="cc78c-105">A yield statement is not allowed in a try block if there is a catch clause associated with the try block.</span></span> <span data-ttu-id="cc78c-106">このエラーを回避するには、yield ステートメントを try/catch/finally ブロックの外に移動するか、catch ブロックを削除します。</span><span class="sxs-lookup"><span data-stu-id="cc78c-106">To avoid this error, either move the yield statement out of the try/catch/finally block, or remove the catch block.</span></span>
  
 <span data-ttu-id="cc78c-107">次の例では CS1626 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="cc78c-107">The following sample generates CS1626:</span></span>  
  
```csharp  
// CS1626.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
         yield return this;  // CS1626  
      }  
      catch  
      {  
  
      }  
      finally
      {

      }
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
