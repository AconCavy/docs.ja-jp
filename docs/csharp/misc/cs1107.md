## <a name="example"></a><span data-ttu-id="bca74-106">例</span><span class="sxs-lookup"><span data-stu-id="bca74-106">Example</span></span>  
 <span data-ttu-id="bca74-107">次の例では CS1107 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="bca74-107">The following example generates CS1107:</span></span>  
  
```csharp

public static class Test
{
    // Extension methods.  
    public static void TestMethod(this this t) { } // CS1107  

}

public class TestTwo
{
    // Regular Instance Method  
    public void TestMethod(ref ref int i) { } // CS1107  

    // Regular Instance Method  
    public void TestMethod(in in double d) { } // CS1107  
}
```
