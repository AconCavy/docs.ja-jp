<span data-ttu-id="819bf-104">'constructor' : 静的コンストラクターにパラメーターを指定することはできません</span><span class="sxs-lookup"><span data-stu-id="819bf-104">'constructor' : a static constructor must be parameterless</span></span>  
  
 <span data-ttu-id="819bf-105">パラメーターを指定して、 [静的](../language-reference/keywords/static.md) コンストラクターを宣言することはできません。</span><span class="sxs-lookup"><span data-stu-id="819bf-105">A [static](../language-reference/keywords/static.md) constructor cannot be declared with one or more parameters.</span></span> <span data-ttu-id="819bf-106">詳細については、「[コンストラクター](../programming-guide/classes-and-structs/constructors.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="819bf-106">For more information, see [Constructors](../programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="819bf-107">次の例では CS0132 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="819bf-107">The following sample generates CS0132:</span></span>  
  
```csharp  
// CS0132.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public MyClass(int i)  
      {  
      }  
   }  
  
   public class MyClass2 : MyClass  
   {  
      static MyClass2(int i)   // CS0132  
      {  
      }  
   }  
}  
```
