 <span data-ttu-id="85b2d-107">次の例では CS0642 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="85b2d-107">The following sample generates CS0642:</span></span>  
  
```csharp  
// CS0642.cs

class MyClass  
{  
   public static void Main()  
   {  
      int i;  
  
      for (i = 0; i < 10; i += 1);   // CS0642 semicolon intentional?  
      {  
         System.Console.WriteLine (i);  
      }  
   }  
}  
```
