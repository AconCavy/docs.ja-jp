# <a name="compiler-warning-level-3-cs0642"></a>コンパイラの警告 (レベル 3) CS0642
empty ステートメントが間違っている可能性があります  
  
 条件付きステートメントの後にセミコロンがあると、コードが予定どおりに実行されないことがあります。  
  
 使用することができます **/nowarn**コンパイラ オプションまたは`#pragmas warning`; この警告を無効にするを参照してください[/nowarn (C#コンパイラ オプション)](../../csharp/language-reference/compiler-options/nowarn-compiler-option.md)または[#pragma warning](../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md)詳細情報。  
  
 次の例では CS0642 が生成されます。  
  
```csharp  
// CS0642.cs  
// compile with: /W:3  
class MyClass  
{  
   public static void Main()  
   {  
      int i;  
  
      for (i = 0; i < 10; i += 1);   // CS0642 semicolon intentional?  
      {  
         System.Console.WriteLine (i);  
      }  
   }  
}  
```
