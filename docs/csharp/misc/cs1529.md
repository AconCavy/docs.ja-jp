## <a name="example"></a><span data-ttu-id="f5cc7-106">例</span><span class="sxs-lookup"><span data-stu-id="f5cc7-106">Example</span></span>  
 <span data-ttu-id="f5cc7-107">次の例では CS1529 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="f5cc7-107">The following sample generates CS1529:</span></span>  
  
```csharp  
// CS1529.cs  
namespace X  
{  
    namespace Subspace  
    {  
        using Microsoft;  
  
        class SomeClass  
        {  
        };  
  
        using Microsoft;      // CS1529, place before class definition  
    }  
  
    using System.Reflection;  // CS1529, place before namespace 'Subspace'  
}  
  
using System;                 // CS1529, place at the beginning of the file  
```
