 <span data-ttu-id="748d6-105">通常の配列とは異なり、固定サイズ バッファーは宣言の時点で定数のサイズを指定する必要があります。</span><span class="sxs-lookup"><span data-stu-id="748d6-105">Unlike regular arrays, fixed size buffers require a constant size to be specified at the declaration point.</span></span> <span data-ttu-id="748d6-106">このエラーを解決するのには、正の整数リテラルか定数の正の整数を追加し、識別子の後に角かっこを配置します。</span><span class="sxs-lookup"><span data-stu-id="748d6-106">To resolve this error, add a positive integer literal or a constant positive integer and put the square brackets after the identifier.</span></span>  
  
 <span data-ttu-id="748d6-107">次の例では CS1641 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="748d6-107">The following sample generates CS1641:</span></span>  
  
```csharp

   fixed int [] a;  // CS1641

   fixed int d [c];  
}  
```
