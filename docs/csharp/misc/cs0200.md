## <a name="example"></a><span data-ttu-id="65f73-107">例</span><span class="sxs-lookup"><span data-stu-id="65f73-107">Example</span></span>  
 <span data-ttu-id="65f73-108">次の例では CS0200 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="65f73-108">The following sample generates CS0200:</span></span>  
  
```csharp  
// CS0200.cs  
public class MainClass  
{  
    // private int _mi;  
    int I  
    {  
        get  
        {  
            return 1;  
        }  
  
        // uncomment the set accessor and declaration for _mi  
        /*  
        set  
        {  
            _mi = value;  
        }  
        */  
    }  
  
    public static void Main ()  
    {  
        MainClass II = new MainClass();  
        II.I = 9;   // CS0200  
    }  
}  
```
