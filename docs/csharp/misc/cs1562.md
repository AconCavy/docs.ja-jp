# <a name="compiler-error-cs1562"></a>コンパイラ エラー CS1562
ソースのない出力には、/out オプションを指定しなければなりません。  
  
 コンパイルは、出力ファイルを作成できましたが、出力ファイルの名前を暗示する入力としてソース コード ファイルがありませんでした。 たとえば、メタデータ、またはリソースのみのファイルをコンパイルしようとしています。  
  
 [/out](../language-reference/compiler-options/out-compiler-option.md) コンパイラ オプションを使用して、出力ファイルの名前を指定します。
