# <a name="compiler-error-cs1577"></a>コンパイラ エラー CS1577
アセンブリの生成に失敗しました -- 理由  
  
 C# コンパイラがソース コードを正常にコンパイルした後、コンパイルのアセンブリ生成部分が失敗しました。 詳細については、ALink ユーティリティのエラーに関するドキュメント (「[Al.exe ツールのエラーと警告](../../framework/tools/al-exe-assembly-linker.md#errors-and-warnings)」) を参照してください。
