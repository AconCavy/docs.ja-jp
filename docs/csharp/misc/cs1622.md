# <a name="compiler-error-cs1622"></a><span data-ttu-id="36a86-103">コンパイラ エラー CS1622</span><span class="sxs-lookup"><span data-stu-id="36a86-103">Compiler Error CS1622</span></span>
<span data-ttu-id="36a86-104">反復子から値を返すことができません。</span><span class="sxs-lookup"><span data-stu-id="36a86-104">Cannot return a value from an iterator.</span></span> <span data-ttu-id="36a86-105">yield return ステートメントを使用して値を返すか、yield break ステートメントを使用して反復子を終了してください。</span><span class="sxs-lookup"><span data-stu-id="36a86-105">Use the yield return statement to return a value, or yield break to end the iteration.</span></span>  
  
 <span data-ttu-id="36a86-106">反復子は、return ステートメントではなく yield ステートメントで値を返す特殊な関数です。</span><span class="sxs-lookup"><span data-stu-id="36a86-106">An iterator is a special function that returns a value via the yield statement rather than the return statement.</span></span> <span data-ttu-id="36a86-107">詳細については、「 **反復子**」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="36a86-107">For more information, see **iterators**.</span></span>  
  
 <span data-ttu-id="36a86-108">次の例では CS1622 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="36a86-108">The following sample generates CS1622:</span></span>  
  
```csharp  
// CS1622.cs  
// compile with: /target:library  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      return (IEnumerator) this;  // CS1622  
      yield return this;   // OK  
   }  
}  
```
