# <a name="compiler-warning-level-2-cs0252"></a><span data-ttu-id="72ed6-103">コンパイラの警告 (レベル 2) CS0252</span><span class="sxs-lookup"><span data-stu-id="72ed6-103">Compiler Warning (level 2) CS0252</span></span>
<span data-ttu-id="72ed6-104">予期しない参照比較です。比較値を取得するには、型 'type' に左辺をキャストしてください</span><span class="sxs-lookup"><span data-stu-id="72ed6-104">Possible unintended reference comparison; to get a value comparison, cast the left hand side to type 'type'</span></span>  
  
 <span data-ttu-id="72ed6-105">コンパイラは参照比較を行っています。</span><span class="sxs-lookup"><span data-stu-id="72ed6-105">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="72ed6-106">文字列の値を比較する場合は、式の左辺を `type`にキャストします。</span><span class="sxs-lookup"><span data-stu-id="72ed6-106">If you want to compare the value of strings, cast the left side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="72ed6-107">次の例では CS0252 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="72ed6-107">The following sample generates CS0252:</span></span>  
  
```csharp  
// CS0252.cs  
// compile with: /W:2  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool b = o == s;   // CS0252  
      // try the following line instead  
      // bool b = (string)o == s;  
   }  
}  
```
