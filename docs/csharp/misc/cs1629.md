 <span data-ttu-id="f7b83-106">次の例では CS1629 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="f7b83-106">The following sample generates CS1629:</span></span>  
  
```csharp  
// CS1629.cs

{  
   IEnumerator<int> IteratorMeth() {

   }  
}
```
