# <a name="compiler-error-cs1625"></a><span data-ttu-id="91665-103">コンパイラ エラー CS1625</span><span class="sxs-lookup"><span data-stu-id="91665-103">Compiler Error CS1625</span></span>
<span data-ttu-id="91665-104">finally 句の本体で生成することはできません。</span><span class="sxs-lookup"><span data-stu-id="91665-104">Cannot yield in the body of a finally clause</span></span>  
  
 <span data-ttu-id="91665-105">yield ステートメントは、finally 句の本体で使用することはできません。</span><span class="sxs-lookup"><span data-stu-id="91665-105">A yield statement is not allowed in the body of a finally clause.</span></span> <span data-ttu-id="91665-106">このエラーを回避するには、yield ステートメントを finally 句の外に移動します。</span><span class="sxs-lookup"><span data-stu-id="91665-106">To avoid this error, move the yield statement out of the finally clause.</span></span>  
  
 <span data-ttu-id="91665-107">次の例では CS1625 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="91665-107">The following sample generates CS1625:</span></span>  
  
```csharp  
// CS1625.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
      }  
      finally  
      {  
        yield return this;  // CS1625  
      }  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
