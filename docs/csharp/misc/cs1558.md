# <a name="compiler-error-cs1558"></a><span data-ttu-id="0b044-103">コンパイラ エラー CS1558</span><span class="sxs-lookup"><span data-stu-id="0b044-103">Compiler Error CS1558</span></span>
<span data-ttu-id="0b044-104">'class' は適切なスタティック Main メソッドを含んでいません</span><span class="sxs-lookup"><span data-stu-id="0b044-104">'class' does not have a suitable static Main method</span></span>  
  
 <span data-ttu-id="0b044-105">[-Main](../language-reference/compiler-options/main-compiler-option.md)コンパイラオプションで、 **main**メソッドを検索するクラスが指定されました。</span><span class="sxs-lookup"><span data-stu-id="0b044-105">The [-main](../language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method.</span></span> <span data-ttu-id="0b044-106">しかし、 [Main](../programming-guide/main-and-command-args/index.md) メソッドが正しく定義されていません。</span><span class="sxs-lookup"><span data-stu-id="0b044-106">However, the [Main](../programming-guide/main-and-command-args/index.md) method was not defined correctly.</span></span>  
  
 <span data-ttu-id="0b044-107">次の例では、戻り値の型が無効であるため、CS1558 エラーを生成します。</span><span class="sxs-lookup"><span data-stu-id="0b044-107">The following example generates CS1558 because of invalid return type.</span></span>  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
