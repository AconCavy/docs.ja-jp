# <a name="compiler-error-cs1558"></a>コンパイラ エラー CS1558
'class' は適切なスタティック Main メソッドを含んでいません  
  
 [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) コンパイラ オプションで、 **Main** メソッドを検索するクラスを指定しました。 しかし、 [Main](../../csharp/programming-guide/main-and-command-args/index.md) メソッドが正しく定義されていません。  
  
 次の例では、戻り値の型が無効であるため、CS1558 エラーを生成します。  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()   
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
