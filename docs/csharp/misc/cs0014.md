# <a name="compiler-error-cs0014"></a>コンパイラ エラー CS0014
必要なファイル 'file' が見つかりませんでした  
  
 コンパイラで必要なファイルがシステムにありません。 パスが正しいことをご確認ください。 ファイルが Visual Studio システムファイルの場合は、インストールを修復するか、Visual Studio を完全に削除して再インストールする必要があります。
