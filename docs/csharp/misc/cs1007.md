## <a name="example"></a><span data-ttu-id="305e9-107">例</span><span class="sxs-lookup"><span data-stu-id="305e9-107">Example</span></span>

```csharp  
    {  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method

            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
