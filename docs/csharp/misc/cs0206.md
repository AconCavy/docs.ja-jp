## <a name="example"></a><span data-ttu-id="b0ec1-107">例</span><span class="sxs-lookup"><span data-stu-id="b0ec1-107">Example</span></span>  
 <span data-ttu-id="b0ec1-108">次の例では CS0206 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="b0ec1-108">The following sample generates CS0206:</span></span>  
  
```csharp  
// CS0206.cs  
public class MyClass

    public static int P  
    {  
        get  
        {  
            return 0;  
        }  
        set  
        {  
        }  
    }  
  
    public static void MyMeth(ref int i)  
    // public static void MyMeth(int i)  
    {  
    }

    {  
        MyMeth(ref P);   // CS0206  
        // try the following line instead  
        // MyMeth(P);   // CS0206  
    }  
}  
```
