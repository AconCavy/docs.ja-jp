# <a name="compiler-warning-level-3-cs0659"></a><span data-ttu-id="9f2b0-103">コンパイラの警告 (レベル 3) CS0659</span><span class="sxs-lookup"><span data-stu-id="9f2b0-103">Compiler Warning (level 3) CS0659</span></span>

<span data-ttu-id="9f2b0-104">'class' は Object.Equals(object o) をオーバーライドしますが、Object.GetHashCode() をオーバーライドしません。</span><span class="sxs-lookup"><span data-stu-id="9f2b0-104">'class' overrides Object.Equals(object o) but does not override Object.GetHashCode()</span></span>  

<span data-ttu-id="9f2b0-105">コンパイラは、メソッドのオーバーライドを検出しました <xref:System.Object.Equals%2A?displayProperty=nameWithType> が、メソッドのオーバーライドは検出しませんでした <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> 。</span><span class="sxs-lookup"><span data-stu-id="9f2b0-105">The compiler detected an override of the <xref:System.Object.Equals%2A?displayProperty=nameWithType> method but no override of the <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> method.</span></span> <span data-ttu-id="9f2b0-106">のオーバーライドは、 <xref:System.Object.Equals%2A> もオーバーライドすることを意味し <xref:System.Object.GetHashCode%2A> ます。</span><span class="sxs-lookup"><span data-stu-id="9f2b0-106">An override of <xref:System.Object.Equals%2A> implies that you also want to override <xref:System.Object.GetHashCode%2A>.</span></span>  

<span data-ttu-id="9f2b0-107">次のコードでは、CS0659 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="9f2b0-107">The following code generates CS0659:</span></span>  

```csharp
// CS0659.cs  
// compile with: /W:3 /target:library  
class Test
{  
   public override bool Equals(object o) { return true; }   // CS0659  
}  
  
// OK  
class Test2  
{  
   public override bool Equals(object o) { return true; }  
   public override int GetHashCode() { return 0; }  
}  
```
