# <a name="compiler-error-cs0674"></a><span data-ttu-id="13347-103">コンパイラ エラー CS0674</span><span class="sxs-lookup"><span data-stu-id="13347-103">Compiler Error CS0674</span></span>
<span data-ttu-id="13347-104">'System.ParamArrayAttribute' は使用しないでください。</span><span class="sxs-lookup"><span data-stu-id="13347-104">Do not use 'System.ParamArrayAttribute'.</span></span> <span data-ttu-id="13347-105">代わりに 'params' キーワードを使用します。</span><span class="sxs-lookup"><span data-stu-id="13347-105">Use the 'params' keyword instead.</span></span>  
  
 <span data-ttu-id="13347-106">C# コンパイラでは、 <xref:System.ParamArrayAttribute?displayProperty=nameWithType>を使用できません。代わりに [params](../language-reference/keywords/params.md) を使用します。</span><span class="sxs-lookup"><span data-stu-id="13347-106">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="13347-107">次の例では CS0674 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="13347-107">The following sample generates CS0674:</span></span>  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
