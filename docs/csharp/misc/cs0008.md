# <a name="compiler-error-cs0008"></a>コンパイラ エラー CS0008
ファイル 'file' からメタデータを読み込んでいるときに予期しないエラーが発生しました -- 'description'  
  
 メタデータを取得するために DLL が正常に開かれましたが、破損しているためデータを読み取ることができませんでした。 詳しくは、「[/reference (C# コンパイラ オプション)](../language-reference/compiler-options/reference-compiler-option.md)」をご覧ください。
