# <a name="compiler-error-cs0523"></a><span data-ttu-id="719b6-103">コンパイラ エラー CS0523</span><span class="sxs-lookup"><span data-stu-id="719b6-103">Compiler Error CS0523</span></span>

<span data-ttu-id="719b6-104">型 'struct1' の構造体メンバー 'struct2 field' により、構造体レイアウトで循環参照が発生します</span><span class="sxs-lookup"><span data-stu-id="719b6-104">Struct member 'struct2 field' of type 'struct1' causes a cycle in the struct layout</span></span>  
  
 <span data-ttu-id="719b6-105">2 つの構造体の定義に再帰参照が含まれています。</span><span class="sxs-lookup"><span data-stu-id="719b6-105">The definitions of two structs include recursive references.</span></span> <span data-ttu-id="719b6-106">他の構造体に基づいて定義しないように、[構造体](../builtin-types/struct.md)の定義を変更してください。</span><span class="sxs-lookup"><span data-stu-id="719b6-106">Change the [struct](../builtin-types/struct.md) definitions such that each does not define itself on the other.</span></span> <span data-ttu-id="719b6-107">構造体は値の型なので、この制限は構造体にのみ適用されます。</span><span class="sxs-lookup"><span data-stu-id="719b6-107">This limitation applies only to structs, since structs are value types.</span></span> <span data-ttu-id="719b6-108">再帰参照を使用している場合は、型をクラスとして宣言してください。</span><span class="sxs-lookup"><span data-stu-id="719b6-108">If using recursive references, declare your types as classes.</span></span>  
  
 <span data-ttu-id="719b6-109">次の例では CS0523 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="719b6-109">The following sample generates CS0523:</span></span>  
  
```csharp  
// CS0523.cs  
// compile with: /target:library  
struct RecursiveLayoutStruct1  
{  
   public RecursiveLayoutStruct2 field;  
}  
  
struct RecursiveLayoutStruct2  
{  
   public RecursiveLayoutStruct1 field;   // CS0523  
}  
```
