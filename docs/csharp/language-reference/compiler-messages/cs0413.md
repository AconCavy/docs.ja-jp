# <a name="compiler-error-cs0413"></a><span data-ttu-id="3ec30-103">コンパイラ エラー CS0413</span><span class="sxs-lookup"><span data-stu-id="3ec30-103">Compiler Error CS0413</span></span>

<span data-ttu-id="3ec30-104">型パラメーター 'type parameter' にはクラス型制約も 'class' 制約も含まれないため、'as' 演算子で使用できません</span><span class="sxs-lookup"><span data-stu-id="3ec30-104">The type parameter 'type parameter' cannot be used with the 'as' operator because it does not have a class type constraint nor a 'class' constraint</span></span>  
  
<span data-ttu-id="3ec30-105">このエラーは、ジェネリック型で [as](../operators/type-testing-and-cast.md#as-operator) 演算子を使用しているにもかかわらず、ジェネリック型にクラス型制約がない場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="3ec30-105">This error occurs if a generic type uses the [as](../operators/type-testing-and-cast.md#as-operator) operator, but that generic type does not have a class type constraint.</span></span> <span data-ttu-id="3ec30-106">`as` 演算子は、参照型および null 許容値型にのみ使用できるため、型パラメーターを制約して、値の型ではないことを保証する必要があります。</span><span class="sxs-lookup"><span data-stu-id="3ec30-106">The `as` operator is only allowed with reference and nullable value types, so the type parameter must be constrained to guarantee that it is not a value type.</span></span> <span data-ttu-id="3ec30-107">このエラーを回避するには、クラス型制約または参照型の制約を使用します。</span><span class="sxs-lookup"><span data-stu-id="3ec30-107">To avoid this error, use a class type constraint or a reference type constraint.</span></span>
  
<span data-ttu-id="3ec30-108">これは、`as` 演算子が、値の型で可能な値ではない `null` を返す可能性があるためです。クラス型制約または参照型の制約でない場合、型パラメーターは値の型として扱う必要があります。</span><span class="sxs-lookup"><span data-stu-id="3ec30-108">This is because the `as` operator could return `null`, which is not a possible value for a value type, and the type parameter must be treated as a value type unless it is a class type constraint or a reference type constraint.</span></span>
  
## <a name="example"></a><span data-ttu-id="3ec30-109">例</span><span class="sxs-lookup"><span data-stu-id="3ec30-109">Example</span></span>

<span data-ttu-id="3ec30-110">次の例では CS0413 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="3ec30-110">The following sample generates CS0413.</span></span>  
  
```csharp  
// CS0413.cs  
// compile with: /target:library  
class A {}  
class B : A {}  
  
class CMain  
{  
   A a = null;  
   public void G<T>()  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  // CS0413  
   }  
  
   // OK  
   public void H<T>() where T : A  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  
   }  
}  
```
