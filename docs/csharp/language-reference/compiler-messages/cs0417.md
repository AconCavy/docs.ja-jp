# <a name="compiler-error-cs0417"></a><span data-ttu-id="28160-103">コンパイラ エラー CS0417</span><span class="sxs-lookup"><span data-stu-id="28160-103">Compiler Error CS0417</span></span>

<span data-ttu-id="28160-104">'<識別子>': 変数型のインスタンスを作成するときに、引数を指定できません</span><span class="sxs-lookup"><span data-stu-id="28160-104">'identifier': cannot provide arguments when creating an instance of a variable type</span></span>  
  
 <span data-ttu-id="28160-105">このエラーは、型パラメーターの `new` 演算子の呼び出しで、引数を指定した場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="28160-105">This error occurs if a call to the `new` operator on a type parameter has arguments.</span></span> <span data-ttu-id="28160-106">未知のパラメーター型の `new` 演算子を使用して呼び出すことができる唯一のコンストラクターは、引数を持たないコンストラクターです。</span><span class="sxs-lookup"><span data-stu-id="28160-106">The only constructor that can be called by using the `new` operator on an unknown parameter type is a constructor that has no arguments.</span></span> <span data-ttu-id="28160-107">他のコンストラクターを呼び出す場合は、クラス型制約またはインターフェイス制約を使用してください。</span><span class="sxs-lookup"><span data-stu-id="28160-107">If you need to call another constructor, consider using a class type constraint or interface constraint.</span></span>  
  
## <a name="example"></a><span data-ttu-id="28160-108">例</span><span class="sxs-lookup"><span data-stu-id="28160-108">Example</span></span>  

 <span data-ttu-id="28160-109">次の例では CS0417 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="28160-109">The following example generates CS0417:</span></span>  

```csharp
// CS0417  
class ExampleClass<T> where T : new()  
{  
    // The following line causes CS0417.  
    T instance1 = new T(1);
  
    // The following line doesn't cause the error.  
    T instance2 = new T();  
}  
```
