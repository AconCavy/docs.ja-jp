 <span data-ttu-id="10bb8-104">アプリケーション内で参照される 2 つの異なるアセンブリに同じ名前空間と型が含まれているため、あいまいさが生じています。</span><span class="sxs-lookup"><span data-stu-id="10bb8-104">Two different assemblies referenced in your application contain the same namespace and type, which produces ambiguity.</span></span>

## <a name="example"></a><span data-ttu-id="10bb8-110">例</span><span class="sxs-lookup"><span data-stu-id="10bb8-110">Example</span></span>

```csharp  
}  
```

## <a name="example"></a><span data-ttu-id="10bb8-114">例</span><span class="sxs-lookup"><span data-stu-id="10bb8-114">Example</span></span>  
 <span data-ttu-id="10bb8-115">次の例では CS0433 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="10bb8-115">The following example generates CS0433.</span></span>  
  
```csharp

using TypeBindConflicts;  
public class Test
{  
   public static void Main()
   {  
      AggPubImpAggPubImp n6 = new AggPubImpAggPubImp();   // CS0433  
   }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="10bb8-116">例</span><span class="sxs-lookup"><span data-stu-id="10bb8-116">Example</span></span>

```csharp  
   }  
}  
```
