## <a name="example"></a><span data-ttu-id="4a491-118">例</span><span class="sxs-lookup"><span data-stu-id="4a491-118">Example</span></span>

 <span data-ttu-id="4a491-119">次の例では CS0106 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="4a491-119">The following sample generates CS0106:</span></span>

```csharp
// CS0106.cs
namespace MyNamespace
{
   interface I
   {
      void M();
   }

   public class MyClass : I
   {
      public void I.M() {}   // CS0106
      public static void Main() {}
   }
}
```
