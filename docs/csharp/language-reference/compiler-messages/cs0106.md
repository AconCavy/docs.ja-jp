## <a name="example"></a><span data-ttu-id="58ace-114">例</span><span class="sxs-lookup"><span data-stu-id="58ace-114">Example</span></span>

 <span data-ttu-id="58ace-115">次の例では CS0106 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="58ace-115">The following sample generates CS0106:</span></span>

```csharp
// CS0106.cs

{
   interface I
   {
      void m();
   }

   public class MyClass
   {
      public void I.m() {}   // CS0106
      public static void Main() {}
   }
}
```
