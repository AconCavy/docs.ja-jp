# <a name="compiler-warning-level-2-cs0467"></a><span data-ttu-id="2283d-102">コンパイラの警告 (レベル 2) CS0467</span><span class="sxs-lookup"><span data-stu-id="2283d-102">Compiler Warning (level 2) CS0467</span></span>
<span data-ttu-id="2283d-103">メソッド '<メソッド>' とメソッド以外の 'non-method' があいまいです。</span><span class="sxs-lookup"><span data-stu-id="2283d-103">Ambiguity between method 'method' and non-method 'non-method'.</span></span> <span data-ttu-id="2283d-104">メソッド グループを使用しています。</span><span class="sxs-lookup"><span data-stu-id="2283d-104">Using method group.</span></span>  
  
 <span data-ttu-id="2283d-105">別のインターフェイスからメンバーが継承され、同じシグネチャによってあいまいさのエラーが発生します。</span><span class="sxs-lookup"><span data-stu-id="2283d-105">Inherited members from different interfaces that have the same signature  cause an ambiguity error.</span></span>  
  
## <a name="example"></a><span data-ttu-id="2283d-106">例</span><span class="sxs-lookup"><span data-stu-id="2283d-106">Example</span></span>  
 <span data-ttu-id="2283d-107">次の例では CS0467 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="2283d-107">The following example generates CS0467.</span></span>  
  
```csharp  
// CS0467.cs  
interface IList
{  
    int Count { get; set; }  
}  
  
interface ICounter  
{  
    void Count(int i);  
}  
  
interface IListCounter : IList, ICounter {}  
  
class Driver
{  
    void Test(IListCounter x)  
    {  
        // The following line causes the warning. The assignment also  
        // causes an error because you can't assign a value to a method.  
        x.Count = 1;  
        x.Count(3);
        // To resolve the warning, you can change the name of the method or
        // the property.  
  
        // You can also disambiguate by specifying IList or ICounter.  
        ((IList)x).Count = 1;  
        ((ICounter)x).Count(3);  
    }  
  
    static void Main()
    {  
    }  
}  
```
