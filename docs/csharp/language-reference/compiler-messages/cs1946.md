# <a name="compiler-error-cs1946"></a><span data-ttu-id="c3e03-103">コンパイラ エラー CS1946</span><span class="sxs-lookup"><span data-stu-id="c3e03-103">Compiler Error CS1946</span></span>

<span data-ttu-id="c3e03-104">匿名メソッド式を式のツリーに変換することはできません。</span><span class="sxs-lookup"><span data-stu-id="c3e03-104">An anonymous method expression cannot be converted to an expression tree.</span></span>

<span data-ttu-id="c3e03-105">[匿名メソッド](../operators/delegate-operator.md)は一連のステートメントを表しますが、式ツリーにステートメントを含めることはできません。</span><span class="sxs-lookup"><span data-stu-id="c3e03-105">An [anonymous method](../operators/delegate-operator.md) represents a set of statements but an expression tree must not contain a statement.</span></span> <span data-ttu-id="c3e03-106">したがって、匿名メソッドを式のツリーで表すことはできません。</span><span class="sxs-lookup"><span data-stu-id="c3e03-106">Therefore an anonymous method cannot be represented by an expression tree.</span></span>

<span data-ttu-id="c3e03-107">このエラーを修正するには、匿名メソッドを[ラムダ式](../operators/lambda-expressions.md)に変更します。</span><span class="sxs-lookup"><span data-stu-id="c3e03-107">To correct this error, change the anonymous method to a [lambda expression](../operators/lambda-expressions.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="c3e03-108">例</span><span class="sxs-lookup"><span data-stu-id="c3e03-108">Example</span></span>

<span data-ttu-id="c3e03-109">次の例では CS1946 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="c3e03-109">The following example generates CS1946:</span></span>  

```csharp
// cs1946.cs  
using System;  
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.  
            // Expression<D> tree = (x) => x + 1;  
        }  
    }  
```
