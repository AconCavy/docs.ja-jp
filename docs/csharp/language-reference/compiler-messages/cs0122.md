## <a name="example"></a><span data-ttu-id="4d3e2-108">例</span><span class="sxs-lookup"><span data-stu-id="4d3e2-108">Example</span></span>

 <span data-ttu-id="4d3e2-109">次の例では CS0122 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="4d3e2-109">The following sample generates CS0122:</span></span>

```csharp
// CS0122.cs
public class MyClass
{
    // Make public to resolve CS0122.
    void MyMethod()
    {
    }
}

public class MyClass2
{
    public static int Main()
    {  
        var a = new MyClass();  
        // MyMethod is private.
        a.MyMethod();   // CS0122
        return 0;
   }
}
```
