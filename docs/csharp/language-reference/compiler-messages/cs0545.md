# <a name="compiler-error-cs0545"></a><span data-ttu-id="ea143-103">コンパイラ エラー CS0545</span><span class="sxs-lookup"><span data-stu-id="ea143-103">Compiler Error CS0545</span></span>

<span data-ttu-id="ea143-104">'function': 'property' に、オーバーライド可能な get アクセサーがないため、オーバーライドできません。</span><span class="sxs-lookup"><span data-stu-id="ea143-104">'function' : cannot override because 'property' does not have an overridable get accessor</span></span>  
  
 <span data-ttu-id="ea143-105">基底クラスにこのようなオーバーライドする定義がない場合に、プロパティ アクセサーのオーバーライドを定義する処理が試行されました。</span><span class="sxs-lookup"><span data-stu-id="ea143-105">A try was made to define an override for a property accessor when the base class has no such definition to override.</span></span> <span data-ttu-id="ea143-106">このエラーは、次の方法で解決できます。</span><span class="sxs-lookup"><span data-stu-id="ea143-106">You can resolve this error by:</span></span>  
  
- <span data-ttu-id="ea143-107">基底クラスに `set` アクセサーを追加します。</span><span class="sxs-lookup"><span data-stu-id="ea143-107">Adding a `set` accessor in the base class.</span></span>  
  
- <span data-ttu-id="ea143-108">派生クラスから `set` アクセサーを削除します。</span><span class="sxs-lookup"><span data-stu-id="ea143-108">Removing the `set` accessor from the derived class.</span></span>  
  
- <span data-ttu-id="ea143-109">[new](../keywords/new-modifier.md) キーワードを派生クラスのプロパティに追加することで、基底クラス プロパティを非表示にします。</span><span class="sxs-lookup"><span data-stu-id="ea143-109">Hiding the base class property by adding the [new](../keywords/new-modifier.md) keyword to a property in a derived class.</span></span>  
  
- <span data-ttu-id="ea143-110">基底クラスのプロパティを [virtual](../keywords/virtual.md) にします。</span><span class="sxs-lookup"><span data-stu-id="ea143-110">Making the base class property [virtual](../keywords/virtual.md).</span></span>  
  
 <span data-ttu-id="ea143-111">詳細については、「 [Using Properties](../../programming-guide/classes-and-structs/using-properties.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="ea143-111">For more information, see [Using Properties](../../programming-guide/classes-and-structs/using-properties.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="ea143-112">例</span><span class="sxs-lookup"><span data-stu-id="ea143-112">Example</span></span>  

 <span data-ttu-id="ea143-113">次の例では CS0545 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="ea143-113">The following sample generates CS0545.</span></span>  
  
```csharp  
// CS0545.cs  
// compile with: /target:library  
// CS0545  
public class a  
{  
   public virtual int i  
   {  
      set {}  
  
      // Uncomment the following line to resolve.  
      // get { return 0; }  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      get { return 0; }  
      set {}   // OK  
   }  
}  
```
