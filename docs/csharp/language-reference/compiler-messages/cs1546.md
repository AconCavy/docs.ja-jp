## <a name="example"></a><span data-ttu-id="9ba54-108">例</span><span class="sxs-lookup"><span data-stu-id="9ba54-108">Example</span></span>  
 <span data-ttu-id="9ba54-109">このコードは、.dll にコンパイルされる .cpp ファイルと、その .dll を使う .cs ファイルで構成されています。</span><span class="sxs-lookup"><span data-stu-id="9ba54-109">This code sample consists of a .cpp file, which compiles to a .dll, and a .cs file, which uses that .dll.</span></span> <span data-ttu-id="9ba54-110">次のコードは、.dll ファイルのものであり、.cs ファイルのコードからアクセスされるプロパティを定義しています。</span><span class="sxs-lookup"><span data-stu-id="9ba54-110">The following code is for the .dll file and defines a property to be accessed by the code in the .cs file.</span></span>  
  
```cpp  
// CPP1546.cpp  
// compile with: /clr /LD  
using namespace System;  
public ref class MCPP  
{  
public:  
    property int Prop [int,int]  
    {  
        int get( int i, int b )  
        {  
            return i;  
        }  
    }  
};  
```
