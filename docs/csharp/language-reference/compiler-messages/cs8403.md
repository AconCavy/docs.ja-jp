# <a name="compiler-error-cs8403"></a>コンパイラ エラー CS8403

反復子ブロックのある "method" は "async" で "{IAsyncEnumerable\<T>|IAsyncEnumerator\<T>}" を返す必要があります

## <a name="to-correct-this-error"></a>このエラーを解決するには

メソッドを `async` 修飾子でマークします。
