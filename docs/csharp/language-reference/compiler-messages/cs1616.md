# <a name="compiler-warning-level-1-cs1616"></a><span data-ttu-id="59b41-103">コンパイラの警告 (レベル 1) CS1616</span><span class="sxs-lookup"><span data-stu-id="59b41-103">Compiler Warning (level 1) CS1616</span></span>
<span data-ttu-id="59b41-104">オプション 'option' は、ソース ファイルまたは追加されたモジュールで指定された属性 'attribute' をオーバーライドします</span><span class="sxs-lookup"><span data-stu-id="59b41-104">Option 'option' overrides attribute 'attribute' given in a source file or added module</span></span>  
  
 <span data-ttu-id="59b41-105">この警告は、ソースで見つかったアセンブリ属性 <xref:System.Reflection.AssemblyKeyFileAttribute> または <xref:System.Reflection.AssemblyKeyNameAttribute> が、コマンド ライン オプション [-keyfile](../compiler-options/keyfile-compiler-option.md) または [-keycontainer](../compiler-options/keycontainer-compiler-option.md) と競合している場合、またはプロジェクトのプロパティで指定されたキー ファイル名またはキー コンテナーと競合している場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="59b41-105">This warning occurs if the assembly attributes <xref:System.Reflection.AssemblyKeyFileAttribute> or <xref:System.Reflection.AssemblyKeyNameAttribute> found in source conflict with the [-keyfile](../compiler-options/keyfile-compiler-option.md) or [-keycontainer](../compiler-options/keycontainer-compiler-option.md) command line option or key file name or key container specified in the Project Properties.</span></span>  
  
 <span data-ttu-id="59b41-106">次の例では、`cs1616.snk` というキー ファイルを想定しています。</span><span class="sxs-lookup"><span data-stu-id="59b41-106">For the example below, assume you have a key file named `cs1616.snk`.</span></span> <span data-ttu-id="59b41-107">このファイルは、コマンド ラインで次のように入力して生成します。</span><span class="sxs-lookup"><span data-stu-id="59b41-107">Generate this file with the command line:</span></span>  
  
```console  
sn -k CS1616.snk  
```  
  
 <span data-ttu-id="59b41-108">次の例では CS1616 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="59b41-108">The following sample generates CS1616:</span></span>  
  
```csharp

// compile with: /keyfile:cs1616.snk  
using System.Reflection;  
  
// To fix the error, remove the next line

{  
  public static void Main()  
  {  
  }  
}  
```
