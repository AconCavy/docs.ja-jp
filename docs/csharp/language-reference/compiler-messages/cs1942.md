# <a name="compiler-error-cs1942"></a><span data-ttu-id="2547a-102">コンパイラ エラー CS1942</span><span class="sxs-lookup"><span data-stu-id="2547a-102">Compiler Error CS1942</span></span>

<span data-ttu-id="2547a-103">'clause' 句の式の型が正しくありません。</span><span class="sxs-lookup"><span data-stu-id="2547a-103">The type of the expression in the 'clause' clause is incorrect.</span></span> <span data-ttu-id="2547a-104">'method' の呼び出しで型を推論できませんでした。</span><span class="sxs-lookup"><span data-stu-id="2547a-104">Type inference failed in the call to 'method'.</span></span>  
  
<span data-ttu-id="2547a-105">通常、このエラーは、範囲変数に明示的な型が正しく指定されていない場合に生成されます。</span><span class="sxs-lookup"><span data-stu-id="2547a-105">This error is typically generated when the range variable has been given an incorrect explicit type.</span></span>  

## <a name="to-correct-this-error"></a><span data-ttu-id="2547a-106">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="2547a-106">To correct this error</span></span>

1. <span data-ttu-id="2547a-107">範囲変数が明示的に型指定されている場合は、その型が、反復処理するコレクション内の要素と同じ型か、またはその要素の型から暗黙的に変換できる型どうかを確認します。</span><span class="sxs-lookup"><span data-stu-id="2547a-107">If the range variable is explicitly typed, make sure that the type is either the same as, or implicitly convertible from, the type of the elements in the collection it iterates.</span></span> <span data-ttu-id="2547a-108">範囲変数の前に `var` キーワードが指定されている場合は、`var` を削除します。</span><span class="sxs-lookup"><span data-stu-id="2547a-108">If the range variable is preceded with the `var` keyword, remove `var`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="2547a-109">例</span><span class="sxs-lookup"><span data-stu-id="2547a-109">Example</span></span>

<span data-ttu-id="2547a-110">次のコードでは CS1942 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="2547a-110">The following code generates CS1942:</span></span>  

```csharp
// cs1942.cs  
class Program  
    {  
        static void Main(string[] args)  
        {  
            var x = from var i in Enumerable.Range(1, 100) // CS1949  
                    select i; //CS1942  
        }  
    }  
```

<span data-ttu-id="2547a-111">範囲変数に `var` を使用すると、`var` が型ではないことで基になる `Cast<T>` 操作に失敗するため、CS1942 は CS1949 に関連して発生します。</span><span class="sxs-lookup"><span data-stu-id="2547a-111">CS1942 is related to CS1949 because the use of `var` with a range variable causes the underlying `Cast<T>` operation to fail because `var` is not a type.</span></span>
