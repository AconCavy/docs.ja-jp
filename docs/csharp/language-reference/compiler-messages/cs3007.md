# <a name="compiler-warning-level-1-cs3007"></a><span data-ttu-id="e886b-102">コンパイラの警告 (レベル 1) CS3007</span><span class="sxs-lookup"><span data-stu-id="e886b-102">Compiler Warning (level 1) CS3007</span></span>
<span data-ttu-id="e886b-103">名前のない配列型のみが異なるオーバーロードされたメソッド 'メソッド' は、CLS に準拠していません。</span><span class="sxs-lookup"><span data-stu-id="e886b-103">Overloaded method 'method' differing only by unnamed array types is not CLS-compliant</span></span>  
  
 <span data-ttu-id="e886b-104">このエラーは、ジャグ配列を受け取る、オーバーロードされたメソッドが存在し、配列要素の型以外にメソッド シグネチャの違いがない場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="e886b-104">This error occurs if you have an overloaded method that takes a jagged array and the only difference between the method signatures is the element type of the array.</span></span> <span data-ttu-id="e886b-105">このエラーを回避するには、ジャグ配列ではなく四角形配列を使用する、パラメーターを追加して関数呼び出しを明確にする、または、オーバーロードされたメソッドの名前を変更します。CLS 準拠でなくてもかまわない場合は、<xref:System.CLSCompliantAttribute> 属性を削除します。</span><span class="sxs-lookup"><span data-stu-id="e886b-105">To avoid this error, consider using a rectangular array rather than a jagged array; use an additional parameter to disambiguate the function call; rename one or more of the overloaded methods; or, if CLS Compliance is not needed, remove the <xref:System.CLSCompliantAttribute> attribute.</span></span> <span data-ttu-id="e886b-106">CLS 準拠については詳しくは、「[言語への非依存性、および言語非依存コンポーネント](../../../standard/language-independence-and-language-independent-components.md)」をご覧ください。</span><span class="sxs-lookup"><span data-stu-id="e886b-106">For more information on CLS Compliance, see [Language Independence and Language-Independent Components](../../../standard/language-independence-and-language-independent-components.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="e886b-107">例</span><span class="sxs-lookup"><span data-stu-id="e886b-107">Example</span></span>  
 <span data-ttu-id="e886b-108">次の例では、CS3007 エラーが生成されます。</span><span class="sxs-lookup"><span data-stu-id="e886b-108">The following example generates CS3007:</span></span>  
  
```csharp  
// CS3007.cs  
[assembly: System.CLSCompliant(true)]  
public struct S  
{  
    public void F(int[][] array) { }  
    public void F(byte[][] array) { }  // CS3007  
    // Try this instead:  
    // public void F1(int[][] array) {}  
    // public void F2(byte[][] array) {}  
    // or
    // public void F(int[,] array) {}  
    // public void F(byte[,] array) {}  
}  
```
