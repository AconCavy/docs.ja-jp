# <a name="compiler-error-cs8400"></a>コンパイラ エラー CS8400

機能 "feature" は C# 8.0 では利用できません。 言語バージョン \<version> 以上を使用してください。

## <a name="to-correct-this-error"></a>このエラーを解決するには

エラーで言及されたバージョン以上にプロジェクトをアップグレードしてください。
