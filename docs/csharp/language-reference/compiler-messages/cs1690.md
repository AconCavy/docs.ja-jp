# <a name="compiler-warning-level-1-cs1690"></a><span data-ttu-id="3cf0c-103">コンパイラの警告 (レベル 1) CS1690</span><span class="sxs-lookup"><span data-stu-id="3cf0c-103">Compiler Warning (level 1) CS1690</span></span>

<span data-ttu-id="3cf0c-104">参照マーシャリング クラスのフィールドであるため、'member' のメンバーにアクセスすると、ランタイム例外が発生する可能性があります</span><span class="sxs-lookup"><span data-stu-id="3cf0c-104">Accessing a member on 'member' may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="3cf0c-105">この警告は、<xref:System.MarshalByRefObject> から派生するクラスの値型のメンバーに対して、メソッド、プロパティ、またはインデクサーを呼び出そうとした場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="3cf0c-105">This warning occurs when you try to call a method, property, or indexer on a member of a class that derives from <xref:System.MarshalByRefObject>, and the member is a value type.</span></span> <span data-ttu-id="3cf0c-106">`MarshalByRefObject` から継承されたオブジェクトは、通常、アプリケーション ドメインを越えて参照によってマーシャリングされることを意図しています。</span><span class="sxs-lookup"><span data-stu-id="3cf0c-106">Objects that inherit from `MarshalByRefObject` are typically intended to be marshaled by reference across an application domain.</span></span> <span data-ttu-id="3cf0c-107">そのようなオブジェクトの値型メンバーに、アプリケーション ドメインを越えてコードから直接アクセスしようとすると、ランタイム <xref:System.InvalidOperationException> が発生します。</span><span class="sxs-lookup"><span data-stu-id="3cf0c-107">If any code ever attempts to directly access the value-type member of such an object across an application domain, a runtime <xref:System.InvalidOperationException> will occur.</span></span> <span data-ttu-id="3cf0c-108">警告を解決するには、まずメンバーをローカル変数にコピーし、この変数でメソッドを呼び出します。</span><span class="sxs-lookup"><span data-stu-id="3cf0c-108">To resolve the warning, first copy the member into a local variable and call the method on that variable.</span></span>
  
 <span data-ttu-id="3cf0c-109">次の例では CS1690 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="3cf0c-109">The following sample generates CS1690:</span></span>  
  
```csharp  
// CS1690.cs  
using System;

class WarningCS1690 : MarshalByRefObject
{
    int i = 5;

    public static void Main()
    {
        AppDomain domain = AppDomain.CreateDomain("MyDomain");
        Type t = typeof(WarningCS1690);
        WarningCS1690 e = (WarningCS1690)domain.CreateInstanceAndUnwrap(t.Assembly.FullName,t.FullName);

        e.i.ToString(); // CS1690

        // OK  
        int i = e.i;
        i.ToString();
        e.i = i;
    }
}
```
