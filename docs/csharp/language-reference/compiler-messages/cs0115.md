## <a name="example"></a><span data-ttu-id="6761a-107">例</span><span class="sxs-lookup"><span data-stu-id="6761a-107">Example</span></span>

 <span data-ttu-id="6761a-108">次の例では CS0115 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="6761a-108">The following sample generates CS0115.</span></span> <span data-ttu-id="6761a-109">2 つの方法のいずれかで CS0115 で解決できます。</span><span class="sxs-lookup"><span data-stu-id="6761a-109">You can resolve CS0115 in one of two ways:</span></span>

- <span data-ttu-id="6761a-110">`MyClass2` のメソッドから `override` キーワードを削除します。</span><span class="sxs-lookup"><span data-stu-id="6761a-110">Remove the `override` keyword from the method in `MyClass2`.</span></span>

- <span data-ttu-id="6761a-111">`MyClass2` の基底クラスとして `MyClass1` を使用します。</span><span class="sxs-lookup"><span data-stu-id="6761a-111">Use `MyClass1` as a base class for `MyClass2`.</span></span>

```csharp
// CS0115.cs
namespace MyNamespace
{
    abstract public class MyClass1
    {
        public abstract int f();
    }

    abstract public class MyClass2
    {
        public override int f()   // CS0115
        {
            return 0;
        }

        public static void Main()
        {
        }
    }
}
```
