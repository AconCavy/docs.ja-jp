## <a name="example-of-ambiguous-conversion"></a><span data-ttu-id="1b652-107">あいまいな変換の例</span><span class="sxs-lookup"><span data-stu-id="1b652-107">Example of ambiguous conversion</span></span>

<span data-ttu-id="1b652-108">このエラーは、変換がなかった場合、または使用できる暗黙的な変換結果があいまいな状況になった場合に発生する可能性があります。</span><span class="sxs-lookup"><span data-stu-id="1b652-108">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="1b652-109">次の例では CS0151 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="1b652-109">The following example generates CS0151:</span></span>

```csharp
public class MyClass
{
   public static implicit operator int (MyClass aa)
   {
      return 0;
   }

   public static implicit operator long (MyClass aa)
   {
      return 0;
   }

   public static void Main()
   {
      MyClass a = new MyClass();

      // Compiler cannot choose between int and long.
      switch (a)   // CS0151
      // try the following line instead
      // switch ((int)a)
      {
         case 1:
            break;
      }
   }
}
```

## <a name="example-of-void-method"></a><span data-ttu-id="1b652-110">void メソッドの例</span><span class="sxs-lookup"><span data-stu-id="1b652-110">Example of void method</span></span>

<span data-ttu-id="1b652-111">[switch](../keywords/switch.md) match 式で [void](../builtin-types/void.md) メソッドを呼び出すと、CS0151 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="1b652-111">A [void](../builtin-types/void.md) method invocation in a [switch](../keywords/switch.md) match expression generates CS0151.</span></span> <span data-ttu-id="1b652-112">このエラーを修正するには、代わりに [int](../builtin-types/integral-numeric-types.md) や [long](../builtin-types/integral-numeric-types.md) などの整数値を返すメソッドを呼び出します。</span><span class="sxs-lookup"><span data-stu-id="1b652-112">You can fix the error by calling a method that returns an integral type such as [int](../builtin-types/integral-numeric-types.md) or [long](../builtin-types/integral-numeric-types.md) instead.</span></span>

```csharp
class C
{
    static void Main()
    {
        switch (M()) // CS0151
        {
            default:
                break;
        }
    }

    static void M()
    {
    }
}
```
