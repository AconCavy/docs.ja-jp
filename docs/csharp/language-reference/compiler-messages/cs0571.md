# <a name="compiler-error-cs0571"></a><span data-ttu-id="8ad7e-103">コンパイラ エラー CS0571</span><span class="sxs-lookup"><span data-stu-id="8ad7e-103">Compiler Error CS0571</span></span>
<span data-ttu-id="8ad7e-104">'function': 演算子またはアクセサーを明示的に呼び出すことはできません</span><span class="sxs-lookup"><span data-stu-id="8ad7e-104">'function' : cannot explicitly call operator or accessor</span></span>  
  
 <span data-ttu-id="8ad7e-105">一部の演算子には内部名があります。</span><span class="sxs-lookup"><span data-stu-id="8ad7e-105">Certain operators have internal names.</span></span> <span data-ttu-id="8ad7e-106">たとえば、**op_Increment** は ++ 演算子の内部名です。</span><span class="sxs-lookup"><span data-stu-id="8ad7e-106">For example, **op_Increment** is the internal name of the ++ operator.</span></span> <span data-ttu-id="8ad7e-107">このようなメソッド名を使用したり、明示的に呼び出したりしないでください。</span><span class="sxs-lookup"><span data-stu-id="8ad7e-107">You should not use or explicitly call such method names.</span></span>  
  
 <span data-ttu-id="8ad7e-108">次の例では CS0571 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="8ad7e-108">The following sample generates CS0571:</span></span>  
  
```csharp  
// CS0571.cs  
public class MyClass  
{  
   public static MyClass operator ++ (MyClass c)  
   {  
      return null;  
   }  
  
   public static int prop  
   {  
      get  
      {  
         return 1;  
      }  
      set  
      {  
      }  
   }  
  
   public static void Main()  
   {  
      op_Increment(null);   // CS0571  
      // use the increment operator as follows  
      // MyClass x = new MyClass();  
      // x++;  
  
      set_prop(1);      // CS0571  
      // try the following line instead  
      // prop = 1;  
   }  
}  
```
