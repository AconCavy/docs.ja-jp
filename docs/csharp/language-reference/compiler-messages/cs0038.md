# <a name="compiler-error-cs0038"></a><span data-ttu-id="c4272-103">コンパイラ エラー CS0038</span><span class="sxs-lookup"><span data-stu-id="c4272-103">Compiler Error CS0038</span></span>

<span data-ttu-id="c4272-104">入れ子になっている型 'type2' を経由して、外の型 'type1' の静的でないメンバーにアクセスすることはできません</span><span class="sxs-lookup"><span data-stu-id="c4272-104">Cannot access a nonstatic member of outer type 'type1' via nested type 'type2'</span></span>

 <span data-ttu-id="c4272-105">クラスのフィールドは、入れ子になったクラスに自動的に使用できません。</span><span class="sxs-lookup"><span data-stu-id="c4272-105">A field in a class is not automatically available to a nested class.</span></span> <span data-ttu-id="c4272-106">入れ子になったクラスに使用できるようにするには、フィールドを[静的](../keywords/static.md)にする必要があります。</span><span class="sxs-lookup"><span data-stu-id="c4272-106">To be available to a nested class, the field must be [static](../keywords/static.md).</span></span> <span data-ttu-id="c4272-107">静的ではない場合、外側のクラスのインスタンスを作成する必要があります。</span><span class="sxs-lookup"><span data-stu-id="c4272-107">Otherwise, you must create an instance of the outer class.</span></span> <span data-ttu-id="c4272-108">詳細については、「[入れ子にされた型](../../programming-guide/classes-and-structs/nested-types.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="c4272-108">For more information, see [Nested Types](../../programming-guide/classes-and-structs/nested-types.md).</span></span>

 <span data-ttu-id="c4272-109">次の例では CS0038 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="c4272-109">The following sample generates CS0038:</span></span>

```csharp
// CS0038.cs
class OuterClass
{
    public int count;
    // Try the following line instead.
    // public static int count;

    class InnerClass
    {
        void Func()
        {
            // or, create an instance
            // OuterClass class_inst = new OuterClass();
            // int count2 = class_inst.count;
            int count2 = count;   // CS0038
        }
    }

    public static void Main()
    {
    }
}
```
