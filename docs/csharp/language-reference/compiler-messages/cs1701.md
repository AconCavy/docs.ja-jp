<span data-ttu-id="29f57-103">アセンブリ参照 "Assembly Name #1" が "Assembly Name #2" に一致した場合、実行時ポリシーの指定が必要になることがあります。</span><span class="sxs-lookup"><span data-stu-id="29f57-103">Assuming assembly reference "Assembly Name #1" matches "Assembly Name #2", you may need to supply runtime policy</span></span>

## <a name="example"></a><span data-ttu-id="29f57-106">例</span><span class="sxs-lookup"><span data-stu-id="29f57-106">Example</span></span>  
 <span data-ttu-id="29f57-107">次のマルチファイルの例では、2 つの異なる外部エイリアスを使ってアセンブリを参照しています。</span><span class="sxs-lookup"><span data-stu-id="29f57-107">The following multifile sample references an assembly using two different external aliases.</span></span> <span data-ttu-id="29f57-108">この最初の例では、CS1701_d アセンブリを生成する古いバージョンのコードを作成します。</span><span class="sxs-lookup"><span data-stu-id="29f57-108">This first sample builds the older version of the code that creates assembly CS1701_d.</span></span>  
  
```csharp  
// CS1701_a.cs  
// compile with: /target:library /out:cs1701_d.dll /keyfile:mykey.snk  
using System.Reflection;  
[assembly:AssemblyVersion("1.0")]  
public class A {  
   public void M1() {}  
}  
  
public class C1 {}  
```  
  
## <a name="example"></a><span data-ttu-id="29f57-109">例</span><span class="sxs-lookup"><span data-stu-id="29f57-109">Example</span></span>  
 <span data-ttu-id="29f57-110">これは、CS1701_d アセンブリの新しいバージョンを生成するコードです。</span><span class="sxs-lookup"><span data-stu-id="29f57-110">This is the code that creates the newer version of assembly CS1701_d.</span></span> <span data-ttu-id="29f57-111">これ古いバージョンとは異なるディレクトリにコンパイルされることに注意してください。これは、出力ファイルが同じ名前であるために必要な処理です。</span><span class="sxs-lookup"><span data-stu-id="29f57-111">Note that it compiles into a different directory than the older version, necessary since the output files have the same names.</span></span>

```csharp  
// compile with: /target:library /out:c:\\cs1701_d.dll /keyfile:mykey.snk  
using System.Reflection;  
[assembly:AssemblyVersion("2.0")]  
public class A {  
   public void M2() {}  
   public void M1() {}  
}  
  
public class C2 {}  
public class C1 {}  
```  
  
## <a name="example"></a><span data-ttu-id="29f57-112">例</span><span class="sxs-lookup"><span data-stu-id="29f57-112">Example</span></span>  
 <span data-ttu-id="29f57-113">この例では、外部エイリアス A1 および A2 が設定されます。</span><span class="sxs-lookup"><span data-stu-id="29f57-113">This sample sets up the external aliases A1 and A2.</span></span>  
  
```csharp  
// CS1701_c.cs  
// compile with: /target:library /reference:A2=c:\\cs1701_d.dll /reference:A1=cs1701_d.dll  
  
extern alias A1;  
extern alias A2;  
// using System;  
using a1 = A1::A;  
using a2 = A2::A;  
  
public class Ref {  
   public static a1 A1() { return new a1(); }  
   public static a2 A2() { return new a2(); }  
  
   public static A1::C1 M1() { return new A1::C1(); }  
   public static A2::C2 M2() { return new A2::C2(); }  
}  
```

 <span data-ttu-id="29f57-115">この例では、A の 2 つの異なるエイリアスを使用してメソッドが呼び出されます。次の例では C1701 エラーが生成されます。</span><span class="sxs-lookup"><span data-stu-id="29f57-115">This sample calls methods using two different aliases of A. The following sample generates C1701.</span></span>

