# <a name="compiler-error-cs0504"></a><span data-ttu-id="bc954-103">コンパイラ エラー CS0504</span><span class="sxs-lookup"><span data-stu-id="bc954-103">Compiler Error CS0504</span></span>
<span data-ttu-id="bc954-104">定数 'variable' を static に設定することはできません。</span><span class="sxs-lookup"><span data-stu-id="bc954-104">The constant 'variable' cannot be marked static</span></span>  
  
 <span data-ttu-id="bc954-105">変数が [const](../keywords/const.md) の場合、[static](../keywords/static.md) でもあります。</span><span class="sxs-lookup"><span data-stu-id="bc954-105">If a variable is [const](../keywords/const.md), it is also [static](../keywords/static.md).</span></span> <span data-ttu-id="bc954-106">**const** で **static** な変数を使用するには、単に **const** として変数を宣言してください。**static** 変数のみが必要な場合は、単に **static** として宣言してください。</span><span class="sxs-lookup"><span data-stu-id="bc954-106">If you want a **const** and **static** variable, just declare that variable as **const**; if all you want is a **static** variable, just mark it **static**.</span></span>  
  
 <span data-ttu-id="bc954-107">次の例では CS0504 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="bc954-107">The following sample generates CS0504:</span></span>  
  
```csharp  
// CS0504.cs  
namespace x  
{  
   abstract public class clx  
   {  
      static const int i = 0;   // CS0504, cannot be both static and const  
      abstract public void f();  
   }  
}  
```
