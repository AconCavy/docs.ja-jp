
2. <span data-ttu-id="16eb1-109">*My Documents* フォルダーに *cs2032.cs* というテキスト ファイルを作成し、任意の必要な内容を入力します。</span><span class="sxs-lookup"><span data-stu-id="16eb1-109">In the *My Documents* folder, create a text file that is named *cs2032.cs* and that contains whatever you want.</span></span>
