# <a name="compiler-error-cs0507"></a><span data-ttu-id="1437d-103">コンパイラ エラー CS0507</span><span class="sxs-lookup"><span data-stu-id="1437d-103">Compiler Error CS0507</span></span>

<span data-ttu-id="1437d-104">'function1': 'access' の継承メンバー 'function2' をオーバーライドするときに、アクセス修飾子を変更できません</span><span class="sxs-lookup"><span data-stu-id="1437d-104">'function1' : cannot change access modifiers when overriding 'access' inherited member 'function2'</span></span>  
  
 <span data-ttu-id="1437d-105">メソッド オーバーライドで、アクセス指定を変更しようとしています。</span><span class="sxs-lookup"><span data-stu-id="1437d-105">An attempt was made to change the access specification in a method override.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1437d-106">例</span><span class="sxs-lookup"><span data-stu-id="1437d-106">Example</span></span>  

 <span data-ttu-id="1437d-107">次の例では CS0507 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="1437d-107">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="1437d-108">例</span><span class="sxs-lookup"><span data-stu-id="1437d-108">Example</span></span>  

 <span data-ttu-id="1437d-109">CS0507 は、参照されているメタデータで定義されている `protected internal` とマークされたメソッドを、クラスがオーバーライドしようとした場合にも発生する可能性があります。</span><span class="sxs-lookup"><span data-stu-id="1437d-109">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="1437d-110">この場合、オーバーライドするメソッドは `protected` とマークする必要があります。</span><span class="sxs-lookup"><span data-stu-id="1437d-110">In this situation, the overriding method should be marked as `protected`.</span></span>  
  
```csharp  
// CS0507_b.cs  
// compile with: /target:library  
abstract public class clx  
{  
   virtual protected internal void f() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="1437d-111">例</span><span class="sxs-lookup"><span data-stu-id="1437d-111">Example</span></span>  

 <span data-ttu-id="1437d-112">次の例では CS0507 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="1437d-112">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507_c.cs

public class cly : clx  
{  
   protected internal override void f() {}   // CS0507  
   // try the following line instead  
   // protected override void f() {}   // OK

}  
```
