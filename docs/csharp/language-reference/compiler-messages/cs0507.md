# <a name="compiler-error-cs0507"></a><span data-ttu-id="21ab6-103">コンパイラ エラー CS0507</span><span class="sxs-lookup"><span data-stu-id="21ab6-103">Compiler Error CS0507</span></span>
<span data-ttu-id="21ab6-104">'function1': 'access' の継承メンバー 'function2' をオーバーライドするときに、アクセス修飾子を変更できません</span><span class="sxs-lookup"><span data-stu-id="21ab6-104">'function1' : cannot change access modifiers when overriding 'access' inherited member 'function2'</span></span>  
  
 <span data-ttu-id="21ab6-105">メソッド オーバーライドで、アクセス指定を変更しようとしています。</span><span class="sxs-lookup"><span data-stu-id="21ab6-105">An attempt was made to change the access specification in a method override.</span></span>  
  
## <a name="example"></a><span data-ttu-id="21ab6-106">例</span><span class="sxs-lookup"><span data-stu-id="21ab6-106">Example</span></span>  
 <span data-ttu-id="21ab6-107">次の例では CS0507 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="21ab6-107">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="21ab6-108">例</span><span class="sxs-lookup"><span data-stu-id="21ab6-108">Example</span></span>  
 <span data-ttu-id="21ab6-109">CS0507 は、参照されているメタデータで定義されている `protected internal` とマークされたメソッドを、クラスがオーバーライドしようとした場合にも発生する可能性があります。</span><span class="sxs-lookup"><span data-stu-id="21ab6-109">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="21ab6-110">この場合、オーバーライドするメソッドは `protected` とマークする必要があります。</span><span class="sxs-lookup"><span data-stu-id="21ab6-110">In this situation, the overriding method should be marked as `protected`.</span></span>  
  
```csharp  
// CS0507_b.cs  
// compile with: /target:library  
abstract public class clx  
{  
   virtual protected internal void f() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="21ab6-111">例</span><span class="sxs-lookup"><span data-stu-id="21ab6-111">Example</span></span>  
 <span data-ttu-id="21ab6-112">次の例では CS0507 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="21ab6-112">The following sample generates CS0507.</span></span>

