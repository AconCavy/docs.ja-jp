<span data-ttu-id="0eefd-103">属性 'attribute' はこの宣言型では無効です。</span><span class="sxs-lookup"><span data-stu-id="0eefd-103">Attribute 'attribute' is not valid on this declaration type.</span></span> <span data-ttu-id="0eefd-104">'type' 宣言でのみ有効です。</span><span class="sxs-lookup"><span data-stu-id="0eefd-104">It is valid on 'type' declarations only.</span></span>

## <a name="example"></a><span data-ttu-id="0eefd-108">例</span><span class="sxs-lookup"><span data-stu-id="0eefd-108">Example</span></span>

 <span data-ttu-id="0eefd-109">次の例では CS0592 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="0eefd-109">The following sample generates CS0592:</span></span>  

```csharp
// CS0592.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Interface)]  
public class MyAttribute : Attribute
{  
}  
  
[MyAttribute]  
// Generates CS0592 because MyAttribute is not valid for a class.
public class A
{  
    public static void Main()  
    {  
    }  
}  
```
