# <a name="compiler-error-cs0311"></a><span data-ttu-id="dfc34-102">コンパイラ エラー CS0311</span><span class="sxs-lookup"><span data-stu-id="dfc34-102">Compiler Error CS0311</span></span>

<span data-ttu-id="dfc34-103">型 'type1' をジェネリック型の型パラメーター 'T' またはメソッド '\<name>' として使用することはできません。</span><span class="sxs-lookup"><span data-stu-id="dfc34-103">The type 'type1' cannot be used as type parameter 'T' in the generic type or method '\<name>'.</span></span> <span data-ttu-id="dfc34-104">'type1' から 'type2' への暗黙的な参照変換がありません。</span><span class="sxs-lookup"><span data-stu-id="dfc34-104">There is no implicit reference conversion from 'type1' to 'type2'.</span></span>  
  
 <span data-ttu-id="dfc34-105">ジェネリック型パラメーターに制約を適用する場合は、具象型の引数から制約の型への暗黙的な ID 変換または参照変換が必要です。</span><span class="sxs-lookup"><span data-stu-id="dfc34-105">When a constraint is applied to a generic type parameter, an implicit identity or reference conversion must exist from the concrete argument to the type of the constraint.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="dfc34-106">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="dfc34-106">To correct this error</span></span>

2. <span data-ttu-id="dfc34-108">クラスがある場合は、制約またはその他の操作を削除し、暗黙的な参照変換または ID 変換を有効にできます。</span><span class="sxs-lookup"><span data-stu-id="dfc34-108">If you own the class, you can remove the constraint or else do something to enable an implicit reference or identity conversion.</span></span> <span data-ttu-id="dfc34-109">たとえば、最初の型から 2 番目の型を継承できます。</span><span class="sxs-lookup"><span data-stu-id="dfc34-109">For example, you can make the second type inherit from the first.</span></span>  
  
## <a name="example"></a><span data-ttu-id="dfc34-110">例</span><span class="sxs-lookup"><span data-stu-id="dfc34-110">Example</span></span>  
  
```csharp  
// cs0311.cs  
class B {}  
class C {}  
class Test<T> where T : C  
{ }  
  
class Program  
{  
    static void Main()  
    {  
        Test<B> test = new Test<B>(); //CS0311  
    }  
}  
```  
  
 <span data-ttu-id="dfc34-111">値型引数を使用するときにこのエラーが発生する場合は、暗黙的な数値変換 (たとえば、`short` から `int` への変換) が、ジェネリック型パラメーターの条件を満たしていません。</span><span class="sxs-lookup"><span data-stu-id="dfc34-111">If this error occurs when trying to use a value-type argument, notice that an implicit numeric conversion, for example from `short` to `int`, does not satisfy a generic type parameter.</span></span>
