## <a name="example"></a><span data-ttu-id="5285b-111">例</span><span class="sxs-lookup"><span data-stu-id="5285b-111">Example</span></span>  
  
```csharp  
// cs0311.cs  
class B {}  
class C {}  
class Test<T> where T : C  
{ }  
  
class Program  
{  
    static void Main()  
    {  
        Test<B> test = new Test<B>(); //CS0311  
    }  
}  
```
