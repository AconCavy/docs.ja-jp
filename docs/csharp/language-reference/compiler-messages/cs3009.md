## <a name="example"></a><span data-ttu-id="ca947-108">例</span><span class="sxs-lookup"><span data-stu-id="ca947-108">Example</span></span>  

 <span data-ttu-id="ca947-109">次の例では CS3009 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="ca947-109">The following example generates CS3009:</span></span>  
  
```csharp  
// CS3009.cs

[CLSCompliant(false)]

{  
}

{  
    public static void Main () {}  
}  
```
