## <a name="example"></a><span data-ttu-id="ec7f8-106">例</span><span class="sxs-lookup"><span data-stu-id="ec7f8-106">Example</span></span>

 <span data-ttu-id="ec7f8-107">次の例では、`MyClass` にアクセス修飾子が指定されていないため、アクセシビリティは既定の `private` なので、CS0050 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="ec7f8-107">The following sample generates CS0050 because no accessibility modifier is supplied for `MyClass`, and its accessibility therefore defaults to `private`:</span></span>

```csharp

class MyClass // Accessibility defaults to private.
// Try the following line instead.
// public class MyClass
{
}

public class MyClass2
{
    public static MyClass MyMethod()   // CS0050
    {
        return new MyClass();
    }

    public static void Main() { }
}
```
