# <a name="compiler-error-cs1721"></a><span data-ttu-id="cb408-103">コンパイラ エラー CS1721</span><span class="sxs-lookup"><span data-stu-id="cb408-103">Compiler Error CS1721</span></span>

<span data-ttu-id="cb408-104">クラス 'class' に複数の基本クラスを指定することはできません: 'class_1' および 'class_2'</span><span class="sxs-lookup"><span data-stu-id="cb408-104">Class 'class' cannot have multiple base classes: 'class_1' and 'class_2'</span></span>

<span data-ttu-id="cb408-105">このエラー メッセージの最も一般的な原因としては、多重継承の使用があります。</span><span class="sxs-lookup"><span data-stu-id="cb408-105">The most common cause of this error message is attempting to use multiple inheritance.</span></span> <span data-ttu-id="cb408-106">C# のクラスは 1 つのクラスから直接継承することしかできません。</span><span class="sxs-lookup"><span data-stu-id="cb408-106">A class in C# may only inherit directly from one class.</span></span> <span data-ttu-id="cb408-107">ただし、1 つのクラスは任意の数のインターフェイスを実装できます。</span><span class="sxs-lookup"><span data-stu-id="cb408-107">However, a class can implement any number of interfaces.</span></span>

## <a name="example"></a><span data-ttu-id="cb408-108">例</span><span class="sxs-lookup"><span data-stu-id="cb408-108">Example</span></span>

<span data-ttu-id="cb408-109">次の例では、CS1721 が生成される仕組みの 1 つを示しています。</span><span class="sxs-lookup"><span data-stu-id="cb408-109">The following example shows one way in which CS1721 is generated:</span></span>

```csharp
// CS1721.cs

public class MyClass : A, B {}   // CS1721
```

## <a name="to-correct-this-error"></a><span data-ttu-id="cb408-110">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="cb408-110">To correct this error</span></span>

<span data-ttu-id="cb408-111">このエラーを解決するには、次のようなさまざまな方法があります。</span><span class="sxs-lookup"><span data-stu-id="cb408-111">The following are different ways to correct this error:</span></span>

- <span data-ttu-id="cb408-112">クラス `B` を `A` から継承し、`MyClass` を `B` から継承します。</span><span class="sxs-lookup"><span data-stu-id="cb408-112">Make class `B` inherit from `A`, and `MyClass` inherit from `B`:</span></span>

    ```csharp
    public class A {}
    public class B : A {}
    public class MyClass : B {}
    ```

- <span data-ttu-id="cb408-113">`B` をインターフェイスとして宣言します。</span><span class="sxs-lookup"><span data-stu-id="cb408-113">Declare `B` as an interface.</span></span> <span data-ttu-id="cb408-114">`MyClass` をインターフェイス `B`、およびクラス `A` から継承します。</span><span class="sxs-lookup"><span data-stu-id="cb408-114">Make `MyClass` inherit from the interface `B`, and the class `A`:</span></span>

    ```csharp
    public class A {}
    public interface B {}
    public class MyClass : A, B {}
    ```
