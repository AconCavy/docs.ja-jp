# <a name="compiler-warning-level-2-cs0108"></a><span data-ttu-id="e319f-103">コンパイラの警告 (レベル 2) CS0108</span><span class="sxs-lookup"><span data-stu-id="e319f-103">Compiler Warning (level 2) CS0108</span></span>

<span data-ttu-id="e319f-104">'member1' は継承したメンバー 'member2' を非表示にします。</span><span class="sxs-lookup"><span data-stu-id="e319f-104">'member1' hides inherited member 'member2'.</span></span> <span data-ttu-id="e319f-105">非表示にする場合は、キーワード new を使用してください。</span><span class="sxs-lookup"><span data-stu-id="e319f-105">Use the new keyword if hiding was intended.</span></span>

 <span data-ttu-id="e319f-106">変数は、基底クラスの変数と同じ名前で宣言されました。</span><span class="sxs-lookup"><span data-stu-id="e319f-106">A variable was declared with the same name as a variable in a base class.</span></span> <span data-ttu-id="e319f-107">ただし、[new](../keywords/new-modifier.md) キーワードは使用されませんでした。</span><span class="sxs-lookup"><span data-stu-id="e319f-107">However, the [new](../keywords/new-modifier.md) keyword was not used.</span></span> <span data-ttu-id="e319f-108">この警告は、**new** を使用することを推奨しています。この変数は、**new** が宣言で使用されたかのように宣言されています。</span><span class="sxs-lookup"><span data-stu-id="e319f-108">This warning informs you that you should use **new**; the variable is declared as if **new** had been used in the declaration.</span></span>

 <span data-ttu-id="e319f-109">次の例では CS0108 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="e319f-109">The following sample generates CS0108:</span></span>

```csharp
// CS0108.cs
// compile with: /W:2
using System;

namespace x
{
    public class clx
    {
        public int i = 1;
    }

    public class cly : clx
    {
        public static int i = 2;   // CS0108, use the new keyword
        // Use the following line instead:
        // public static new int i = 2;

      public static void Main()
      {
         Console.WriteLine(i);
      }
   }
}
```
