# <a name="compiler-error-cs0188"></a><span data-ttu-id="b216d-102">コンパイラ エラー CS0188</span><span class="sxs-lookup"><span data-stu-id="b216d-102">Compiler Error CS0188</span></span>

<span data-ttu-id="b216d-103">すべてのフィールドが割り当てられるまでは、'this' オブジェクトは使用できません</span><span class="sxs-lookup"><span data-stu-id="b216d-103">The 'this' object cannot be used before all of its fields are assigned to</span></span>  
  
 <span data-ttu-id="b216d-104">コンストラクターから `struct` でメソッドを呼び出す前に、コンストラクターによって `struct` 内のすべてのフィールドが割り当てられる必要があります。</span><span class="sxs-lookup"><span data-stu-id="b216d-104">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>  
  
 <span data-ttu-id="b216d-105">構造体コンストラクターでプロパティを初期化しようとしたときにこのエラーが発生した場合の解決策は、プロパティ自体ではなく、そのプロパティのバッキング フィールドを直接初期化することです。</span><span class="sxs-lookup"><span data-stu-id="b216d-105">If you see this error when trying to initialize a property in a struct constructor, the solution is to initialize the backing field of that property directly instead of the property itself.</span></span>

## <a name="example"></a><span data-ttu-id="b216d-106">例</span><span class="sxs-lookup"><span data-stu-id="b216d-106">Example</span></span>

 <span data-ttu-id="b216d-107">次の例では CS0188 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="b216d-107">The following sample generates CS0188:</span></span>  

```csharp
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
