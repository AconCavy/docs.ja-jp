# <a name="compiler-error-cs0233"></a><span data-ttu-id="cfef3-103">コンパイラ エラー CS0233</span><span class="sxs-lookup"><span data-stu-id="cfef3-103">Compiler Error CS0233</span></span>
<span data-ttu-id="cfef3-104">'identifier' には定義済みのサイズが指定されていないため、sizeof は unsafe コンテキストでのみ使用できます</span><span class="sxs-lookup"><span data-stu-id="cfef3-104">'identifier' does not have a predefined size, therefore sizeof can only be used in an unsafe context</span></span>
  
 <span data-ttu-id="cfef3-105">[unsafe](../keywords/unsafe.md) コンテキストがない場合、[sizeof](../operators/sizeof.md) 演算子は、サイズがコンパイル時の定数である型にのみ使用できます。</span><span class="sxs-lookup"><span data-stu-id="cfef3-105">Without [unsafe](../keywords/unsafe.md) context, the [sizeof](../operators/sizeof.md) operator can only be used for types which size is a compile-time constant.</span></span> <span data-ttu-id="cfef3-106">このエラーが発生した場合は、unsafe コンテキストを使用します。</span><span class="sxs-lookup"><span data-stu-id="cfef3-106">If you are getting this error, use an unsafe context.</span></span>
  
<span data-ttu-id="cfef3-107">次の例では CS0233 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="cfef3-107">The following example generates CS0233:</span></span>
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following instead:  
        // unsafe
        // {
        //     Console.WriteLine(sizeof(S));
        // }
   }  
}  
```
