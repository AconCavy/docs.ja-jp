# <a name="compiler-error-cs1919"></a><span data-ttu-id="079e7-103">コンパイラ エラー CS1919</span><span class="sxs-lookup"><span data-stu-id="079e7-103">Compiler Error CS1919</span></span>

<span data-ttu-id="079e7-104">安全でない型 '型名' をオブジェクトの作成に使用することはできません。</span><span class="sxs-lookup"><span data-stu-id="079e7-104">Unsafe type 'type name' cannot be used in object creation.</span></span>  
  
 <span data-ttu-id="079e7-105">`new` 演算子では、マネージド ヒープ上にのみオブジェクトが作成されます。</span><span class="sxs-lookup"><span data-stu-id="079e7-105">The `new` operator creates objects only on the managed heap.</span></span> <span data-ttu-id="079e7-106">ただし、言語の相互運用機能を使用して、ポインターを返すネイティブ メソッドを呼び出すことで、アンマネージ メモリに間接的にオブジェクトを作成できます。</span><span class="sxs-lookup"><span data-stu-id="079e7-106">However, you can create objects in unmanaged memory indirectly by using the interoperability capabilities of the language to call native methods that return pointers.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="079e7-107">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="079e7-107">To correct this error</span></span>  
  
1. <span data-ttu-id="079e7-108">オブジェクト作成式の new で、安全な型を使用します。</span><span class="sxs-lookup"><span data-stu-id="079e7-108">Use a safe type in the new object creation expression.</span></span> <span data-ttu-id="079e7-109">たとえば、`char*` または `int*` ではなく、`char` または `int` を使用します。</span><span class="sxs-lookup"><span data-stu-id="079e7-109">For example, use `char` or `int` instead of `char*` or `int*`.</span></span>  
  
2. <span data-ttu-id="079e7-110">アンマネージ メモリにオブジェクトを作成する必要がある場合、Win32 メソッドまたは COM メソッドを使用します。または、C または C++ で独自の関数を記述して、それを C# から呼び出します。</span><span class="sxs-lookup"><span data-stu-id="079e7-110">If you must create objects in unmanaged memory, use a Win32 or COM method or else write your own function in C or C++ and call it from C#.</span></span>  
  
## <a name="example"></a><span data-ttu-id="079e7-111">例</span><span class="sxs-lookup"><span data-stu-id="079e7-111">Example</span></span>

 <span data-ttu-id="079e7-112">次の例では、ポインター型が安全でないため、CS1919 エラーが発生します。</span><span class="sxs-lookup"><span data-stu-id="079e7-112">The following example generates CS1919 because a pointer type is unsafe:</span></span>

```csharp
// cs1919.cs  
// Compile with: /unsafe  
unsafe public class C  
{  
    public static int Main()  
    {  
        var col1 = new int* { }; // CS1919  
        var col2 = new char* { }; // CS1919  
        return 1;  
    }  
}  
```
