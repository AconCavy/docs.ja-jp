<span data-ttu-id="762b5-109">*expr* の箇所には、ブール値に評価される式を指定します。</span><span class="sxs-lookup"><span data-stu-id="762b5-109">where *expr* is an expression that evaluates to a Boolean value.</span></span> <span data-ttu-id="762b5-110">`true` が返された場合は、例外ハンドラーが実行されます。`false` の場合は実行されません。</span><span class="sxs-lookup"><span data-stu-id="762b5-110">If it returns `true`, the exception handler executes; if `false`, it does not.</span></span>
