# <a name="ascending-c-reference"></a>ascending (C# リファレンス)
`ascending` コンテキスト キーワードは、クエリ式の [orderby 句](./orderby-clause.md)で使用され、並べ替え順序として昇順を指定します。 `ascending` は既定の並べ替え順序であるため、指定する必要はありません。
