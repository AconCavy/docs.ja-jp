# <a name="endregion-c-reference"></a>#endregion (C# リファレンス)
`#endregion` は [#region](./preprocessor-region.md) ブロックの終了を示します。 次に例を示します。  
  
```csharp
#region MyClass definition  
class MyClass
{  
    static void Main()
    {  
    }  
}  
#endregion  
```
