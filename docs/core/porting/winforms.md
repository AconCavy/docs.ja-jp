## <a name="fix-assembly-info-generation"></a><span data-ttu-id="1f95a-160">アセンブリ情報の生成を修正する</span><span class="sxs-lookup"><span data-stu-id="1f95a-160">Fix assembly info generation</span></span>

<span data-ttu-id="1f95a-161">.NET Framework で作成された Windows Forms プロジェクトには、生成されるアセンブリのバージョンなどのアセンブリの属性を格納する `AssemblyInfo.cs` ファイルが含まれています。</span><span class="sxs-lookup"><span data-stu-id="1f95a-161">Windows Forms projects that were created with .NET Framework include an `AssemblyInfo.cs` file, which contains assembly attributes such as the version of the assembly to be generated.</span></span> <span data-ttu-id="1f95a-162">SDK スタイルのプロジェクトでは、SDK プロジェクト ファイルに基づいて、この情報が自動的に生成されます。</span><span class="sxs-lookup"><span data-stu-id="1f95a-162">SDK-style projects automatically generate this information for you based on the SDK project file.</span></span> <span data-ttu-id="1f95a-163">両方の種類の "アセンブリ情報" があると、競合が発生します。</span><span class="sxs-lookup"><span data-stu-id="1f95a-163">Having both types of "assembly info" creates a conflict.</span></span> <span data-ttu-id="1f95a-164">この問題は、自動生成を無効にして、既存の `AssemblyInfo.cs` ファイルをプロジェクトに強制的に使用させることで解決します。</span><span class="sxs-lookup"><span data-stu-id="1f95a-164">Resolve this problem by disabling automatic generation, which forces the project to use your existing `AssemblyInfo.cs` file.</span></span>

<span data-ttu-id="1f95a-165">メインの `<PropertyGroup>` ノードに追加する 3 つの設定があります。</span><span class="sxs-lookup"><span data-stu-id="1f95a-165">There are three settings to add to the main `<PropertyGroup>` node.</span></span>

- <span data-ttu-id="1f95a-169">**AssemblyName**</span><span class="sxs-lookup"><span data-stu-id="1f95a-169">**AssemblyName**</span></span>\

- <span data-ttu-id="1f95a-173">**RootNamespace**</span><span class="sxs-lookup"><span data-stu-id="1f95a-173">**RootNamespace**</span></span>\
<span data-ttu-id="1f95a-174">プロジェクトで使用される既定の名前空間。</span><span class="sxs-lookup"><span data-stu-id="1f95a-174">The default namespace used by your project.</span></span> <span data-ttu-id="1f95a-175">これは、.NET Framework プロジェクトの既定の名前空間と一致させる必要があります。</span><span class="sxs-lookup"><span data-stu-id="1f95a-175">This should match the default namespace of the .NET Framework project.</span></span>

<span data-ttu-id="1f95a-176">これら 3 つの要素を、`MyFormsCore.csproj` ファイルの `<PropertyGroup>` ノードに追加します。</span><span class="sxs-lookup"><span data-stu-id="1f95a-176">Add these three elements to the `<PropertyGroup>` node in the `MyFormsCore.csproj` file:</span></span>

    <OutputType>WinExe</OutputType>
    <TargetFramework>netcoreapp3.0</TargetFramework>
    <UseWindowsForms>true</UseWindowsForms>
