- <span data-ttu-id="54781-174">[System.Collections](https://github.com/dotnet/corefx/tree/master/src/System.Collections) と [System.Linq](https://github.com/dotnet/corefx/tree/master/src/System.Linq) は、データ構造上で作成および操作を行うので、プラットフォームに依存しません。</span><span class="sxs-lookup"><span data-stu-id="54781-174">[System.Collections](https://github.com/dotnet/corefx/tree/master/src/System.Collections) and [System.Linq](https://github.com/dotnet/corefx/tree/master/src/System.Linq) are platform-neutral, given that they create and operate over data structures.</span></span>

## <a name="comparisons-to-other-net-implementations"></a><span data-ttu-id="54781-175">その他の .NET 実装との比較</span><span class="sxs-lookup"><span data-stu-id="54781-175">Comparisons to other .NET implementations</span></span>

### <a name="comparison-with-net-framework"></a><span data-ttu-id="54781-177">.NET Framework との比較</span><span class="sxs-lookup"><span data-stu-id="54781-177">Comparison with .NET Framework</span></span>

<span data-ttu-id="54781-178">.NET は、最初に 2000 年に Microsoft によって発表され、進化してきました。</span><span class="sxs-lookup"><span data-stu-id="54781-178">.NET was first announced by Microsoft in 2000 and then evolved from there.</span></span> <span data-ttu-id="54781-179">.NET Framework はその後 20 年近くにわたり、Microsoft によって製造される主要な .NET 実装となっています。</span><span class="sxs-lookup"><span data-stu-id="54781-179">The .NET Framework has been the primary .NET implementation produced by Microsoft during that nearly two decade period.</span></span>

<span data-ttu-id="54781-180">.NET Core と .NET Framework の主な違いは、次のとおりです。</span><span class="sxs-lookup"><span data-stu-id="54781-180">The major differences between .NET Core and the .NET Framework:</span></span>

- <span data-ttu-id="54781-181">**アプリ モデル** -- .NET Core は一部の .NET Framework アプリ モデルをサポートしていません。</span><span class="sxs-lookup"><span data-stu-id="54781-181">**App-models** -- .NET Core doesn't support all the .NET Framework app-models.</span></span> <span data-ttu-id="54781-182">具体的には、ASP.NET Web フォームと ASP.NET MVC はサポートしませんが、ASP.NET Core MVC をサポートしています。</span><span class="sxs-lookup"><span data-stu-id="54781-182">In particular, it doesn't support ASP.NET Web Forms and ASP.NET MVC, but it supports ASP.NET Core MVC.</span></span> <span data-ttu-id="54781-183">また、.Net Core 3.0 以降、.NET Core は Windows 上の WPF と Windows フォームのみをサポートしています。</span><span class="sxs-lookup"><span data-stu-id="54781-183">And starting with .NET Core 3.0, .NET Core also supports WPF and Windows Forms on Windows only.</span></span>

- <span data-ttu-id="54781-188">**サブシステム** -- .NET Core は、より単純な実装とプログラミング モデルを目的として、.NET Framework 内のサブシステムのサブセットを実装します。</span><span class="sxs-lookup"><span data-stu-id="54781-188">**Subsystems** -- .NET Core implements a subset of the subsystems in the .NET Framework, with the goal of a simpler implementation and programming model.</span></span> <span data-ttu-id="54781-189">たとえば、コード アクセス セキュリティ (CAS) はサポートされていませんが、リフレクションはサポートされています。</span><span class="sxs-lookup"><span data-stu-id="54781-189">For example, Code Access Security (CAS) isn't supported, while reflection is supported.</span></span>
- <span data-ttu-id="54781-190">**プラットフォーム** -- .NET Framework は Windows と Windows Server をサポートしており、.NET Core は macOS と Linux もサポートしています。</span><span class="sxs-lookup"><span data-stu-id="54781-190">**Platforms** -- The .NET Framework supports Windows and Windows Server while .NET Core also supports macOS and Linux.</span></span>
- <span data-ttu-id="54781-191">**オープン ソース** -- .NET Core はオープン ソースであり、[.NET Framework の読み取り専用のサブセット](https://github.com/microsoft/referencesource)はオープン ソースです。</span><span class="sxs-lookup"><span data-stu-id="54781-191">**Open Source** -- .NET Core is open source, while a [read-only subset of the .NET Framework](https://github.com/microsoft/referencesource) is open source.</span></span>

<span data-ttu-id="54781-192">.NET Core はユニークな製品で、.NET Framework およびその他の .NET 実装とは大きな違いがありますが、ソースまたはバイナリの共有技術を使用してこれらの実装の間でコードを簡単に共有できます。</span><span class="sxs-lookup"><span data-stu-id="54781-192">While .NET Core is unique and has significant differences to the .NET Framework and other .NET implementations, it's straightforward to share code between these implementations, using either source or binary sharing techniques.</span></span>

<span data-ttu-id="54781-193">.NET Core は横並びのインストールに対応しており、そのランタイムは .NET Framework から完全に独立しているため、.NET Framework がインストールされているコンピューターに何の問題もなくインストールできます。</span><span class="sxs-lookup"><span data-stu-id="54781-193">Because .NET Core supports side-by-side installation and its runtime is completely independent of the .NET Framework, it can be installed on machines with .NET Framework installed without any issues.</span></span>

### <a name="comparison-with-mono"></a><span data-ttu-id="54781-194">Mono との比較</span><span class="sxs-lookup"><span data-stu-id="54781-194">Comparison with Mono</span></span>
