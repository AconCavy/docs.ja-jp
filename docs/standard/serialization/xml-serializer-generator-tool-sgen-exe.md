## <a name="remarks"></a><span data-ttu-id="8cfa9-145">コメント</span><span class="sxs-lookup"><span data-stu-id="8cfa9-145">Remarks</span></span>  
 <span data-ttu-id="8cfa9-146">XML シリアライザー ジェネレーターを使用しない場合、<xref:System.Xml.Serialization.XmlSerializer> はアプリケーションを実行するたびに、各型に対してシリアル化コードとシリアル化アセンブリを生成します。</span><span class="sxs-lookup"><span data-stu-id="8cfa9-146">When the XML Serializer Generator is not used, a <xref:System.Xml.Serialization.XmlSerializer> generates serialization code and a serialization assembly for each type every time an application is run.</span></span> <span data-ttu-id="8cfa9-147">XML シリアル化の起動時のパフォーマンスを向上させるには、Sgen ツールを使用して、これらのアセンブリを事前に生成します。</span><span class="sxs-lookup"><span data-stu-id="8cfa9-147">To improve the performance of XML serialization startup, use the Sgen.exe tool to generate those assemblies in advance.</span></span> <span data-ttu-id="8cfa9-148">生成したアセンブリは、アプリケーションで配置できます。</span><span class="sxs-lookup"><span data-stu-id="8cfa9-148">These assemblies can then be deployed with the application.</span></span>  
  
 <span data-ttu-id="8cfa9-149">XML シリアライザー ジェネレーターは、サーバーとの通信に XML Web サービス プロキシを使用するクライアントのパフォーマンスも向上させますが、これは型が初めて読み込まれるとき、シリアル化プロセスによってパフォーマンスが低下しないためです。</span><span class="sxs-lookup"><span data-stu-id="8cfa9-149">The XML Serializer Generator can also improve the performance of clients that use XML Web service proxies to communicate with servers because the serialization process will not incur a performance hit when the type is loaded the first time.</span></span>  
  
 <span data-ttu-id="8cfa9-150">これらの生成されたアセンブリは、Web サービスのサーバー側では使用できません。</span><span class="sxs-lookup"><span data-stu-id="8cfa9-150">These generated assemblies cannot be used on the server side of a Web service.</span></span> <span data-ttu-id="8cfa9-151">このツールは、Web サービス クライアントと手動シリアル化シナリオに対してのみ使用できます。</span><span class="sxs-lookup"><span data-stu-id="8cfa9-151">This tool is only for Web service clients and manual serialization scenarios.</span></span>  
  
 <span data-ttu-id="8cfa9-152">シリアル化する型を含むアセンブリの名前が MyType.dll の場合、関連するシリアル化アセンブリの名前は MyType.XmlSerializers.dll となります。</span><span class="sxs-lookup"><span data-stu-id="8cfa9-152">If the assembly containing the type to serialize is named MyType.dll, then the associated serialization assembly will be named MyType.XmlSerializers.dll.</span></span>
