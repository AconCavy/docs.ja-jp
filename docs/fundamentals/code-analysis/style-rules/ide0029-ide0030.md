### <a name="example"></a><span data-ttu-id="4b0af-134">例</span><span class="sxs-lookup"><span data-stu-id="4b0af-134">Example</span></span>

```csharp
// dotnet_style_coalesce_expression = true

// dotnet_style_coalesce_expression = false
var v = x != null ? x : y; // or
var v = x == null ? y : x;
```

