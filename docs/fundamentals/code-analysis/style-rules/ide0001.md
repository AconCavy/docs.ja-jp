## <a name="example"></a><span data-ttu-id="42068-120">例</span><span class="sxs-lookup"><span data-stu-id="42068-120">Example</span></span>

```csharp
using System.IO;
class C
{
    // IDE0001: 'System.IO.FileInfo' can be simplified to 'FileInfo'
    System.IO.FileInfo file;

    // Fixed code
    FileInfo file;
}
```

```vb
Imports System.IO

```
